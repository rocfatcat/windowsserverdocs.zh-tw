# <a name="get-started-with-windows-server-essentials"></a>開始使用 Windows Server Essentials 

>適用於：Windows Server 2016 Essentials;Windows Server 2019 Essentials

**Windows Server Essentials** edition 是連接到雲端的第一部伺服器的小型企業設計具有多達 25 名使用者及 50 台裝置。 不過，如果您考慮安裝任何版本的 Windows Server Essentials，我們會建議您考慮**Microsoft 365**。 如需有關超出 Microsoft 365 的 Windows Server Essentials 的詳細資訊，請參閱[What's New in Windows Server 2019 Essentials](what-s-new-19.md)。
