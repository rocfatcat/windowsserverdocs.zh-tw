##  <a name="BKMK_WhatIs"></a>電腦的完整的系統還原為何？  
 必須改硬碟或您的電腦無法位置無法使用的或無法開機，您可以還原系統從先前的備份的電腦。 完整的系統還原系統會是一次備份的狀態。  
  
> [!IMPORTANT]
>  您無法在電腦的硬體，例如系統版，不會取代電腦硬體類似來執行完整的系統還原。 安裝的作業系統為基礎電腦的硬體上密切相關。 不過，您可以執行硬碟的大小等於或大於要取代的完整的系統還原。  
  
 執行完整的系統還原，您可以選擇將所有應用程式、設定和設定熟悉使用者之前失敗、災難或遭竊，系統還原備份特定的電腦。 您也可以選擇您想要還原的磁碟區。  
  
 當規劃或準備完整的系統復原到網路的電腦，請考慮下列動作：  
  
### <a name="windows-preinstallation-environment"></a>Windows 預先安裝環境  
 Windows 預先安裝環境 (Windows PE\) 是設計用來安裝 Windows 的電腦準備最低作業系統。 針對執行 Windows Server Essentials 的伺服器，，當您插入還原媒體來還原電腦上的時，會自動安裝 Windows PE\。 針對執行 Windows Server Essentials 的伺服器，，當您開始使用 client 還原服務或 USB 快閃磁碟機的電腦時，會自動安裝 Windows PE\。  
  
 Windows PE\ 不支援 wireless 連接。 因為，所還原的電腦必須實際連接小型企業網路。  
  
### <a name="bitlocker"></a>BitLocker  
 BitLocker 磁碟機加密 (BitLocker) 是可在某些版本的 Windows Vista、Windows 7 和 Windows 8 的資料保護功能。 BitLocker 保護資料遭竊，或曝光遺失或遭竊，並提供更安全的資料刪除當電腦解除。  
  
 **針對 Windows Server Essentials:**如果您需要還原電腦已加密，使用 BitLocker（是否是只作業系統的磁碟機或作業系統磁碟機和單一多個其他的修正磁碟機），您仍然可以使用提供完整的系統還原精靈伺服器與光碟中所包含完整的系統還原媒體重新安裝映像硬碟包括作業系統，從備份及還原到新的或修復電腦的資料。  
  
 **針對 Windows Server Essentials:**如果您需要還原電腦已加密，使用 BitLocker（是否是只作業系統的磁碟機或作業系統磁碟機和單一多個其他的修正磁碟機），您仍然可以使用全系統還原精靈重新安裝硬碟映像，包括作業系統，從備份及還原資料的全新或已修復電腦。  
  
 伺服器備份磁碟機、資料夾和檔案，便可以伺服器以未加密的版本。 完整的系統還原]，在這個未加密的版本被複製到電腦。  
  
> [!NOTE]
>  成功完整的系統復原之後，您必須在電腦上重新啟用 BitLocker。

>

### <a name="encrypting-file-system-encrypted-files"></a>加密檔案系統加密的檔案

> [!WARNING]

##  <a name="BKMK_HowDoes"></a>「系統還原」環境如何運作？  
 提供與 Windows ServerÂ® 2012 年」程式集系統還原媒體的電腦上安裝 Windows 預先安裝環境 (Windows PE\)。 Windows PE\ 取代 MS-DOS 環境，以及包含 Windows 核心程式檔案。 在 Windows Server Essentials，有支援兩種方式可以還原系統：使用 client 還原服務的使用網路，並不會依賴媒體，或使用 USB 快閃磁碟機。  
  
> [!NOTE]

 在大部分案例中，程式檔案與驅動程式「系統還原」環境中所包含的所有所需的新或還原電腦重新開機。 根據 [新的或還原電腦的硬體，不可能會包含所有的儲存空間及網路介面卡驅動程式，當您重新開機新或還原電腦所需的「系統還原」環境。 完整的系統還原精靈讓您有機會安裝驅動程式，如果需要。 尋找您的硬體驅動程式的相關資訊，請查看[在尋找驅動程式我的硬體？](Restore-a-full-system-from-an-existing-client-computer-backup.md#BKMK_FindDrivers). 了解如何使用「系統還原媒體資訊，請查看[使用完整的系統還原精靈](Restore-a-full-system-from-an-existing-client-computer-backup.md#BKMK_Using)。  
  
##  <a name="BKMK_CreateBootable"></a>建立可開機 USB 快閃磁碟機來還原 client 的電腦  
 如果您需要從現有的備份還原 client 的電腦，但是找不到還原 CD 隨附（在 Windows Server Essentials) 伺服器或不想要設定 client 還原服務（在 Windows Server Essentials) 伺服器上，您可以建立可開機的 USB 快閃磁碟機。 您再可以使用 USB 快閃磁碟機開機 client 和系統還原。 您可以使用 USB 快閃磁碟機必須至少為 1 GB 或更大。  
  
#### <a name="to-create-a-bootable-usb-flash-drive"></a>若要建立可開機的 USB 快閃磁碟機  
  
1.  開放**儀表板**。  
  
2.  按一下**裝置**索引標籤。  
  
3.  在**工作**窗格中，按**設定自訂電腦備份與檔案歷史**。  
  
    > [!NOTE]
    >  在 Windows Server Essentials，按一下**Client 電腦備份工作**。  
  
4.  按一下**工具**索引標籤，然後按一下 [**建立鍵**中**電腦修復**區段。 建立電腦修復金鑰精靈開啟。  
  
5.  插入 1 GB 或更大的 USB 快閃磁碟機，在 [伺服器，然後依照精靈中的指示。  
  
    > [!CAUTION]
    >  USB 快閃磁碟機上的所有資料將會繼續都嗎。  
  
##  <a name="BKMK_Using"></a>使用完整的系統還原精靈  
 之後使用成功還原媒體、client 還原服務或 USB 快閃磁碟機，以開始您的電腦，確認所有的裝置驅動程式會載入還原或新 client 在電腦，[全系統還原」精靈會出現。 這個精靈可讓您存取伺服器、備份的電腦，以及您想要還原到電腦，以及執行實際還原程序的來源磁碟區。  
  
> [!NOTE]
>   Windows Server Essentials 不支援下列還原案例：  
>   
>  -   還原磁碟主開機記錄 (MBR) 以整合可延伸韌體介面 (UEFI) 根據電腦。  
> -   BIOS 系統還原 UEFI 日 GPT 備份。  
>   
>  如果您要還原這些案例中的資料，您將無法再開機系統。 此外，您可能無法使用硬碟大於兩個 tb 的大小。  
  
 **必要條件：**

##  <a name="BKMK_FindDrivers"></a>何處尋找我的硬體驅動程式？  
 根據 [新的或還原電腦的硬體，不可能會包含所有的儲存空間和網路介面卡驅動程式您重新開機還原您的電腦時，需要還原媒體。 您必須判斷的驅動程式遺失，在現有的媒體或 s 製造商網站上尋找驅動程式，將它們複製到快閃磁碟機，然後從快閃磁碟機將它們複製到新或還原電腦時執行完整的系統還原精靈。  
  
 時備份的電腦時，電腦的驅動程式會儲存在備份。 如果復原媒體不包含的所有需要驅動程式，您可以開放備份的電腦，並再複製到 USB 快閃磁碟機的驅動程式。  
  
#### <a name="to-copy-drivers-from-a-backup-to-a-usb-flash-drive"></a>若要將驅動程式從備份複製到 USB 快閃磁碟機  
  
1.  在另一部電腦，開放儀表板。  
  
2.  按一下**裝置**，然後按一下 [您需要的驅動程式的電腦。  
  
3.  按一下**還原的檔案或資料夾的電腦**。 還原的檔案或資料夾精靈會開啟。  
  
4.  按一下 [最新的成功備份，然後按一下**下一步**。  
  
5.  按一下 [磁碟區以開放，然後按一下**下一步**。 出現一個視窗列出的檔案和資料夾的備份。  
  
6.  在電腦上的 USB 連接器插入 USB 快閃磁碟機，然後將驅動程式的全系統還原] 資料夾複製到 USB 快閃磁碟機。  
  
    > [!NOTE]
    >  您可能需要按一下**上一層級**直到您看到的系統磁碟區根。  
  
7.  快閃磁碟機中移除，然後將它插入還原您的電腦。  
  
 當您將它還原時，為您的電腦安裝的驅動程式，您可以使用 USB 快閃磁碟機。 還原的檔案或資料夾精靈會尋找其他 USB 快閃磁碟機上的驅動程式時使用完整的系統還原精靈。 您有最有可能需要驅動程式是網路介面卡驅動程式和存放裝置驅動程式。
