# <a name="overview-of-the-dashboard-in-windows-server-essentials"></a>在 Windows Server Essentials 儀表板的概觀

>適用於：Windows Server 2016 Essentials 程式集 Windows Server 2012 R2、Windows Server 2012 程式集
 
 Windows Server Essentials 和 Windows Server 2012 R2 標準與 Windows Server Essentials 體驗功能的角色包含管理儀表板，可簡化以管理您的 Windows Server Essentials 的網路及伺服器您執行的工作。 使用 Windows Server Essentials 儀表板，您可以：  
  
-   完成您的伺服器設定  
  
-   存取和執行一般管理工作  
  
-   檢視伺服器警示和執行動作  
  
-   設定並變更的伺服器設定  
  
-   存取或搜尋網路上的協助主題  
  
-   在網頁上存取社群資源  
  
-   管理使用者帳號  
  
-   管理裝置與備份  
  
-   存取和設定伺服器的資料夾和硬碟管理  
  
-   檢視及管理增益集應用程式  
  
-   Microsoft online services 的整合  
  
 本主題包括：

##  <a name="BKMK_Design"></a>基本功能儀表板

|2|小節索引標籤|小節索引標籤提供 Windows Server Essentials 的系統管理工作第二個層級的存取。|  
|3|清單窗格|清單檢視] 中顯示的物件，您可以管理，並包含每個物件的基本資訊。|  
|4|詳細資料窗格|詳細資料窗格中會顯示在 [清單檢視] 中選取物件的其他資訊。|  
|5|工作] 窗格|[工作] 窗格包含連結工具和資訊，可協助您管理或物件分類通用設定（例如帳號或電腦）任一特定物件的屬性。 [工作] 窗格分為這些兩個區段︰<br /><br /> **工作的物件**œ 包含工具和資訊，可協助您管理連結（例如帳號或電腦）檢視清單中選取物件的屬性。<br /><br /> **全球工作**œ 包含工具和資訊，可協助您管理連結全球功能區域的工作。 全球工作包含工作，以新增新的物件、設定原則，等等。|

##  <a name="BKMK_Home"></a>儀表板首頁的功能  
 當您開放儀表板，**Home**頁面隨即顯示預設的**設定**分類顯示。 **Home**頁面上的 Windows Server Essentials 儀表板可讓您快速存取工作和資訊，協助您自訂您的伺服器，並設定鍵的功能。 首頁組成四個功能的區域，公開設定的資訊和工作您選取的選項。 下表描述的功能。  
  
|項目|功能|描述|  
|----------|-------------|-----------------|  
|1|瀏覽列|按一下存取的資訊和相關聯的該區段的工作，導覽列上的區段。 每當您開放儀表板，**Home**頁面預設會顯示。|  
|2|分類] 窗格|此窗格顯示功能區域，提供快速存取資訊，並設定工具，可協助您設定及自訂伺服器。 按一下顯示的任務和該分類相關聯的資源分類。|  
|3|工作] 窗格|此窗格顯示工作和會套用到所選分類的資訊連結。 按一下 [任務或資源來顯示其他資訊]。|  
|4|控制項目窗格|這個窗格中提供的簡短描述的功能或 [工作]，並提供開放精靈設定和資訊頁面的連結。 按一下連結以需要進一步動作。|

##  <a name="BKMK_AccessDb"></a>存取 Windows Server Essentials 儀表板  
 您可以使用其中一項下列方法存取儀表板。 是否您存取儀表板從伺服器、的電腦已連接到 Windows Server Essentials 的網路，或是遠端電腦根據您選擇的方法。  
  
 為了維持安全的網路，只系統管理員權限的使用者可以存取 Windows Server Essentials 儀表板。 此外，您無法使用建 Launchpad 從登入 Windows Server Essentials 儀表板。  
  
###  <a name="BKMK_Server"></a>從伺服器存取儀表板  
 當您安裝 Windows Server Essentials 時，安裝程序上建立的快速鍵儀表板**[開始]**畫面，並也在桌面上。 如果這些位置的快顯遺失，您可以使用**搜尋**窗格中尋找並執行程式儀表板。  
  
##### <a name="to-access-the-dashboard-from-the-server"></a>若要從伺服器存取儀表板  
  
-   伺服器以系統管理員的身分登入，然後執行下列其中一項：  
  
    -   在**[開始]**畫面中，按**儀表板**。  
  
    -   在桌面上，按兩下 [**儀表板**。  
  
    -   在**搜尋**窗格中，輸入**儀表板**，然後按一下 [**儀表板**中搜尋結果。  
  
###  <a name="BKMK_Network"></a>從電腦已連接到該網路的存取儀表板  
 在 Windows Server Essentials 之後將電腦加入網路, 系統管理員可以使用 Launchpad 存取伺服器儀表板的電腦。  
  
> [!WARNING]
>  在 Windows Server Essentials，連接的電腦上 client 儀表板，系統匣圖示，以滑鼠右鍵按一下，然後從操作功能表開放儀表板。  
  
##### <a name="to-access-the-dashboard-by-using-the-launchpad"></a>若要使用 Launchpad 存取儀表板  
  
1.  已加入該網路的電腦，請打開**Launchpad**。  
  
2.  在 Launchpad 功能表中，按一下 [**儀表板**。  
  
3.  儀表板**登入**頁面中，輸入您的網路系統管理員認證，然後按 ENTER 鍵。  
  
     Windows Server Essentials 儀表板來連接遠端開啟。  
  
###  <a name="BKMK_Remote"></a>從遠端電腦存取儀表板  
 當您從遠端電腦使用時，您可以使用遠端 Web 存取網站存取 Windows Server Essentials 儀表板。  
  
##### <a name="to-access-the-dashboard-by-using-remote-web-access"></a>使用遠端 Web 存取存取儀表板  
  
1.  遠端電腦時，從左網際網路瀏覽器，例如 Internet Explorer。  
  
2.  網際網路瀏覽器網址列中，輸入下列命令：**https://<DomainName\>/remote**，其中*網域名稱、*（例如 contoso.com) 貴組織的外部網域名稱。  
  
3.  輸入您的使用者名稱和密碼登入遠端 Web 存取網站。  
  
4.  在**電腦**區段 Web 遠端存取的**Home**頁面。找出您的伺服器，並按一下 [**連接**。  
  
     儀表板來連接遠端開啟。  
  
    > [!NOTE]
    >  如果您的伺服器不會出現在**電腦**區段的 [首頁] 頁面中，按**連接到更多電腦**，在清單中，尋找您的伺服器，然後按一下 [連接至該伺服器。  
    >   
    >  若要讓使用者存取儀表板從遠端網路存取權限，開放帳號，屬性頁面，然後選取**伺服器儀表板**選項，在**隨處存取**索引標籤。  
  
##  <a name="BKMK_UseSafeMode"></a>使用「安全模式  
 Windows Server Essentials 的「安全模式」功能會安裝在伺服器增益集。 安全模式儀表板變得不穩定或無法回應時應，如果偵測及隔離增益集，可能會造成問題，並顯示在**「安全模式設定**頁面上的下一次您開放儀表板。 從**「安全模式設定**頁面上，您可以停用以及增益集判斷造成問題的增益集一個。 您可以選擇將針對未來啟動，停用增益集，或您可以解除安裝增益集。  
  
 您可以檢視所有增益集，隨時都在伺服器上安裝的清單。 您也可以在安全模式下，會自動停用所有非 Microsoft 增益集開放儀表板。 Windows Server Essentials 也可讓您從另一部電腦在網路上存取「安全模式儀表板。  
  
#### <a name="to-view-a-list-of-installed-add-ins"></a>若要檢視清單安裝增益集  
  
-   從儀表板，請按一下**協助**，然後按一下 [ **「安全模式設定**。  
  
#### <a name="to-open-the-dashboard-in-safe-mode"></a>在安全模式打開儀表板  
  
-   在**搜尋**窗格中，輸入**安全**，然後按一下 [**儀表板（安全模式）**在搜尋結果。  
  
#### <a name="to-open-the-dashboard-in-safe-mode-from-another-computer-on-the-network"></a>打開儀表板中從網路上的另一部電腦的安全模式  
  
1.  已連接到該網路的電腦，開放 Windows Server Essentials Launchpad，，然後按一下**儀表板**。  
  
2.  儀表板登入頁面上，輸入的使用者名稱和密碼 account 具有權限來登入伺服器、選取**允許我選取的增益集載入**核取方塊，然後箭號來登入。
