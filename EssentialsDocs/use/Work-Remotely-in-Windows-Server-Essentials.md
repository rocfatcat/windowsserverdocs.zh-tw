##  <a name="BKMA_RWA"></a> Windows Server Essentials 中使用遠端 Web 存取  

 「遠端 Web 存取」可協助您在置身他處時與 Windows Server Essentials 網路保持連線。 如需詳細資訊，請參閱主題[Use Remote Web Access](Use-Remote-Web-Access-in-Windows-Server-Essentials.md)。  

 「遠端 Web 存取」可協助您在置身他處時與 Windows Server Essentials 網路保持連線。 如需詳細資訊，請參閱主題[Use Remote Web Access](../use/Use-Remote-Web-Access-in-Windows-Server-Essentials.md)。  

  
##  <a name="BKMK_3"></a> 使用 VPN 來連線到 Windows Server Essentials  
 如果您有已設定網路帳戶的用戶端電腦，並且那些帳戶可透過 VPN 連線來連線到執行 Windows Server Essentials 的託管伺服器，則所有在該託管伺服器上新建立的使用者帳戶在第一次登入該用戶端電腦時都必須使用 VPN。 從連線到伺服器的用戶端電腦完成以下程序。  
  
#### <a name="to-use-vpn-to-remotely-access-server-resources"></a>使用 VPN 從遠端存取伺服器資源  
  
1.  在用戶端電腦上按 Ctrl + Alt + Delete。  
  
2.  在登入畫面按一下 [切換使用者]  。  
  
3.  按一下畫面右下角的網路登入圖示。  
  
4.  使用您的網路使用者名稱和密碼來登入 Windows Server Essentials 網路。  
  
##  <a name="BKMK_App"></a> 用於連線到 Windows Server Essentials 的 My Server 應用程式  
 My Server 應用程式可讓您連接到資源，以及在 Windows Server Essentials 伺服器從以 Windows 為基礎的電腦、 膝上型電腦或 Surface 裝置上執行簡單的系統管理工作。 如果您的伺服器執行 Windows Server 2012，下載原始的 My Server 應用程式，從[應用程式的 Windows](https://windows.microsoft.com/windows-8/apps)。 如果您的伺服器正在執行 Windows Server Essentials，您必須改為下載 My Server 2012 R2 應用程式。  
  
 使用擴充的 My Server 2012 R2 應用程式，您便可以使用「遠端桌面」來連線到伺服器或用戶端電腦。 如果您的 Windows Server Essentials 伺服器與 Office 365 整合，您的訂閱包含 SharePoint Online，您也可以使用 在您的 SharePoint Online 文件庫中的 文件，開啟 SharePoint 小組網站從 My Server 2012 R2。  
  

 如需安裝和使用這些應用程式的詳細資訊，請參閱[Use the My Server App](Use-the-My-Server-App-to-Connect-to-Windows-Server-Essentials.md)。  

 如需安裝和使用這些應用程式的詳細資訊，請參閱[Use the My Server App](../use/Use-the-My-Server-App-to-Connect-to-Windows-Server-Essentials.md)。  

  
##  <a name="BKMK_2"></a> 使用適用於 Windows Phone 的 My Server 應用程式  
 （適用於 Windows Server 2012) 的 Windows Phone 的 My Server Windows 應用程式和 （適用於 Windows Server Essentials) 的 Windows Phone 的 My Server 2012 R2 應用程式專門設計來協助您在遠端工作時保持順暢連線透過智慧型手機與伺服器位置。 這是其中一個來設定伺服器供遠端存取之後，存取 Windows Server Essentials 的各種方式。  
  
 您可以從 Windows Phone 市集下載這兩個應用程式中的任何一個：  
  
- [安裝適用於 Windows Phone 的 My Server](http://www.windowsphone.com/store/app/my-server/6c2f98d5-6fcf-4e1d-b8b1-cde62ea1a94a)  
  
- [安裝適用於 Windows Phone 的 My Server 2012 R2](http://www.windowsphone.com/store/app/my-server-2012-r2/44f596b5-0477-4096-b96e-ddd6ef64ad6b)  
  
  如需有關 My Server phone 應用程式的詳細資訊，請參閱部落格文章[適用於 Windows Server Essentials 的 My Server phone 應用程式](http://blogs.technet.com/b/sbs/archive/2012/09/18/my-server-phone-app-for-windows-server-2012-essentials.aspx)。 如需 My Server 2012 R2 Phone 應用程式的詳細資訊，請參閱部落格文章 [My Server 2012 R2 Windows 和 Windows Phone 應用程式](http://blogs.technet.com/b/sbs/archive/2013/11/19/my-server-2012-r2-windows-and-windows-phone-apps.aspx)。
