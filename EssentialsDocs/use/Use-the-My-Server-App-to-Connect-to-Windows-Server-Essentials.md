    4.  CAROOT.cer 檔案，以滑鼠右鍵按一下，然後按一下**開放**。  
  
    5.  在**憑證**對話方塊中，按**安裝憑證**。  
  
    6.  選取 [**本機**安裝位置，然後再按一下為**下**。  
  
    7.  在**憑證存放區**精靈頁面上，選取**將所有憑證都放在市集中下列**，並使用**瀏覽**選擇**受信任的根憑證授權單位**市集。 然後按一下**完成**。

##  <a name="BKMK_ConnectServer"></a>如何從您的本機網路連接到您的伺服器  
 成功使用我的伺服器 app 在 Windows Server Essentials 的 Windows Phone、 Windows 8 和 Windows 8.1 中，您必須先在您的裝置上安裝伺服器的憑證。 這會裝置連接到您區域網路中執行 Windows Server Essentials 的伺服器。 使用下列程序連接到您區域網路中的伺服器，並安裝您的 Windows Phone 上的電腦執行 Windows 8 或 Windows 8.1 的伺服器的憑證。  
  
#### <a name="to-connect-to-your-server-from-your-windows-phone"></a>連接到您的伺服器，從您的 Windows Phone  
  
1.  在執行 Windows 8 或 Windows 8.1 的 Windows Phone 上開放 Internet Explorer。  
  
2.  在 [位址列中，輸入**http://<yourservername\>/connect/default.aspx?get=caroot.cer**，然後按 Enter 鍵。  
  
3.  當系統提示您安裝 caroot.cer 憑證時，按一下**安裝**。  
  
4.  等待憑證安裝完成。  
  
    > [!NOTE]
    >  憑證安裝完成後，您可以登入我伺服器應用程式適用於 Windows Phone 使用您的伺服器名稱和認證網路。  
  
#### <a name="to-connect-to-your-server-in-your-local-network-from-a-computer-running-windows-8-or-windows-81"></a>若要執行的是 Windows 8 或 Windows 8.1 的電腦連接到您區域網路中的伺服器  
  
1.  在電腦上執行 Windows 8 或 Windows 8.1，開放 Internet Explorer。  
  
2.  在 [位址列中，輸入**http://<yourservername\>/connect/default.aspx?get=caroot.cer**，然後按 Enter 鍵。  
  
3.  當系統提示您安裝 caroot.cer 憑證時，按一下**開放**。  
  
4.  在**憑證**對話方塊中，按**安裝憑證**。  
  
5.  在憑證匯入精靈中，選取 [**本機**做為儲存位置。  
  
6.  選取 [**將所有憑證都放在市集中下列**，然後瀏覽至**受信任的根憑證授權單位**的位置。  
  
7.  請依照下列指示完成精靈。  
  
    > [!NOTE]
    >  憑證安裝完成後，您可以登入我伺服器應用程式適用於 Windows 8 或 Windows 8.1，使用您的伺服器名稱和認證網路。
