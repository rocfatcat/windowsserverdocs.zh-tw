# <a name="step-8-run-the-windows-server-essentials-best-practices-analyzer"></a>步驟 8：執行 Windows Server Essentials 最佳做法分析程式

>適用於：Windows Server 2016 Essentials、 Windows Server 2012 R2 Essentials 中，Windows Server 2012 Essentials

當您完成您的設定和資料移轉到 Windows Server Essentials 時，您應該在目的地伺服器上執行 Windows Server Essentials Best Practices Analyzer (BPA)。 BPA 會檢查執行 Windows Server Essentials，並提供一份報告，描述問題並提供解決問題的建議的伺服器。 適用於 Windows Server Essentials 的產品支援單位所開發的建議。  
  
### <a name="to-run-the--windows-server-essentials-bpa-on-the-destination-server"></a>若要在目的地伺服器上執行 Windows Server Essentials BPA  
  
1.  以系統管理員身分登入目的地伺服器，然後開啟伺服器管理員。  
  
2.  在 [伺服器管理員] 中，按一下 [Windows Server Essentials 體驗] 索引標籤。  
  
3.  在詳細資料窗格中，選取 [最佳做法分析程式] 。  
  
4.  按一下 [工作]，然後按一下 [開始 BPA 掃描]。  
  
5.  檢閱每個 BPA 訊息，並依照指示來解決問題 (如有必要)。  
  
## <a name="next-steps"></a>後續步驟  
 您已執行 Windows Server Essentials Best Practices Analyzer。 移轉至 Windows Server Essentials 現在已完成的。
