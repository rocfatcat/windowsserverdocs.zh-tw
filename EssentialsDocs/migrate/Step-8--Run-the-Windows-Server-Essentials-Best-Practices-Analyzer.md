# <a name="step-8-run-the-windows-server-essentials-best-practices-analyzer"></a>步驟 8： 執行 Windows Server Essentials 最佳做法分析

>適用於：Windows Server 2016 Essentials 程式集 Windows Server 2012 R2、Windows Server 2012 程式集

將您的設定和資料移轉到 Windows Server Essentials 完成後，您應該目的伺服器上執行 Windows Server Essentials 最佳做法分析 (BPA)。 BPA 會檢查您執行的 Windows Server Essentials 並提供的報告描述問題，並提供建議解析他們的伺服器。 建議的 product 支援 Windows Server Essentials 的組織所開發。  
  
### <a name="to-run-the--windows-server-essentials-bpa-on-the-destination-server"></a>若要執行 Windows Server Essentials BPA 目的伺服器  
  
1.  目的地伺服器以系統管理員的身分登入，然後打開伺服器管理員。  
  
2.  在伺服器管理員中，按一下**Windows Server Essentials 體驗**索引標籤。  
  
3.  在詳細資料窗格中，選取 [**最佳做法分析**。  
  
4.  按一下**工作**，然後按**開始 BPA 掃描**。  
  
5.  檢視每個 BPA 訊息，並依照指示修正的相關問題，如有需要。  
  
## <a name="next-steps"></a>後續步驟  
 您已經執行 Windows Server Essentials 最佳做法分析。 移轉到 Windows Server Essentials 現在已完成。
