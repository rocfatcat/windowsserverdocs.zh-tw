# <a name="getting-started-with-the-windows-server-essentials-adk"></a>Windows Server Essentials ADK 快速入門

>適用於：Windows Server 2016 Essentials、 Windows Server 2012 R2 Essentials 中，Windows Server 2012 Essentials

評定及部署套件 (ADK) 的 Windows Server Essentials 伺服器軟體 ADK 為基礎的 Windows Server® 2012年作業系統，因此，文件 ADK 也適用於此處，除非另有說明。 此處的資訊是特別針對想要為客戶預先安裝作業系統的合作夥伴所提供。
