# <a name="install-add-ins"></a>安裝 Add-Ins

>適用於：Windows Server 2016 Essentials 程式集 Windows Server 2012 R2、Windows Server 2012 程式集

您可以加入增益集所有伺服器或 client 電腦上建立映像前進行安裝。 這些增益集會再自動包含所有電腦都複製使用該映像。 您可以安裝增益集執行的.wssx 檔案或您可以依照下列指導方針中的安裝個人增益集檔案[SDK 文件，](https://go.microsoft.com/fwlink/?LinkID=248648)針對每種類型的增益集。 如果您使用.wssx 檔案來進行安裝，增益集可以解除安裝透過管理員增益集。 如果您安裝的個人檔案，增益集不被管理增益集管理員。  
  
> [!NOTE]
>  （包括儀表板索引標籤和健康通知） 的伺服器下載或安裝的任何軟體應該會有當地語系化的介面，可符合語言的作業系統已安裝在伺服器。  
  
#### <a name="to-install-an-add-in"></a>若要安裝的增益集  
  
1.  （選擇性）如果您使用.wssx 檔案安裝增益集，請完成下列步驟：  
  
    1.  儲存 < AddinName\ >.wssx 參考電腦上的檔案。  
  
    2.  按兩下 [打開增益集安裝精靈中的.wssx 檔案。  
  
    3.  依照精靈中的指示，完成安裝。  
  
2.  （選擇性）所定義在每種類型的增益集 SDK 安裝個人增益集檔案的適當位置。
