# <a name="prepare-the-technician-computer"></a>準備技術電腦

>適用於：Windows Server 2016 Essentials 程式集 Windows Server 2012 R2、Windows Server 2012 程式集

技術電腦包含的工具和文件中的 Windows 評定及部署套件 (ADK)，而且您自訂的作業系統，您要部署映像的位置。 完成下列準備技術電腦。  
  
-   Windows 7、Windows 8 或 Windows 10 的電腦準備就緒。  
  
-   執行 Windows 7、Windows 8 或 Windows 10 的電腦上安裝 Windows ADK [Microsoft 網站](https://go.microsoft.com/fwlink/?LinkID=248647)。
