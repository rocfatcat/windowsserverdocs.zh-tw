# <a name="commands-by-server-role"></a>依伺服器角色的命令

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

伺服器角色說明伺服器的主要功能。 系統管理員可以選擇讓整部伺服器專屬於一個角色，或在單一電腦上安裝多個伺服器角色和子角色。 每個角色可能包括其他的命令列工具，安裝為角色的一部分。 下列主題提供每個伺服器角色相關聯的命令清單。

-   [列印命令參考資料](print-command-reference.md)
-   [如需網路檔案系統命令參考的服務](services-for-network-file-system-command-reference.md)
-   [遠端桌面服務&#40;終端機服務&#41;命令參考](remote-desktop-services-terminal-services-command-reference.md)
-   [Windows Server Backup 命令參考資料](windows-server-backup-command-reference.md)
