# <a name="bitsadmin-cache-and-help"></a>bitsadmin 快取與協助

顯示快**取參數的**命令列使用方式。

## <a name="syntax"></a>語法

```
bitsadmin /cache /help
```

## <a name="examples"></a><a name=BKMK_examples></a>典型

下列命令會顯示快**取參數的**命令列說明。

```
bitsadmin /cache /help
```
