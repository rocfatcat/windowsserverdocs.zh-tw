# <a name="fveupdate"></a>fveupdate



FveUpdate 是內部使用的工具，當電腦從 Windows Vista 或 Windows Server 2008 升級至 Windows 7 或 Windows Server 2008 R2 時，會由安裝程式執行。 它會將與 BitLocker 相關聯的中繼資料更新為最新版本。 此工具無法獨立執行。
