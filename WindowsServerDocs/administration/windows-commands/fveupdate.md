# <a name="fveupdate"></a>fveupdate



FveUpdate 是在內部使用的工具，在電腦從 Windows Vista 或 Windows Server 2008 升級至 Windows 7 或 Windows Server 2008 R2 時，會將安裝程式執行。 它會更新為最新版本，與 BitLocker 相關聯的中繼資料。 此工具無法獨立執行。
