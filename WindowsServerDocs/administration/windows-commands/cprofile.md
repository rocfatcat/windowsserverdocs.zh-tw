# <a name="cprofile"></a>cprofile

Cprofile 已被取代，在未來的 Windows 版本中不保證會受到支援。

Cprofile .exe：清理設定檔。 此工具組含在 Windows Server 2003 Web edition 以外的所有 Windows Server 2003 作業系統中。 如需詳細資訊[，請參閱終端機服務工具和設定](https://technet.microsoft.com/library/cc776289(v=ws.10).aspx)。
