Cprofile 已被取代，在未來的 Windows 版本中不保證會受到支援。

終端機服務已更新為遠端桌面服務。 如需詳細資訊，請參閱[歡迎使用遠端桌面服務](https://docs.microsoft.com/windows-server/remote/remote-desktop-services/welcome-to-rds)。
