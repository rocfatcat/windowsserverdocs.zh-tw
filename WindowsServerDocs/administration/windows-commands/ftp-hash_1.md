# <a name="ftp-hash1"></a>ftp: hash_1

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

切換數字符號 （#） 列印的每個資料區塊傳輸。   
## <a name="syntax"></a>語法  
```  
hash  
```  
### <a name="parameters"></a>參數  
無  
## <a name="remarks"></a>備註  
- 資料區塊的大小為 2048 個位元組。  
- 根據預設，**雜湊**已關閉。  
  ## <a name="BKMK_Examples"></a>範例  
  切換數字符號 （#） 所列印的每個資料區塊傳輸。  
  ```  
  hash  
  ```
