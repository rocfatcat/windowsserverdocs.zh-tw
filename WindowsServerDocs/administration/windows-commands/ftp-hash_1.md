# <a name="ftp-hash_1"></a>ftp： hash_1

> 適用于： Windows Server （半年通道）、Windows Server 2019、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

針對每個傳送的資料區塊，切換數位記號（#）列印。   
## <a name="syntax"></a>語法  
```  
hash  
```  
#### <a name="parameters"></a>參數  
無  
## <a name="remarks"></a>備註  
- 資料區塊的大小為2048個位元組。  
- 根據預設，**雜湊**為 off。  
  ## <a name="examples"></a>範例  
  針對每個傳送的資料區塊，切換數位記號（#）列印。  
  ```  
  hash  
  ```
