# <a name="ftp-delete"></a>ftp： delete

>適用於：Windows Server （半年通道）、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

刪除遠端電腦上的檔案。   
## <a name="syntax"></a>語法  
```  
delete <remoteFile>  
```  
### <a name="parameters"></a>參數  

|  參數   |          描述          |
|--------------|-------------------------------|
| <remoteFile> | 指定要刪除的檔案。 |

## <a name="BKMK_Examples"></a>典型  
刪除遠端電腦上的 file test.txt。  
```  
delete test.txt  
```
