# <a name="vol"></a>vol



如果有的話，會顯示磁碟區標籤和序號。  如果未指定參數，使用**vol**會顯示目前的磁碟機的資訊。

## <a name="syntax"></a>語法

```
vol [<Drive>:]
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|\<磁碟機 >:|指定包含您要顯示的磁碟區標籤和序號的磁碟的磁碟機。|
|/?|在命令提示字元顯示說明。|
