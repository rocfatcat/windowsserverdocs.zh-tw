# <a name="vol"></a>vol



顯示磁片區標籤和序號（如果有的話）。  如果在沒有參數的情況下使用， **vol**會顯示目前磁片磁碟機的資訊。

## <a name="syntax"></a>語法

```
vol [<Drive>:]
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|@no__t 0Drive >：|指定包含您要顯示其磁片區標籤和序號的磁片磁碟機。|
|/?|在命令提示字元顯示說明。|
