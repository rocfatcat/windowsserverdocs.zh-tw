# <a name="bitsadmin-peers-and-help"></a>bitsadmin peers and help

顯示 **/peers**參數的命令列使用方式。

## <a name="syntax"></a>語法

```
bitsadmin /peers /help
```

## <a name="examples"></a>範例

若要顯示 **/peers**參數的命令列使用方式：

```
bitsadmin /peers /help
```
