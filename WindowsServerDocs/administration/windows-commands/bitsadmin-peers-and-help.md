# <a name="bitsadmin-peers-and-help"></a>bitsadmin 對等和協助

顯示 **/peers**參數的命令列使用方式。

## <a name="syntax"></a>語法

```
bitsadmin /peers /help
```

## <a name="examples"></a><a name=BKMK_examples></a>典型

下列命令會顯示 **/peers**參數的命令列說明。

```
C:\>bitsadmin /peers /help
```
