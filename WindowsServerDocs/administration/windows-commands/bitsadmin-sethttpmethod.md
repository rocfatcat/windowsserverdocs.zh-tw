# <a name="bitsadmin-sethttpmethod"></a>bitsadmin sethttpmethod

設定要使用的 HTTP 動詞命令。

## <a name="syntax"></a>語法

```
bitsadmin /GetHttpMethod <Job> <HTTPMethod>
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|Job|作業的顯示名稱或 GUID|
|HTTPMethod|若要使用的 HTTP 指令動詞|
