# <a name="bitsadmin-sethttpmethod"></a>bitsadmin setHTTPmethod

設定要使用的 HTTP 指令動詞。

## <a name="syntax"></a>語法

```
bitsadmin /GetHttpMethod <Job> <HTTPMethod>
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|Job|作業的顯示名稱或 GUID|
|HTTPMethod|要使用的 HTTP 指令動詞|
