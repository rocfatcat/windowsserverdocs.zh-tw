# <a name="bitsadmin-cache-and-clear"></a>bitsadmin 快取和清除

清除本機快取。

## <a name="syntax"></a>語法

```
bitsadmin /cache /clear
```

## <a name="examples"></a><a name=BKMK_examples></a>典型

下列範例會清除本機快取。

```
C:\>bitsadmin /cache /clear
```
