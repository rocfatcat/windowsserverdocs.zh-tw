# <a name="bitsadmin-cache-and-clear"></a>bitsadmin 快取，然後清除



清除本機快取。

## <a name="syntax"></a>語法

```
bitsadmin /Cache /Clear 
```

## <a name="BKMK_examples"></a>範例

下列範例會清除本機快取。
```
C:\>bitsadmin /Cache /Clear 
```
