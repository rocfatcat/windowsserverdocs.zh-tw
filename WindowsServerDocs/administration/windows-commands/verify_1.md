# <a name="verify"></a>驗證



告訴**cmd** ，是否要確認您的檔案已正確寫入至磁片。 如果使用時不含參數， **verify**會顯示目前的設定。



## <a name="syntax"></a>語法

```
verify [on | off]
```

### <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|[ \| 關閉]|將 [**驗證**] 設定設為 [開啟] 或 [關閉]。|
|/?|在命令提示字元顯示說明。|

## <a name="examples"></a>範例

若要顯示 [目前的**驗證**] 設定，請輸入：
```
verify
```
若要開啟 [**驗證**] 設定，請輸入：
```
Verify on
```
