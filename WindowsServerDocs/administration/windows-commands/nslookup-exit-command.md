# <a name="nslookup-exit"></a>nslookup/exit

結束 nslookup 命令列工具。

## <a name="syntax"></a>語法

```
nslookup /exit
```

### <a name="parameters"></a>參數

| 參數 | 說明 |
| --------- | ----------- |
| /? | 在命令提示字元顯示說明。 |
| /help | 在命令提示字元顯示說明。 |
