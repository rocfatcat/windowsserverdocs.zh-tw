# <a name="nslookup-set-debug"></a>nslookup set debug



開啟或關閉偵錯模式。

## <a name="syntax"></a>語法

```
set [no]debug
```

## <a name="parameters"></a>參數

|  參數  |                         描述                          |
|-------------|--------------------------------------------------------------|
| **nodebug** | 關閉偵錯模式。 預設語法為**nodebug**。 |
|  **debug**  |                   開啟 [偵錯模式]。                   |
|  \* @ no__t-1 {說明  |                            ?}\* @ no__t-1                            |

## <a name="remarks"></a>備註

-   開啟 [偵錯模式] 時，會列印有關傳送到伺服器的封包和產生的答案的詳細資訊。
