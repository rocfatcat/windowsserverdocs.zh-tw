# <a name="nslookup-set-debug"></a>nslookup set debug



開啟或關閉，請開啟 [偵錯] 模式。

## <a name="syntax"></a>語法

```
set [no]debug
```

## <a name="parameters"></a>參數

|  參數  |                         描述                          |
|-------------|--------------------------------------------------------------|
| **nodebug** | 關閉偵錯 模式。 預設語法是**nodebug**。 |
|  **debug**  |                   開啟偵錯模式。                   |
|  \*\*{help  |                            ?}\*\*                            |

## <a name="remarks"></a>備註

-   偵錯模式中開啟，詳細的資訊會列印到伺服器，以及產生的回應傳送的封包。
