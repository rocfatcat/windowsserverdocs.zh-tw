# <a name="nslookup-set-debug"></a>nslookup set debug

開啟或關閉偵錯模式。

## <a name="syntax"></a>語法

```
set [no]debug
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
| ---------- | ---------- |
| nodebug | 關閉偵測模式。 這是預設值。 |
| debug | 開啟偵測模式。 藉由開啟的偵測模式，您可以查看傳送至伺服器之封包的詳細資訊，以及所產生的答案。 |
| /? | 在命令提示字元顯示說明。 |
| /help | 在命令提示字元顯示說明。 |
