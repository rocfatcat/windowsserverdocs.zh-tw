# <a name="nslookup-set-debug"></a>nslookup set debug

開啟或關閉偵錯模式。

## <a name="syntax"></a>語法

```
set [no]debug
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
| ---------- | ---------- |
| nodebug | 關閉偵錯模式。 這是預設值。 |
| debug | 開啟 [偵錯模式]。 藉由開啟的「偵錯工具模式」，您可以查看傳送至伺服器的封包和所產生答案的詳細資訊。 |
| /? | 在命令提示字元顯示說明。 |
| /help | 在命令提示字元顯示說明。 |
