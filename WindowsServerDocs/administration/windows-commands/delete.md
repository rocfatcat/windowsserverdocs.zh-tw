# <a name="delete"></a>[刪除]



刪除資料分割或磁碟區。 它也會從磁碟清單刪除動態磁碟。

## <a name="syntax"></a>語法

```
delete disk
delete partition
delete volume
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|[刪除磁碟](delete-disk.md)|從磁碟清單刪除遺失的動態磁碟。|
|[刪除磁碟分割](delete-partition.md)|刪除磁碟分割。|
|[刪除磁碟區](delete-volume.md)|刪除磁碟區。|
