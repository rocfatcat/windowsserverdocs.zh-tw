# <a name="delete"></a>delete

刪除磁碟分割或磁片區。 它也會從磁片清單中刪除動態磁碟。

## <a name="syntax"></a>語法

```
delete disk
delete partition
delete shadows
delete volume
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
|---------- | ----------- |
| [刪除磁片](delete-disk.md) | 從磁片清單中刪除遺失的動態磁碟。 |
| [刪除分割區](delete-partition.md) | 刪除資料分割。 |
| [刪除陰影](delete-shadows.md) | 刪除陰影複製。 |
| [刪除磁片區](delete-volume.md) | 刪除磁碟區。 |
