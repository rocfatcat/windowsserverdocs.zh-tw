# <a name="bitsadmin-peers"></a>bitsadmin 對等

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

```
bitsadmin /Peers /help
bitsadmin /Peers /Discover
bitsadmin /Peers /Clear
bitsadmin /Peers /list
```
## <a name="parameters"></a>參數
|參數|描述|
|-------|--------|
|[bitsadmin 同儕和說明](bitsadmin-peers-and-help.md)|顯示 /Peers 參數的命令列使用方式。|
|[bitsadmin 讓對等互連，並探索](bitsadmin-peers-and-discover.md)|再次探索對等。|
|[bitsadmin 同儕和清除](bitsadmin-peers-and-clear.md)|清除對等電腦清單。|
|[bitsadmin 同儕和清單](bitsadmin-peers-and-list.md)|列出所有對等。|
