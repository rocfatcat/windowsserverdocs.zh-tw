# <a name="qappsrv"></a>qappsrv

> 適用于： Windows Server (半年通道) 、Windows Server 2019、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

顯示網路上所有遠端桌面工作階段主機伺服器的清單。 若要瞭解最新版本的新功能，請參閱 [Windows Server 遠端桌面服務中的新功能](/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/dn283323(v=ws.11))。

> [!NOTE]
> 此命令與 [query termserver 命令](query-termserver.md)相同。
