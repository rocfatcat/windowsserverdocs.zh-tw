# <a name="online"></a>線上



為線上狀態，會採用離線的磁碟區。

## <a name="syntax"></a>語法

```
Offline disk
Offline volume
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|[離線的磁碟](offline-disk.md)|需要具有焦點的離線磁碟線上狀態。|
|[磁碟區離線](offline-volume.md)|需要具有焦點的離線磁碟區為線上狀態。|
