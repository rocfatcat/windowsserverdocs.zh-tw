# <a name="online"></a>online



使離線磁片或磁片區處於線上狀態。

## <a name="syntax"></a>語法

```
Offline disk
Offline volume
```

### <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|[離線磁片](offline-disk.md)|將具有焦點的離線磁片帶到線上狀態。|
|[離線磁片區](offline-volume.md)|將具有焦點的離線磁片區帶到線上狀態。|
