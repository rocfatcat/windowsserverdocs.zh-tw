# <a name="detail-disk"></a>detail disk

顯示所選磁碟的內容以及該磁碟上磁碟區。 開始之前，您必須先選取磁片，此作業才會成功。 使用 [ [選取磁片](select-disk.md) ] 命令來選取磁片，並將焦點移至該磁片。 如果您選取虛擬硬碟 (VHD) ，此命令會將磁片的匯流排類型顯示為 *虛擬*。

## <a name="syntax"></a>語法

```
detail disk
```

## <a name="examples"></a>範例

若要查看所選磁片的屬性，以及磁片中磁片區的相關資訊，請輸入：

```
detail disk
```
