# <a name="dnscmd"></a>Dnscmd

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

管理 DNS 伺服器的命令列介面。 此公用程式可用於指令碼批次檔可協助自動化例行的 DNS 管理工作，或在您網路上執行簡單的自動的安裝和設定新的 DNS 伺服器。  
## <a name="syntax"></a>語法  
```  
dnscmd <ServerName> <command> [<command parameters>]  
```  
## <a name="parameters"></a>參數  

|  參數   |                         描述                          |
|--------------|--------------------------------------------------------------|
| <ServerName> | 遠端或本機 DNS 伺服器 IP 位址或主機名稱。 |

### <a name="BKMK_1"></a>dnscmd /ageallrecords  
設定在指定的區域或節點的 DNS 伺服器上的資源記錄的時間戳記的目前時間。  
#### <a name="syntax"></a>語法  
```  
dnscmd [<ServerName>] /ageallrecords <ZoneName>[<NodeName>] | [/tree]|[/f]  
```

FQDN<br/>-的節點 （以句點 （.） 結尾的名稱） 的 FQDN<br/>-單一標籤相對於區域根目錄的名稱<br/>/tree<br/>指定所有的子節點也會收到的時間戳記。<br/>/f<br/>執行命令，而不要求確認。<br/># # # 註解<br/>- **Ageallrecords**命令適用於 DNS 的目前版本和舊版中的過時及清除功能不支援的 DNS 之間的回溯相容性。
