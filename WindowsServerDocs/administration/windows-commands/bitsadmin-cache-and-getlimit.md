# <a name="bitsadmin-cache-and-getlimit"></a>bitsadmin 快取和 getlimit



擷取的快取限制。

## <a name="syntax"></a>語法

```
bitsadmin /Cache /GetLimit 
```

## <a name="BKMK_examples"></a>範例

下列範例會擷取快取限制...
```
C:\>bitsadmin /Cache /GetLimit 
```
