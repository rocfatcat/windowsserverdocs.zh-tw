# <a name="bitsadmin-cache-and-getlimit"></a>bitsadmin cache 和 getlimit

抓取快取限制。

## <a name="syntax"></a>語法

```
bitsadmin /cache /getlimit
```

## <a name="examples"></a><a name=BKMK_examples></a>典型

下列範例會抓取快取限制。

```
C:\>bitsadmin /cache /getlimit
```
