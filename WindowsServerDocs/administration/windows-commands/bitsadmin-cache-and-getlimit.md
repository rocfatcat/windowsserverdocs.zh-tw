# <a name="bitsadmin-cache-and-getlimit"></a>bitsadmin cache and getlimit

抓取快取限制。

## <a name="syntax"></a>語法

```
bitsadmin /cache /getlimit
```

## <a name="examples"></a>範例

若要取得快取限制：

```
bitsadmin /cache /getlimit
```
