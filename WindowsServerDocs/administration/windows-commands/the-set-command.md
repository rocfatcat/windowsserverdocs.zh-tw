# <a name="the-set-command"></a>Set 命令

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

設定 Windows 部署服務伺服器、 預先設置的電腦、 映像、 映像群組和傳輸伺服器的屬性和屬性。
## <a name="subcommands"></a>子命令
|子命令|描述|
|-------|--------|
|[子命令： 設定裝置](subcommand-set-device.md)|變更預先設置電腦的屬性。 預先設置的電腦是已連結到 active directory (AD DS) 的網域伺服器的電腦帳戶物件的電腦。 預先設置的用戶端也稱為已知的電腦。|
|[子命令： 設定影像](subcommand-set-image.md)|變更現有的映像的屬性。|
|[子命令： 集 ImageGroup](subcommand-set-imagegroup.md)|變更現有的映像群組的屬性。|
|[Subcommand: set-Server](subcommand-set-server.md)|設定 Windows 部署服務伺服器的設定。|
|[Subcommand: set-TransportServer](subcommand-set-transportserver.md)|設定傳輸伺服器的設定。|
|[Subcommand: set-DriverPackage](subcommand-set-driverpackage.md)|重新命名及/或啟用/停用伺服器上的驅動程式套件。|
|[Subcommand: set-DriverGroup](subcommand-set-drivergroup.md)|在伺服器上設定現有的驅動程式群組的內容。|
|[Subcommand: set-DriverGroupFilter](subcommand-set-drivergroupfilter.md)|從新增或移除現有的驅動程式群組篩選器驅動程式群組。|
