# <a name="bitsadmin-peers-and-list"></a>bitsadmin 對等和清單



列出所有對等。

## <a name="syntax"></a>語法

```
bitsadmin /Peers /List 
```

## <a name="BKMK_examples"></a>典型

下列範例會列出所有對等。
```
C:\>bitsadmin /Peers /List
```
