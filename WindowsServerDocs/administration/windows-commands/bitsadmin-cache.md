#<a name="bitsadmin-cache"></a>bitsadmin 快取

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

## <a name="syntax"></a>語法

```
bitsadmin /Cache /help
bitsadmin /Cache /Clear
bitsadmin /Cache /delete
bitsadmin /Cache /deleteURL
bitsadmin /Cache /list
bitsadmin /Cache /Info
bitsadmin /Cache /GetLimit
bitsadmin /Cache /SetLimit
bitsadmin /Cache /GetExpirationtime
bitsadmin /Cache /SetExpirationtime
```

|[bitsadmin 快取和 delete](bitsadmin-cache-and-delete.md)|刪除快取項目。|
|[bitsadmin 快取和 deleteurl](bitsadmin-cache-and-deleteurl.md)|刪除所有的快取項目，為指定的 URL。|
|[bitsadmin 快取和清單](bitsadmin-cache-and-list.md)|列出所有快取項目。|
|[bitsadmin 快取和資訊](bitsadmin-cache-and-info.md)|傾印的特定快取項目。|
|[bitsadmin 快取和 getlimit](bitsadmin-cache-and-getlimit.md)|擷取的快取限制。|
