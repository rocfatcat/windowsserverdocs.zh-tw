# <a name="gettype"></a>gettype

**Gettype**命令已被取代，並不保證在未來的 Windows 版本中受到支援。

此工具組含在 Windows Server 2003 中。 如需詳細資訊，請參閱[gettype](https://docs.microsoft.com/previous-versions/orphan-topics/ws.10/cc773104(v=ws.10))。
