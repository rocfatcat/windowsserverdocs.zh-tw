# <a name="ftp-glob_1"></a>ftp： glob_1

>適用於：Windows Server （半年通道）、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

切換本機檔案名的萬用字元展開。   
## <a name="syntax"></a>語法  
```  
glob  
```  
### <a name="parameters"></a>參數  
無  
## <a name="remarks"></a>備註  
- 根據預設，萬用字元是 on。  
- 萬用字元允許在本機檔案或路徑名稱中使用星號（*）和問號（？）做為萬用字元。  
  ## <a name="BKMK_Examples"></a>典型  
  切換本機檔案名的萬用字元展開。  
  ```  
  glob  
  ```
