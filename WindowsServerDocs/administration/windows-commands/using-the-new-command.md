# <a name="using-the-new-command"></a>使用新的命令

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

建立擷取和探索映像、 多點傳送的傳輸，以及命名空間。
## <a name="subcommands"></a>子命令
|子命令|描述|
|-------|--------|
|[使用新 CaptureImage 命令](using-the-new-captureimage-command.md)|從現有的開機映像中建立新的擷取映像。|
|[使用新 DiscoverImage 命令](using-the-new-discoverimage-command.md)|從現有的開機映像中建立新的探索映像。|
|[使用新 MulticastTransmission 命令](using-the-new-multicasttransmission-command.md)|建立新的多點傳送的傳輸。|
|[使用新命名空間命令](using-the-new-namespace-command.md)|建立新的命名空間。|
