# <a name="bitsadmin-peercaching-and-help"></a>bitsadmin 對等與協助

顯示 **/peercaching**參數的命令列使用方式。

## <a name="syntax"></a>語法

```
bitsadmin /peercaching /help
```

## <a name="examples"></a><a name=BKMK_examples></a>典型

下列命令會**顯示對等切換參數**的命令列說明。

```
C:\> bitsadmin /peercaching /help
```
