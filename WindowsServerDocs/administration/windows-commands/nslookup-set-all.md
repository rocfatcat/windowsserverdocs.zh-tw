# <a name="nslookup-set-all"></a>nslookup set all



列印配置設定的目前值。

## <a name="syntax"></a>語法

```
set all 
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
|-----------|-------------|
|   {說明   |     ?}      |

## <a name="remarks"></a>備註

-   [**全部設定**] 也會列印預設伺服器和電腦（也就是主機）的相關資訊。
