# <a name="nslookup-set-all"></a>nslookup set all

輸出目前的設定值，包括預設伺服器和 (主機) 的電腦。

## <a name="syntax"></a>語法

```
set all
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
| --------- | ----------- |
| /? | 在命令提示字元顯示說明。 |
| /help | 在命令提示字元顯示說明。 |
