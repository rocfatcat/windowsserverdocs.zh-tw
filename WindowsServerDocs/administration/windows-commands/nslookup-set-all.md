# <a name="nslookup-set-all"></a>nslookup set all



列印目前的組態設定的值。

## <a name="syntax"></a>語法

```
set all 
```

## <a name="parameters"></a>參數

| 參數 | 描述 |
|-----------|-------------|
|   {說明   |     ?}      |

## <a name="remarks"></a>備註

-   **設定所有**也會列印為預設的伺服器和電腦 （也就是主機） 的相關資訊。
