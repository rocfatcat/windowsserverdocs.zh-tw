# <a name="ftp-close_1"></a>ftp： close_1

>適用於：Windows Server （半年通道）、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

結束遠端伺服器的 ftp 會話，並保留在**ftp >** 提示字元中。   
## <a name="syntax"></a>語法  
```  
close  
```  
### <a name="parameters"></a>參數  
無  
## <a name="BKMK_Examples"></a>典型  
結束遠端伺服器的 ftp 會話，並保留在**ftp >** 提示字元中。  
```  
close  
```
