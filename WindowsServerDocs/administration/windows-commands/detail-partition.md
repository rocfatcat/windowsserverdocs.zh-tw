# <a name="detail-partition"></a>detail partition

顯示所選磁碟分割的內容。 開始之前，您必須選取分割區，此作業才會成功。 使用 [ [選取資料分割](select-partition.md) ] 命令來選取分割區，並將焦點移至該分割區。

## <a name="syntax"></a>語法

```
detail partition
```

## <a name="examples"></a>範例

若要查看所選磁碟分割的屬性，請輸入：

```
detail partition
```
