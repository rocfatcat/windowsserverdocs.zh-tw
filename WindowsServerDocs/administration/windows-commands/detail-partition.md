# <a name="detail-partition"></a>detail partition

顯示所選磁碟分割的內容。

## <a name="syntax"></a>語法

```
detail partition
```

## <a name="remarks"></a>備註

-   必須選取一個磁碟分割，這項操作才能繼續。 使用 [**選取資料分割**] 命令來選取磁碟分割，並將焦點移至該資料分割。

## <a name="examples"></a>範例

若要查看選取之分割區的屬性，請輸入：
```
detail partition
```
