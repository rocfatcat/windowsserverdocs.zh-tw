# <a name="detail-partition"></a>詳細資料分割



顯示選取之分割區的屬性。

## <a name="syntax"></a>語法

```
detail partition
```

## <a name="remarks"></a>備註

-   必須選取分割區，此作業才會成功。 使用 [**選取資料分割**] 命令來選取磁碟分割，並將焦點移至該資料分割。

## <a name="BKMK_examples"></a>典型

若要查看選取之分割區的屬性，請輸入：
```
detail partition
```
