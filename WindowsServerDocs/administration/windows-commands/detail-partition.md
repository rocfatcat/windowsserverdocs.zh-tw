# <a name="detail-partition"></a>detail partition

顯示所選磁碟分割的內容。 開始之前，您必須選取資料分割，此作業才會成功。 使用 [[選取資料分割](select-partition.md)] 命令來選取磁碟分割，並將焦點移至該資料分割。

## <a name="syntax"></a>語法

```
detail partition
```

## <a name="examples"></a>範例

若要查看選取之分割區的屬性，請輸入：

```
detail partition
```
