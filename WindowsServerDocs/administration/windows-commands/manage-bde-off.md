# <a name="manage-bde-off"></a>管理 bde： 關閉



解密該磁碟機，並關閉 BitLocker。 解密完成時，會移除所有金鑰保護裝置。 如需如何使用此命令的範例，請參閱[範例](#BKMK_Examples)。

## <a name="syntax"></a>語法

```
manage-bde -off [<Volume>] [-computername <Name>] [{-?|/?}] [{-help|-h}]
```

### <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|\<磁碟區 >|磁碟機代號，後面接著冒號、 磁碟區 GUID 路徑或掛接的磁碟區。|
|-computername|指定 bde.exe 用以修改不同的電腦上的 BitLocker 保護。 您也可以使用 **-cn**為此命令縮寫版。|
|\<名稱 >|表示要修改 BitLocker 保護之電腦的名稱。 可接受的值包括電腦的 NetBIOS 名稱和電腦的 IP 位址。|
|-? 或 /？|顯示在命令提示字元中，簡短說明。|
|-help 或-h|顯示在命令提示字元完成說明。|

## <a name="BKMK_Examples"></a>範例

下列範例說明如何利用 **-關閉**命令來關閉 BitLocker 磁碟機 c。
```
manage-bde –off C:
```
