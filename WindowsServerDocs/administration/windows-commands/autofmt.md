# <a name="autofmt"></a>autofmt

啟動自動檔案系統格式公用程式，此公用程式會在從 Windows 修復主控台呼叫時格式化磁片磁碟機或磁碟分割。

> [!IMPORTANT]
> 您無法直接從命令列執行 **autofmt** 。
