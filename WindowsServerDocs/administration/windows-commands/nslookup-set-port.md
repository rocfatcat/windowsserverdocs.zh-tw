# <a name="nslookup-set-port"></a>nslookup set port

將預設的 TCP/UDP 網域名稱系統（DNS）名稱伺服器埠變更為指定的值。

## <a name="syntax"></a>語法

```
set port=<port>
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
| ---------- | ---------- |
| `<port>` | 指定預設 TCP/UDP DNS 名稱伺服器埠的新值。 預設通訊埠為**53**。 |
| /? | 在命令提示字元顯示說明。 |
| /help | 在命令提示字元顯示說明。 |
