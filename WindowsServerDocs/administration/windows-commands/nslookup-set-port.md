# <a name="nslookup-set-port"></a>nslookup set port



指定的值變更預設 TCP/UDP 網域名稱系統 (DNS) 名稱的伺服器連接埠。

## <a name="syntax"></a>語法

```
set port=<Port>
```

## <a name="parameters"></a>參數

| 參數 |                                          描述                                          |
|-----------|-----------------------------------------------------------------------------------------------|
|  \<連接埠 >  | 指定預設的 TCP/UDP DNS 名稱伺服器連接埠的新值。 預設連接埠是 53。 |
