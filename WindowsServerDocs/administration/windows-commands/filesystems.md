# <a name="filesystems"></a>檔案系統



顯示目前的檔案系統，具有焦點的磁碟區的相關資訊，並列出支援格式化磁碟區的檔案系統。

## <a name="syntax"></a>語法

```
filesystems
```

## <a name="remarks"></a>備註

-   這項作業成功，就必須選取磁碟區。 使用**選取磁碟區**命令來選取磁碟區，並將焦點移到它。
