# <a name="nslookup-finger-command"></a>nslookup finger 命令



連接目前電腦上的 finger 伺服器。

## <a name="syntax"></a>語法

```
finger [<UserName>] [{[>] <FileName>|[>>] <FileName>}]
```

### <a name="parameters"></a>參數

|  參數  |                                                                               描述                                                                               |
|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \<使用者名稱 > |                                                               指定要查閱之使用者的名稱。                                                                |
| \<FileName > | 指定用來儲存輸出的檔案名。 您可以使用大於（>）和 double 大於（> >）字元，以平常的方式重新導向輸出。 |
|    {說明    |                                                                                   ?}                                                                                    |
