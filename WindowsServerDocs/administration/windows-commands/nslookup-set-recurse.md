# <a name="nslookup-set-recurse"></a>nslookup set recurse



告訴網域名稱系統（DNS）名稱伺服器在沒有資訊的情況之下查詢其他伺服器。

## <a name="syntax"></a>語法

```
set [no]recurse
```

## <a name="parameters"></a>參數

|   參數   |                                                                  描述                                                                  |
|---------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| **norecurse** |                當網域名稱系統（DNS）名稱伺服器沒有資訊時，停止查詢其他伺服器。                |
|  **遞迴**  | 告訴網域名稱系統（DNS）名稱伺服器在沒有資訊的情況之下查詢其他伺服器。 預設語法為**遞迴**。 |
|     {說明     |                                                                      ?}                                                                       |
