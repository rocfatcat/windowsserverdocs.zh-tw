# <a name="netsh"></a>netsh



**Netsh**是命令列腳本公用程式，可讓您在本機或遠端顯示或修改目前執行中電腦的網路設定。

Windows Server®2003、Windows Server®2008和 Windows Server® 2008 R2 上的**netsh**命令之間有功能上的差異：
-   如需 Windows Server 2003 上**netsh**命令的詳細資訊，請參閱[netsh](https://technet.microsoft.com/library/cc779693(v=ws.10).aspx)。
-   如需有關 Windows Server 2008 和 Windows Server 2008 R2 的**netsh**命令的詳細資訊，請參閱[netsh 技術參考](https://technet.microsoft.com/library/cc754753(v=ws.10).aspx)。
