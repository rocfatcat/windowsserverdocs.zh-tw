# <a name="telnet-quit"></a>telnet： quit

>適用於：Windows Server (半年通道)、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

結束 telnet。   

## <a name="syntax"></a>語法  
```  
q[uit]  
```  
#### <a name="parameters"></a>參數  
無  
## <a name="examples"></a><a name=BKMK_Examples></a>典型  
結束 telnet。  
```  
q  
```
