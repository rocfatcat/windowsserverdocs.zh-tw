# <a name="offline"></a>離線

使線上磁片或磁片區進入離線狀態。

## <a name="syntax"></a>語法

```
offline disk
offline volume
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
| --------- | ----------- |
| [offline disk](offline-disk.md) | 將具有焦點的線上磁片帶到離線狀態。 |
| [offline volume](offline-volume.md) | 將具有焦點的線上磁片區帶到離線狀態。 |
