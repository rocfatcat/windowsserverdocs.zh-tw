# <a name="offline"></a>離線



會為離線狀態使用的線上磁碟區。

## <a name="syntax"></a>語法

```
Offline disk
Offline volume
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|[離線的磁碟](offline-disk.md)|需要具有焦點的線上磁碟為離線狀態。|
|[磁碟區離線](offline-volume.md)|需要具有焦點的線上磁碟區離線的狀態。|
