# <a name="pentnt"></a>pentnt

> 適用于： Windows Server 2003、Windows Server 2003 R2、Windows Server 2003 （含 SP1）、Windows Server 2003 （含 SP2）

>[!IMPORTANT]
> 此命令已被取代。

偵測浮點除法錯誤 (如果在 Pentium 晶片中出現) 、停用浮點硬體，然後開啟浮點模擬。 如需描述和使用方式資訊，請參閱[pentnt](/previous-versions/orphan-topics/ws.10/cc755868(v=ws.10))。
