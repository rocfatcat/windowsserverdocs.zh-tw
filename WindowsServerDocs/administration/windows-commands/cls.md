# <a name="cls"></a>cls

清除 [命令提示字元] 視窗。

## <a name="syntax"></a>語法

```
cls
```

### <a name="parameters"></a>參數

| 參數 | 說明 |
| --------- | ----------- |
| /? | 在命令提示字元顯示說明。 |

## <a name="examples"></a>範例

若要清除出現在 [命令提示字元] 視窗中並返回空白視窗的所有資訊，請輸入：

```
cls
```
