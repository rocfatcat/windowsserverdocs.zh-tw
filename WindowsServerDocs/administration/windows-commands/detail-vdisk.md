# <a name="detail-vdisk"></a>詳細資料 vdisk

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

顯示選取的虛擬硬碟的屬性\(VHD\)。  
  
## <a name="syntax"></a>語法  
  
```  
detail vdisk  
```  
  
## <a name="remarks"></a>備註  
  
-   VHD 必須選取此作業才會成功。 使用**選取 vdisk**命令來選取 vdisk，並將焦點移到它。  
  
## <a name="BKMK_examples"></a>範例  
若要查看有關所選取的 VHD 的詳細資訊，請輸入：  
  
```  
detail vdisk  
```
