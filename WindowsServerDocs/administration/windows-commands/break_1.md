# <a name="break"></a>break



設定或清除擴充的 CTRL + C MS-DOS 系統上檢查。 如果未指定參數，使用**中斷**顯示目前的設定。

> [!NOTE]
> 此命令已不再使用中。 只是為了保持與現有 MS-DOS 檔案相容才包含它，但是在命令列上它沒有效果，因為此功能是自動的。

## <a name="syntax"></a>語法

```
break=[on|off]
```

## <a name="remarks"></a>備註

如果命令延伸模組已啟用且正在執行 Windows 平台上，插入**中斷**批次檔命令進入硬式編碼的中斷點，如果正在偵錯的偵錯工具。
