# <a name="using-the-enable-command"></a>使用 enable 命令

> 適用于： Windows Server (半年通道) 、Windows Server 2019、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

啟用 Windows 部署服務的所有服務。

## <a name="subcommands"></a>子
|Subommand|描述|
|-------|--------|
|[使用 enable-Server 命令](using-the-enable-server-command.md)|啟用指定 Windows 部署服務 server (部署伺服器上的所有服務) 。|
|[使用 TransportServer 命令](using-the-enable-transportserver-command.md)|啟用指定之傳輸伺服器上的所有服務。|
