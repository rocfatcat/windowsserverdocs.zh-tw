# <a name="ftp-cd"></a>ftp： cd

>適用於：Windows Server （半年通道）、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

變更遠端電腦上的工作目錄。   
## <a name="syntax"></a>語法  
```  
cd <remotedirectory>  
```  
### <a name="parameters"></a>參數  

|     參數     |                                 描述                                 |
|-------------------|-----------------------------------------------------------------------------|
| <remotedirectory> | 指定您要變更的遠端電腦上的目錄。 |

## <a name="BKMK_Examples"></a>典型  
將遠端電腦上的目錄**變更為 [** 檔]。  
```  
cd Docs  
```  
將遠端電腦上的目錄變更為 [**可能**的影片]。  
```  
cd  May Videos  
```
