# <a name="ftp-cd"></a>ftp: cd

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

變更遠端電腦上的工作目錄。   
## <a name="syntax"></a>語法  
```  
cd <remotedirectory>  
```  
### <a name="parameters"></a>參數  
|參數|描述|  
|-------|--------|  
|<remotedirectory>|您要變更遠端電腦上指定的目錄。|  
## <a name="BKMK_Examples"></a>範例  
變更遠端電腦上的目錄**Docs**。  
```  
cd Docs  
```  
變更遠端電腦上的目錄**可能影片**。  
```  
cd  May Videos  
```
