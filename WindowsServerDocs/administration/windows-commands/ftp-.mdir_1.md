# <a name="ftp-rmdir1"></a>ftp: rmdir_1

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

刪除遠端目錄。   
## <a name="syntax"></a>語法  
```  
rmdir <directory>  
```  
### <a name="parameters"></a>參數  
|參數|描述|  
|-------|--------|  
|<directory>|指定要刪除的遠端目錄的名稱。|  
## <a name="BKMK_Examples"></a>範例  
刪除**圖片**遠端目錄。  
```  
rmdir Pictures  
```
