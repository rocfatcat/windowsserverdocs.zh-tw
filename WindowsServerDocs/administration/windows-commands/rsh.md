# <a name="rsh"></a>rsh



在執行 RSH 服務或精靈的遠端電腦上執行命令。 此命令已被取代。 您可以安裝以 UNIX 為基礎的應用程式使用 [新增功能精靈] 的子系統。 如需詳細資訊，請參閱 < [Windows Server 2008 的 UNIX 互通性元件](https://go.microsoft.com/fwlink/?LinkId=191835)羰 Microsoft 寍鯚。 安裝完成後，您可以開啟 C 殼層或 Korn 殼層的命令視窗並執行**rsh**。 如需詳細資訊，請輸入**man rsh**在 C 殼層或 Korn 殼層提示字元。
