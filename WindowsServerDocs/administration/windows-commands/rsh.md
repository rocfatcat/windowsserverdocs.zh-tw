# <a name="rsh"></a>rsh

在執行 RSH 服務或 daemon 的遠端電腦上執行命令。

> [!IMPORTANT]
> **Rexec**命令已被取代，不保證會在 Windows 中支援。

安裝適用于 UNIX 應用程式的子系統之後，您可以開啟 C Shell 或 Korn Shell 命令視窗並執行 **rsh**。 如需詳細資訊，請在 C Shell 或 Korn Shell 提示字元中輸入「 **男人 rsh** 」。
