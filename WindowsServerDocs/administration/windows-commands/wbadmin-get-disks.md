# <a name="wbadmin-get-disks"></a>wbadmin get disks



列出本機電腦目前上線的內部和外部磁片。

若要列出使用此子命令連線的磁片，您必須是**Backup Operators**群組或**Administrators**群組的成員，或者必須已被委派適當的許可權。 此外，您必須從提升許可權的命令提示字元執行**wbadmin** 。  (開啟已提升許可權的命令提示字元，以滑鼠右鍵按一下**命令提示**字元，然後按一下 [以**系統管理員身分執行**]。 ) 

## <a name="syntax"></a>語法

```
wbadmin get disks
```

### <a name="parameters"></a>參數

這個子命令沒有任何參數。
