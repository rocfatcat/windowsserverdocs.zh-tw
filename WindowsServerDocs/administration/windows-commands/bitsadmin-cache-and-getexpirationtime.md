# <a name="bitsadmin-cache-and-getexpirationtime"></a>bitsadmin 快取和 getexpirationtime



擷取快取到期時間。

## <a name="syntax"></a>語法

```
bitsadmin /Cache /GetExpirationTime 
```

## <a name="BKMK_examples"></a>範例

下列擷取快取到期時間。
```
C:\>bitsadmin /Cache /GetExpirationTime
```
