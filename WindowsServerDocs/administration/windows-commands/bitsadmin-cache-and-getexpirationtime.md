# <a name="bitsadmin-cache-and-getexpirationtime"></a>bitsadmin cache 和 getexpirationtime



抓取快取到期時間。

## <a name="syntax"></a>語法

```
bitsadmin /Cache /GetExpirationTime 
```

## <a name="BKMK_examples"></a>典型

以下會抓取快取到期時間。
```
C:\>bitsadmin /Cache /GetExpirationTime
```
