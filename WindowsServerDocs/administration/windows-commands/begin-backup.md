# <a name="begin-backup"></a>開始備份

> 適用目標︰Windows Server 2012 R2、Windows Server 2012

啟動完整備份會話。 此命令會覆寫預設的複本備份設定。

## <a name="syntax"></a>語法

```
begin backup
```
