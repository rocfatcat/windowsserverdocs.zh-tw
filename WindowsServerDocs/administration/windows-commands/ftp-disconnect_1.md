# <a name="ftp-disconnect_1"></a>ftp： disconnect_1

>適用於：Windows Server （半年通道）、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

中斷與遠端電腦的連線，並保留**ftp >** 提示字元。   
## <a name="syntax"></a>語法  
```  
disconnect  
```  
### <a name="parameters"></a>參數  
無  
## <a name="BKMK_Examples"></a>典型

```  
disconnect  
```
