# <a name="attributes"></a>屬性

顯示、設定或清除磁片或磁片區的屬性。

## <a name="syntax"></a>語法

```
attributes disk
attributes volume
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
| --------- | ----------- |
| [屬性磁片](attributes-disk.md) | 顯示、設定或清除磁片的屬性。 |
| [屬性磁片區](attributes-volume.md) | 顯示、設定或清除磁片區的屬性。 |
