# <a name="attributes"></a>屬性



顯示、 設定，或清除磁碟或磁碟區的屬性。

## <a name="syntax"></a>語法

```
Attributes disk
Attributes volume
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|[磁碟屬性](attributes-disk.md)|顯示、 設定，或清除磁碟的屬性。|
|[屬性的磁碟區](attributes-volume.md)|顯示、 設定，或清除磁碟區的屬性。|
