# <a name="set-verbose"></a>設定詳細資訊



指定是否要將詳細資訊輸出提供建立陰影複製期間。 如果未指定參數，使用**設定 verbose**在命令提示字元中顯示說明。

## <a name="syntax"></a>語法

```
set verbose {on | off}
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|{on | off}|開啟或關閉，請開啟詳細資訊模式。|

## <a name="remarks"></a>備註

-   如果已開啟的詳細資訊模式**設定**提供的寫入器包含或排除的詳細資料和中繼資料壓縮和擷取的詳細資料。
