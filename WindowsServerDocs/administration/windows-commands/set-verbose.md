# <a name="set-verbose"></a>設定詳細資訊



指定是否在建立陰影複製時提供詳細資訊輸出。 如果使用時不含參數， **set verbose**會在命令提示字元中顯示說明。

## <a name="syntax"></a>語法

```
set verbose {on | off}
```

## <a name="parameters"></a>參數

| 參數 | 描述 |
|-----------|-------------|
|    {開啟    |    停止     |

## <a name="remarks"></a>備註

-   如果 verbose 模式是 on，則**set**會提供寫入器包含或排除的詳細資料，以及中繼資料壓縮與解壓縮的詳細資訊。
