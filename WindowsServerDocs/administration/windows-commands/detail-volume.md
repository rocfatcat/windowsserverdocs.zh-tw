# <a name="detail-volume"></a>detail volume

顯示目前磁碟區所位於的磁碟。 開始之前，您必須選取磁片區，此作業才會成功。 使用 [ [選取磁片](select-volume.md) 區] 命令來選取磁片區，並將焦點移至該磁片區。 磁片區詳細資料不適用於唯讀磁片區，例如 DVD-ROM 或 CD-ROM 光碟機。

## <a name="syntax"></a>語法

```
detail volume
```

## <a name="examples"></a>範例

若要查看目前磁片區所在的所有磁片，請輸入：

```
detail volume
```
