# <a name="begin-restore"></a>begin restore

> 適用目標︰Windows Server 2012 R2、Windows Server 2012

啟動還原會話，併發出 **PreRestore** 事件給相關的寫入器。

## <a name="syntax"></a>語法

```
begin restore
```
