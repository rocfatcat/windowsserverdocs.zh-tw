# <a name="command-line-syntax-key"></a>命令列語法關鍵

下表說明用來指出命令列語法的表示法。

|表示法|描述|
|--------|-----------|
|不含括弧或大括弧的文字|必須依照顯示輸入的項目|
|\<角括弧內的文字 >|必須在其中提供值的保留區|
|[方括弧內的文字]|選擇性的項目|
|{大括弧內的文字}|必要的項目集；選擇一個|
|分隔號（\|）|互斥項目的分隔字元；選擇一個|
|省略符號 (…)|可重複的項目|
