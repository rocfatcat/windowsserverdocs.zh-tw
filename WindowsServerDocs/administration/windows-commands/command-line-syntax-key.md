# <a name="command-line-syntax-key"></a>命令列語法關鍵



下表描述用來表示命令列語法標記法。

|標記法|描述|
|--------|-----------|
|文字，而不需要方括號或大括號|您必須依照顯示輸入的項目|
|\<角括弧內的文字 >|您必須提供值的預留位置|
|[方括號內的文字]|選擇性的項目|
|{大括號內的文字}|一組必要的項目;選擇其中一個|
|分隔號 (\|)|分隔符號是互斥的項目;選擇其中一個|
|省略符號 （...）|可重複的項目|
