# <a name="command-line-syntax-key"></a>命令列語法索引鍵

下表說明用來指出命令列語法的表示法。

| 表示法 | 描述 |
| -------- | ----------- |
| 不含括弧或大括弧的文字 | 您必須輸入的專案，如下所示。 |
| `<Text inside angle brackets>` | 您必須為其提供值的預留位置。 |
| `[Text inside square brackets]` | 選擇性的項目 |
| `{Text inside braces}` | 必要專案的集合。 您必須選擇其中一個。 |
| 分隔號形圖 `(|)`| 互斥專案的分隔符號。 您必須選擇其中一個。 |
| 省略號 `(…)` | 可以重複並使用多次的專案。 |
