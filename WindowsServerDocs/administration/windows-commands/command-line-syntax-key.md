# <a name="command-line-syntax-key"></a>命令列語法關鍵



下表描述用來表示命令列語法的標記法。

|表示法|描述|
|--------|-----------|
|不含括弧或大括弧的文字|您必須輸入的專案, 如下所示|
|\<角括弧內的文字 >|您必須為其提供值的預留位置|
|[括弧內的文字]|選擇性的項目|
|{大括弧內的文字}|必要專案集;選擇一個|
|分隔號（\|）|互斥專案的分隔符號;選擇一個|
|省略號 (...)|可以重複的專案|
