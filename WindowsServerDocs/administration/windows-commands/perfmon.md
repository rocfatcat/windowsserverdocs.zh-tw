# <a name="perfmon"></a>perfmon

以特定獨立模式啟動 Windows 可靠性和效能監視器。

## <a name="syntax"></a>語法

```
perfmon </res|report|rel|sys>
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|/res|啟動資源檢視。|
|/report|啟動「系統診斷」資料收集器集合工具，並顯示結果的報告。|
|/rel|啟動可靠性監視器。|
|/sys|啟動效能監視器。|
