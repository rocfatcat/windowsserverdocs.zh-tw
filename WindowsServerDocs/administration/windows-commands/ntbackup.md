# <a name="ntbackup"></a>ntbackup



**Ntbackup**命令不適用於 Windows Vista 或 Windows Server 2008。 因此，您應該改用**wbadmin**命令和子命令來備份和還原您的電腦和檔案，從命令提示字元。

您無法復原您建立的備份**ntbackup**利用**wbadmin**。 不過，新版**ntbackup**是可供下載的 Windows Server 2008 和 Windows Vista 使用者想要復原使用他們所建立的備份**ntbackup**。 這個可下載的版本**ntbackup**可讓您執行只復原舊版備份，且無法用於執行 Windows Server 2008 或 Windows Vista 的電腦上建立新的備份。 若要下載此版本**ntbackup**，請參閱[ https://go.microsoft.com/fwlink/?LinkId=82917 ](https://go.microsoft.com/fwlink/?LinkId=82917)。
