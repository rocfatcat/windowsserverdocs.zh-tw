# <a name="mqtgsvc"></a>mqtgsvc



監視內送訊息的佇列和觸發程序的規則會評估為 true 時，執行動作，可執行檔或 COM 元件的形式。 如需如何使用訊息佇列觸發程序服務的範例，請參閱[訊息佇列觸發程序](https://go.microsoft.com/fwlink/?LinkId=248725)MSDN 上。

## <a name="syntax"></a>語法

```
Mqtgsvc.exe
```

### <a name="parameters"></a>參數

None
