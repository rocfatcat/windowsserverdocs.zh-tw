# <a name="mqtgsvc"></a>mqtgsvc



當觸發程式的規則評估為 true 時，會監視傳入訊息的佇列，並以可執行檔或 COM 元件的形式執行動作。 如需如何使用訊息佇列觸發程式服務的範例，請參閱 MSDN 上的[訊息佇列觸發](https://go.microsoft.com/fwlink/?LinkId=248725)程式。

## <a name="syntax"></a>語法

```
Mqtgsvc.exe
```

#### <a name="parameters"></a>參數

無
