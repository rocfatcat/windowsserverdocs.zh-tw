# <a name="mqsvc"></a>mqsvc

訊息佇列技術可讓在不同時間執行的應用程式，在可能暫時離線的異類網路和系統之間進行通訊。 訊息佇列提供有保證的訊息傳遞、有效率的路由、安全性及依優先順序傳訊的方式。 可用於執行非同步處理和同步處理訊息傳送案例的解決方案。 如需訊息佇列的詳細資訊，請參閱[訊息佇列（MSMQ）](/previous-versions/windows/desktop/legacy/ms711472(v=vs.85))。

## <a name="syntax"></a>Syntax

```
mqsvc.exe
```
