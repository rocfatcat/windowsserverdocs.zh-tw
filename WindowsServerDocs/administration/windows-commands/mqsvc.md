# <a name="mqsvc"></a>mqsvc



訊息佇列技術可讓在不同時間執行的應用程式跨越可能暫時離線的異質的網路和系統進行通訊。 訊息佇列會提供保證的訊息傳遞、 有效路由傳送、 安全性和優先順序型傳訊。 它可以用來實作非同步和同步傳訊案例的解決方案。 如需有關此命令的詳細資訊，請參閱 < [Message Queuing (MSMQ)](https://go.microsoft.com/fwlink/?LinkId=248723) MSDN 上。

## <a name="syntax"></a>語法

```
Mqsvc.exe
```

### <a name="parameters"></a>參數

None
