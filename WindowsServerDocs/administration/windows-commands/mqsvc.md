# <a name="mqsvc"></a>mqsvc



訊息佇列技術可讓在不同時間執行的應用程式，在可能暫時離線的異類網路和系統之間進行通訊。 訊息佇列提供有保證的訊息傳遞、有效率的路由、安全性及依優先順序傳訊的方式。 可用於執行非同步處理和同步處理訊息傳送案例的解決方案。 如需此命令的詳細資訊，請參閱 MSDN 上的[訊息佇列（MSMQ）](https://go.microsoft.com/fwlink/?LinkId=248723) 。

## <a name="syntax"></a>語法

```
Mqsvc.exe
```

#### <a name="parameters"></a>參數

None
