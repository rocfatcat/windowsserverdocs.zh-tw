# <a name="ftp-trace1"></a>ftp: trace_1



切換封包追蹤。

## <a name="syntax"></a>語法

```
trace
```

### <a name="parameters"></a>參數

無

## <a name="remarks"></a>備註

**追蹤**執行時，會顯示內部的 FTP 函式呼叫的一系列**ftp**命令。

## <a name="BKMK_Examples"></a>範例

切換開啟和關閉的追蹤。
```
trace
```
