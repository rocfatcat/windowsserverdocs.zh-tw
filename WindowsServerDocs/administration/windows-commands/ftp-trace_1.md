# <a name="ftp-trace_1"></a>ftp： trace_1



切換封包追蹤。

## <a name="syntax"></a>語法

```
trace
```

#### <a name="parameters"></a>參數

無

## <a name="remarks"></a>備註

**追蹤**會顯示執行**ftp**命令時的內部 FTP 函數呼叫系列。

## <a name="examples"></a><a name=BKMK_Examples></a>典型

切換開啟和關閉追蹤。
```
trace
```
