# <a name="bitsadmin-help"></a>bitsadmin help

顯示有關 bitsadmin 命令列參數和選項的說明相關資訊。

## <a name="syntax"></a>語法

```
bitsadmin /help | /?
```

## <a name="examples"></a><a name=BKMK_examples></a>典型

下列範例會抓取命令列說明。

```
C:\>bitsadmin /help
```
