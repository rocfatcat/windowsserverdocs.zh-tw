# <a name="bitsadmin-help"></a>bitsadmin help



顯示命令列使用方式。

## <a name="syntax"></a>語法

```
bitsadmin /help | /?
```

## <a name="BKMK_examples"></a>範例

下列範例會擷取命令列說明。
```
C:\>bitsadmin /help
```
