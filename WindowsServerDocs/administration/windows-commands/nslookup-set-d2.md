# <a name="nslookup-set-d2"></a>nslookup set d2

開啟或關閉 [詳細資訊] 偵錯模式。 列印每個封包的所有欄位。

## <a name="syntax"></a>語法

```
set [no]d2
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
| ---------- | ---------- |
| nod2 | 關閉詳細的偵測模式。 這是預設值。 |
| d2 | 開啟詳細的偵測模式。 |
| /? | 在命令提示字元顯示說明。 |
| /help | 在命令提示字元顯示說明。 |
