# <a name="nslookup-set-d2"></a>nslookup set d2



開啟或關閉，請開啟完整的偵錯模式。 列印每個封包的所有欄位。

## <a name="syntax"></a>語法

```
set [no]d2
```

## <a name="parameters"></a>參數

| 參數 |                             描述                              |
|-----------|----------------------------------------------------------------------|
| **nod2**  | 關閉地毯式偵錯 模式。 預設語法是**nod2**。 |
|  **d2**   |                 會開啟完整的偵錯模式。                  |
|   {說明   |                                  ?}                                  |
