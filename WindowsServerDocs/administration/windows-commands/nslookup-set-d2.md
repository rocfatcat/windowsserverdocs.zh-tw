# <a name="nslookup-set-d2"></a>nslookup set d2



開啟或關閉徹底的偵錯模式。 每個封包的所有欄位都會列印出來。

## <a name="syntax"></a>語法

```
set [no]d2
```

## <a name="parameters"></a>參數

| 參數 |                             描述                              |
|-----------|----------------------------------------------------------------------|
| **nod2**  | 關閉徹底的偵錯模式。 預設語法為**nod2**。 |
|  **d2**   |                 開啟徹底的偵錯模式。                  |
|   {說明   |                                  ?}                                  |
