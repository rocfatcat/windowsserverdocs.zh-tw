# <a name="list-shadows"></a>list shadows

列出系統上的持續性和現有的非持續陰影複製。

## <a name="syntax"></a>語法

```
list shadows {all | set <setID> | id <shadowID>}
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
| ---------- | ---------- |
| all | 列出所有陰影複製。 |
| 設置 `<setID>` | 列出屬於指定之陰影複製集識別碼的陰影複製。 |
| Id `<shadowID>` | 列出具有指定陰影複製識別碼的任何陰影複製。 |
