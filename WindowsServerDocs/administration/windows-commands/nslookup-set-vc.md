# <a name="nslookup-set-vc"></a>nslookup set vc



指定在將要求傳送至伺服器時，使用或不使用虛擬電路。

## <a name="syntax"></a>語法

```
set [no]vc
```

## <a name="parameters"></a>參數

| 參數 |                                              描述                                               |
|-----------|--------------------------------------------------------------------------------------------------------|
| **novc**  | 指定將要求傳送至伺服器時，不要使用虛擬電路。 預設值為**novc**。 |
|  **vc**   |             指定在傳送要求給伺服器時，一律使用虛擬電路。             |
|   {說明   |                                                   ?}                                                   |
