# <a name="nslookup-set-vc"></a>nslookup set vc



指定要使用或不使用虛擬電路時傳送要求到伺服器。

## <a name="syntax"></a>語法

```
set [no]vc
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|**novc**|指定不要使用虛擬電路將要求傳送到伺服器時。 預設值是**novc**。|
|**vc**|指定要將要求傳送到伺服器時，一律使用虛擬電路。|
|{說明 | ?}|顯示的簡短摘要**nslookup**子命令。|
