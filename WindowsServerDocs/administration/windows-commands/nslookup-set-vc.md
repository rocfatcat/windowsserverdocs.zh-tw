# <a name="nslookup-set-vc"></a>nslookup set vc

指定將要求傳送到伺服器時，是否要使用虛擬電路。

## <a name="syntax"></a>語法

```
set [no]vc
```

### <a name="parameters"></a>參數


| 參數 | 描述 |
| ---------- | ---------- |
| novc | 指定將要求傳送到伺服器時，不要使用虛擬電路。 這是預設值。 |
| Vc | 指定將要求傳送到伺服器時，一律使用虛擬電路。 |
| /? | 在命令提示字元顯示說明。 |
| /help | 在命令提示字元顯示說明。 |
