# <a name="dir"></a>dir



顯示目錄的檔案和子目錄清單。 如果使用時不含參數， **dir**會顯示磁片的磁片區標籤和序號，後面接著磁片上的目錄和檔案清單（包括其名稱，以及每個上次修改的日期和時間）。 針對檔案， **dir**會顯示名稱延伸模組和大小（以位元組為單位）。 **Dir**也會顯示所列的檔案和目錄總數、其累計大小，以及磁片上剩餘的可用空間（以位元組為單位）。

如需如何使用此命令的範例，請參閱[範例](#examples)。

## <a name="syntax"></a>語法

```
dir [<Drive>:][<Path>][<FileName>] [...] [/p] [/q] [/w] [/d] [/a[[:]<Attributes>]][/o[[:]<SortOrder>]] [/t[[:]<TimeField>]] [/s] [/b] [/l] [/n] [/x] [/c] [/4]
```

## <a name="remarks"></a>備註

- 若要使用多個*FileName*參數，請以空格、逗號或分號分隔每個檔案名。
- 您可以使用萬用字元（ **&#42;** 或 **？** ）來代表檔案名的一個或多個字元，並顯示檔案或子目錄的子集。

  **星號（\*）：** 使用星號取代任何字元字串，例如：  
  - **dir \*** 會列出目前目錄中副檔名開頭為 .txt 的所有檔案，例如 .txt、. txt1、. txt_old。
  - **dir read\*.txt**會列出目前目錄中開頭為「讀取」的所有檔案，以及副檔名開頭為 .txt 的檔案，例如 .txt、. txt1 或 txt_old。
  - **dir read @ no__t-1。 \\** * 列出目前目錄中所有副檔名開頭為「讀取」的所有檔案。

  星號萬用字元一律使用簡短的檔案名對應，因此您可能會收到非預期的結果。 例如，下列目錄包含兩個檔案（txt2 和 t97）： 
 
  ```
  C:\test>dir /x
  Volume in drive C has no label.
  Volume Serial Number is B86A-EF32
    
  Directory of C:\test
    
  11/30/2004  01:40 PM <DIR>  .
  11/30/2004  01:40 PM <DIR> ..
  11/30/2004  11:05 AM 0 T97B4~1.TXT t.txt2
  11/30/2004  01:16 PM 0 t97.txt
  ```  

  您可能會預期輸入**dir t97 @ no__t-1*** 會傳回 t97 檔案。 不過，鍵入**dir t97 @ no__t-1*** 會傳回這兩個檔案，因為星號萬用字元會使用其簡短名稱對應 T97B4 ~ 1 .txt 來比對 txt2 至 t97。 同樣地，輸入**del t97 @ no__t-1*** 會同時刪除這兩個檔案。

  **問號（？）：** 使用問號取代名稱中的單一字元。 例如，輸入**dir read???。txt**會列出目前目錄中副檔名開頭為「讀取」且最多三個字元的所有檔案。 這包括 Read .txt、Read1 .txt、Read12 .txt、Read123 .txt 和 Readme1，但不包含 Readme12 .txt。
- 指定檔顯示內容

  如果您在*屬性*中使用具有一個以上值的 **/a** ， **dir**只會顯示具有所有指定屬性之檔案的名稱。 例如，如果您使用 **/a**搭配**r**和 **-h**做為屬性（藉由使用 **/a： r-h**或 **/ar-h**）， **dir**只會顯示未隱藏之唯讀檔案的名稱。
- 指定檔案名排序

  如果您指定一個以上的*SortOrder*值， **dir**會依照第一個準則排序檔案名，然後依第二個準則，依此類推。 例如，如果您針對*SortOrder*使用 **/o**搭配**e**和 **-s**值（藉由使用 **/o： e-s**或 **/oe-s**）， **dir**會依副檔名排序目錄和檔案的名稱，並以最大的開頭，然後顯示最終結果。 依副檔名的字母排序會導致不具有副檔名的檔案名先出現，然後是目錄名稱，再加上副檔名為的檔案名。
- 使用重新導向符號和管道

  當您使用重新導向符號（ **>** ）將**目錄**輸出傳送至檔案或管道（ **|** ）以將**目錄**輸出傳送至另一個命令時，請使用 **/a：-d**和 **/b**只列出檔案名。 您可以使用*FileName*搭配 **/b**和 **/s**來指定**dir**是要搜尋目前目錄及其子目錄中所有符合*FileName*的檔案名。 **Dir**只會列出它所找到的每個檔案名的磁碟機號、目錄名稱、檔案名和副檔名（每行一個路徑）。 在您使用管道將**目錄**輸出傳送至另一個命令之前，您應該在 Autoexec nt 檔案中設定 TEMP 環境變數。
- 具有不同參數的**dir**命令可從修復主控台中取得。

## <a name="examples"></a>範例

若要依字母順序逐一顯示所有目錄，請以寬格式，並在每個畫面之後暫停，確定根目錄是目前的目錄，然後輸入：

```
dir /s/w/o/p
```

**Dir**列出根目錄中的根目錄、子目錄和檔案，包括副檔名。 然後， **dir**會列出樹狀目錄中每個子目錄中的子目錄名稱和檔案名。

若要改變上述範例，讓**dir**顯示檔案名和副檔名，但省略目錄名稱，請輸入：

```
dir /s/w/o/p/a:-d
```

若要列印目錄清單，請輸入：

```
dir > prn
```

當您指定**prn**時，會將目錄清單傳送到連接到 LPT1 埠的印表機。 如果您的印表機已連接到不同的埠，您必須將**prn**取代為正確埠的名稱。

您也可以將**prn**取代為檔案名，將**dir**命令的輸出重新導向至檔案。 您也可以輸入路徑。 例如，若要將**dir**輸出導向記錄目錄中的檔案 dir，請輸入：

```
dir > \records\dir.doc
```

如果 dir 不存在， **dir**會建立它，除非記錄目錄不存在。 在此情況下，會出現下列訊息：

`File creation error`

若要顯示磁片磁碟機 C 上所有目錄中副檔名為 .txt 的所有檔案名，請輸入：

```
dir c:\*.txt /w/o/s/p
```

**Dir**會以字母順序顯示每個目錄中相符檔案名的清單，並在每次螢幕填滿時暫停，直到您按任意鍵繼續。
