# <a name="dir"></a>dir



顯示目錄的檔案和子目錄的清單。 如果未指定參數，使用**dir**會顯示磁碟的磁碟區標籤和序號，後面接著目錄和檔案 （包括其名稱的日期和每個上次修改的時間） 在磁碟上的清單。 對於檔案， **dir**顯示副檔名和大小 （位元組）。 **Dir**也會顯示檔案和所列的目錄、 其累計大小和磁碟上剩餘的可用空間 （以位元組為單位） 的總數。

如需如何使用此命令的範例，請參閱[範例](#examples)。

## <a name="syntax"></a>語法

```
dir [<Drive>:][<Path>][<FileName>] [...] [/p] [/q] [/w] [/d] [/a[[:]<Attributes>]][/o[[:]<SortOrder>]] [/t[[:]<TimeField>]] [/s] [/b] [/l] [/n] [/x] [/c] [/4]
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|[\<Drive>:][<Path>]|指定您要查看磁碟機和目錄。|
|[\<FileName>]|指定特定檔案或您想要查看清單的檔案群組。|
|/p|一次顯示一個螢幕的清單。 若要查看下一個畫面，請在鍵盤上按任何鍵。|
|/q|顯示檔案的擁有權資訊。|
|/w|清單以寬格式顯示，最多可達五個檔名或目錄名稱，請在每一行。|
|/d|顯示與相同的格式清單 **/w**，但這些檔案會依資料行排序。|
|/a[[:]\<Attributes>]|顯示目錄及檔案與您指定的屬性的名稱。 如果您省略 **/a**， **dir**顯示以外的所有檔案的名稱隱藏和系統檔案。 如果您使用 **/a**而不指定*屬性*， **dir**顯示隱藏的所有檔案，包括名稱和系統檔案。</br>下列清單描述每一個值，您可以使用*屬性*。 使用冒號 （:）是選擇性的。 使用這些值的任何組合，並不會以空格分隔的值。</br>**d**目錄</br>**h**隱藏檔案</br>**s**系統檔案</br>**l**重新分析點</br>**r**唯讀檔案</br> 封存的檔案</br>**我**不內容索引的檔案</br>**-** 前置詞表示 「 不 」|
|/o[[:]\<SortOrder>]|排序依據的輸出*SortOrder*，這可以是下列值的任何組合：</br>**n**依名稱 （依字母順序排列）</br>**e**延伸模組 （按字母排列）</br>**g**先分組目錄</br>**s**大小 （最小的最先）</br>**d**的日期/時間 （先進先出）</br>**-** 若要反向順序的前置詞</br>注意:使用冒號是選擇性的。 您列出的順序會處理多個值。 不需以空格分隔保留多個值。</br>如果*SortOrder*未指定，則**dir /o**列出英文字母順序，後面接著檔案，也依字母順序排序的目錄。|
|/t[[:]\<TimeField>]|指定要顯示或用於排序的哪些時間 欄位。 下列清單描述每個值可用於*TimeField*:</br>**c**建立</br> 上次存取</br>**w**上次寫入|
|/s|列出每次出現的指定目錄內指定的檔案名稱和所有子目錄。|
|/b|顯示裸機目錄和檔案，不含其他資訊的清單。 **/ b**會覆寫 **/w**。|
|/l|顯示未排序的目錄名稱和檔案名稱，以小寫字母。|
|/n|會與檔案名稱很長的清單格式顯示在螢幕的右邊。|
|/x|顯示產生的非 8.3 檔案名稱的簡短名稱。 顯示會顯示相同 **/n**，但完整的名稱之前插入的簡短名稱。|
|/c|檔案大小，會顯示千位分隔符號。 這是是預設行為。 使用 **/-c**隱藏分隔符號。|
|/4|顯示格式為四位數的年份。|
|/?|在命令提示字元顯示說明。|

## <a name="remarks"></a>備註

- 若要使用多個*FileName*參數，請以空格、 逗號或分號分隔每個檔案名稱。
- 您可以使用萬用字元 ( **&#42;** 或是 **？** )、 代表一或多個字元的檔案名稱和顯示檔案或子目錄的子集。

  **星號 (\*):** 使用星號取代任何字元字串，例如：  
  - **dir \*.txt**列出開頭為.txt，使用如.txt，.txt1，.txt_old 的延伸模組目前的目錄中的所有檔案。
  - **讀取的 dir\*.txt**列出目前的目錄中開始使用 「 讀取 」 和副檔名為.txt，例如.txt、.txt1 或.txt_old 以開頭的所有檔案。
  - **讀取的 dir\*。\\** * 列出的所有檔案目前的目錄中開頭為 「 讀取 」 與任何副檔名。

  星號萬用字元一律會使用簡短檔案名稱對應，因此您可能會收到非預期的結果。 例如，下列目錄包含兩個檔案 （t.txt2 和 t97.txt）： 
 
  ```
  C:\test>dir /x
  Volume in drive C has no label.
  Volume Serial Number is B86A-EF32
    
  Directory of C:\test
    
  11/30/2004  01:40 PM <DIR>  .
  11/30/2004  01:40 PM <DIR> ..
  11/30/2004  11:05 AM 0 T97B4~1.TXT t.txt2
  11/30/2004  01:16 PM 0 t97.txt
  ```  

  您可能會預期輸入**dir t97\\** * 會傳回檔案 t97.txt。 不過，輸入**dir t97\\** * 會傳回兩個檔案，因為星號萬用字元使用比對到 t97.txt 檔案 t.txt2 其簡短名稱對應 T97B4~1.TXT。 同樣地，輸入**del t97\\** * 會刪除這兩個檔案。

  **問號 （？）：** 請使用問號取代單一字元的名稱。 例如，輸入**dir 讀取篇。txt**列出任何副檔名為.txt 的目前目錄中的檔案，以 「 讀取 」 開頭，後面接著三個字元。 這包括 Read.txt、 Read1.txt、 Read12.txt、 Read123.txt，和 Readme1.txt，但不是 Readme12.txt。
- 指定的檔案顯示屬性

  如果您使用 **/a**中的多個值*屬性*， **dir**會顯示這些檔案具有所有指定之屬性的名稱。 比方說，如果您使用 **/a**具有**r**並 **-h**做為屬性 (使用其中一種 **/: r h**或 **/ar-h**)， **dir**會只顯示未隱藏唯讀檔案的名稱。
- 指定檔案名稱排序

  如果您指定多個*SortOrder*的值， **dir**排序的檔案名稱的第一個準則，則第二個準則，依此類推。 比方說，如果您使用 **/o**具有**e**並 **-s**值*SortOrder* (使用其中一種 **/o:e-s**或是 **/oe-s**)， **dir**排序的目錄和檔案名稱的最大的第一個延伸模組，並接著會顯示最後的結果。 延伸模組的英文字母的排序會出現第一，沒有副檔名的檔案名稱的目錄名稱，然後再檔案名稱副檔名。
- 使用重新導向符號和管道

  當您使用的重新導向符號 ( **>** ) 來傳送**dir**輸出到檔案或管道 ( **|** ) 來傳送**dir**的輸出傳送至另一個命令，使用 **/a**並 **/b**列出檔案名稱。 您可以使用*檔名*使用 **/b**並 **/s**指定**dir**是目前的目錄和所有檔案及其子目錄中搜尋名稱符合*FileName*。 **Dir**列出只有磁碟機代號、 目錄名稱、 檔案名稱和副檔名 （每行的一個路徑），每個檔案名稱就會發現。 您可以使用管道傳送之前**dir**輸出至另一個命令，您應該設定 TEMP 環境變數 Autoexec.nt 檔案中。
- **Dir**命令，使用不同的參數，可從修復主控台。

## <a name="examples"></a>範例

若要顯示所有目錄一個接著一個地依字母順序，接著在寬的格式之後每個畫面，, 暫停確定根目錄是目前的目錄，，，然後輸入：

```
dir /s/w/o/p
```

**Dir**列出目錄的根目錄、 子目錄和檔案的根目錄，包括擴充功能中。 然後， **dir**列出子目錄名稱和檔案名稱，在樹狀目錄中的每個子目錄中。

若要變更上述範例中，讓**dir**顯示的檔案名稱和擴充功能，但是會省略的目錄名稱，而型別：

```
dir /s/w/o/p/a:-d
```

若要列印的目錄清單，請輸入：

```
dir > prn
```

當您指定**prn**，目錄清單會傳送至印表機，附加至 LPT1 連接埠。 如果您的印表機連接至不同的連接埠，您必須取代**prn**正確的連接埠的名稱。

您也可以重新導向的輸出**dir**命令來取代檔案**prn**檔案名稱。 您也可以輸入路徑。 例如，若要直接**dir**輸出至檔案 dir.doc，在記錄目錄中，類型：

```
dir > \records\dir.doc
```

Dir.doc 不存在，如果**dir**會建立它，除非記錄目錄不存在。 在此情況下，會顯示下列訊息：

`File creation error`

若要顯示 C 磁碟機上的所有目錄中的副檔名為.txt 的所有檔案名稱的清單，請輸入：

```
dir c:\*.txt /w/o/s/p
```

**Dir**寬格式顯示，在每個目錄中，名稱相符的檔案依字母順序排序的清單，以及它會暫停直到您按任意鍵繼續，會填滿螢幕每次。
