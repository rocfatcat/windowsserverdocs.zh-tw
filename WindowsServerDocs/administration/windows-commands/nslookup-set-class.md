# <a name="nslookup-set-class"></a>nslookup set class



變更查詢類別。 類別會指定資訊的通訊協定群組。

## <a name="syntax"></a>語法

```
set class=<Class>
```

## <a name="parameters"></a>參數

| 參數 |                                                                                                                                    描述                                                                                                                                    |
|-----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \<類別 >  | 預設類別是在中。 以下列出此命令的有效值。</br>在指定網際網路類別。</br>片指定混亂類別。</br>- HESIOD:指定 MIT Athena-io Hesiod 類別。</br>任何指定先前列出的任何萬用字元。 |
