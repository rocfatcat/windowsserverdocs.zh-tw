# <a name="help"></a>help



提供有關系統命令的線上資訊（也就是非網路命令）。 如果在沒有**參數的情況**下使用，說明會列出並簡短地描述每個系統命令。

如需如何使用此命令的範例，請參閱[範例](#BKMK_examples)。

## <a name="syntax"></a>語法

```
help [<Command>] 
[<Command>] /?
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|\<Command >|指定您想要其相關資訊的命令名稱。|

## <a name="BKMK_examples"></a>典型

若要查看**robocopy**命令的相關資訊，請輸入下列其中一項：
```
help robocopy
robocopy /? 
```
