# <a name="ftp-pwd"></a>ftp pwd

> 適用于： Windows Server （半年通道）、Windows Server 2019、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

顯示目前的遠端電腦目錄。

## <a name="syntax"></a>語法

```
pwd
```

### <a name="examples"></a>範例

若要顯示目前的遠端電腦目錄，請輸入：

```
pwd
```
