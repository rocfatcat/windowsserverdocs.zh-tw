# <a name="ftp-pwd_1"></a>ftp： pwd_1

## <a name="syntax"></a>語法  
```  
pwd  
```  
#### <a name="parameters"></a>參數  
無  
## <a name="examples"></a>範例  
顯示目前的遠端電腦目錄。  
```  
pwd  
```
