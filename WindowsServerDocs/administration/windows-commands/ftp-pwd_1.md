# <a name="ftp-pwd1"></a>ftp: pwd_1

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

顯示目前的遠端電腦目錄。   
## <a name="syntax"></a>語法  
```  
pwd  
```  
### <a name="parameters"></a>參數  
無  
## <a name="BKMK_Examples"></a>範例  
顯示目前的遠端電腦目錄。  
```  
pwd  
```
