# <a name="end-backup"></a>結束備份




結束完整的備份會話，並視需要發出具有適當寫入器狀態的**BackupComplete**事件。

## <a name="syntax"></a>語法

```
end backup
```

## <a name="remarks"></a>備註

-   此命令應該用於所有完整備份。
