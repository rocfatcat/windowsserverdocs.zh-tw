# <a name="mode"></a>mode



顯示系統狀態、 變更系統設定，或重新設定連接埠或裝置。 如果未指定參數，使用**模式**會顯示主控台和可用的 COM 裝置的所有可控制的屬性。

您可以使用**模式**來執行下列工作，每個工作使用不同的語法：
-   [若要設定序列通訊埠](#BKMK_1)
-   [若要顯示單一裝置或所有裝置的狀態](#BKMK_2)
-   [從平行連接埠的序列通訊連接埠的輸出重新導向](#BKMK_3)
-   [若要選取，請重新整理或顯示在主控台的字碼頁的數字](#BKMK_4)
-   [若要變更命令提示字元螢幕緩衝區的大小](#BKMK_5)
-   [若要設定鍵盤輸入速率](#BKMK_6)

## <a name="BKMK_1"></a>若要設定序列通訊埠

### <a name="syntax"></a>語法

```
mode com<M>[:] [baud=<B>] [parity=<P>] [data=<D>] [stop=<S>] [to={on|off}] [xon={on|off}] [odsr={on|off}] [octs={on|off}] [dtr={on|off|hs}] [rts={on|off|hs|tg}] [idsr={on|off}]
```

### <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|Com\<M > [::]|指定非同步 Prncnfg.vbshronous 通訊連接埠數目。|
|baud=\<B>|指定位元 / 秒的傳輸速率。 下表列出的有效縮寫*B*和其相關的速率。</br>-   **11** = 110 的傳輸速率</br>-   **15** = 150 傳輸速率</br>-   **30** = 300 的傳輸速率</br>-   **60** = 600 的傳輸速率</br>-   **12** = 1200 的傳輸速率</br>-   **24** = 2400 傳輸速率</br>-   **48** = 4800 傳輸速率</br>-   **96** = 9600 波特</br>-   **19** = 19,200 傳輸速率|
|parity=\<P>|指定系統来檢查的傳輸錯誤所使用的同位位元。 下表列出的有效值*P*。預設值是**e**。 並非所有電腦都支援的值**m**並**s**。</br>-   **n** = 無</br>-   **e** = even</br>-   **o** = odd</br>-   **m** = 標記</br>-   **s** = 空間|
|data=\<D>|字元中指定資料位元的數。 有效值**d** 5 到 8 的範圍。 預設值為 7。 並非所有電腦都支援 5 和 6 的值。|
|stop=\<S>|指定停止位元會定義結尾的字元的數：1、 1.5 倍或 2。 如果傳輸速率為 110 時，預設值為 2。 否則，預設值為 1。 並非所有電腦都支援的值為 1.5。|
|to={on | off}|指定無限逾時處理是否為開啟或關閉。 預設為關閉。|
|xon={on | off}|指定資料流量控制的 xon 或 xoff 通訊協定是否為開啟或關閉。|
|odsr={on | off}|指定是否使用 Data Set Ready (DSR) 的線路輸出信號交換為開啟或關閉。|
|octs={on | off}|指定使用 Clear To Send (CTS) 線路輸出信號交換是否為開啟或關閉。|
|dtr={on | off | hs}|指定 Data Terminal Ready (DTR) 線路是開啟或關閉或設定為信號交換。|
|rts={on | off | hs | tg}|指定是否要求 To Send (RTS) 線路設為 on、 off、 交握或切換。|
|idsr={on | off}|指定的 DSR 線路敏感度為開啟或關閉。|
|/?|在命令提示字元顯示說明。|

## <a name="BKMK_2"></a>若要顯示單一裝置或所有裝置的狀態

### <a name="syntax"></a>語法

```
mode [<Device>] [/status]
```

### <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|\<裝置 >|指定您要顯示狀態的裝置名稱。|
|/status|要求任何重新導向的平行印表機的狀態。 您可以將縮寫 **/status**做為命令列選項 **/sta**。|
|/?|在命令提示字元顯示說明。|

### <a name="remarks"></a>備註

如果未指定參數，使用**模式**顯示安裝在您的系統的所有裝置的狀態。

## <a name="BKMK_3"></a>從平行連接埠的序列通訊連接埠的輸出重新導向

### <a name="syntax"></a>語法

```
mode lpt<N>[:]=com<M>[:]
```

### <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|lpt\<N > [::]|必要。 指定平行連接埠。 有效值*N*位於範圍 1 到 3。|
|com\<M > [::]|必要。 指定的序列連接埠。 有效值*M*位於範圍 1 到 4。|
|/?|在命令提示字元顯示說明。|

### <a name="remarks"></a>備註

您必須是 Administrators 群組的成員，才能重新導向列印。

### <a name="examples"></a>範例

若要設定您的系統，以便平行印表機的輸出傳送至序列的印表機，您必須使用**模式**命令兩次。 第一次，使用**模式**設定序列埠。 第二個階段中，使用**模式**平行的印表機輸出重新導向至您指定在第一個序列埠**模式**命令。

例如，如果序列的印表機在 4800 傳輸速率運作偶數同位檢查，而且它連接至 COM1 連接埠 （第一個序列連線您的電腦上），輸入：
```
mode com1 48,e,,,b
mode lpt1=com1
```
如果您重新導向至 COM1，LPT1 平行的印表機輸出，但是您可以再決定，您會想要列印檔案使用 LPT1、 列印檔案之前，請輸入下列命令：
```
mode lpt1
```
此命令可防止重新導向至 COM1 LPT1 的檔案。

## <a name="BKMK_4"></a>若要選取，請重新整理或顯示在主控台的字碼頁的數字

### <a name="syntax"></a>語法

```
mode <Device> codepage select=<YYY>
mode <Device> codepage [/status]
```

### <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|\<裝置 >|必要。 指定您要選取的字碼頁的裝置。 CON 是裝置的唯一有效的名稱。|
|選取 字碼頁 =|必要。 指定要與指定的裝置搭配使用的字碼頁。 您可以將縮寫**字碼頁****選取**作為**cp** **sel**。|
|\<YYY>|必要。 指定要選取的字碼頁數目。 下表顯示每個程式碼支援的頁面和國家/地區或語言。</br>437:美國</br>850:多語系 (拉丁文 I)</br>852:斯拉夫文 (拉丁文 II)</br>855:斯拉夫文 （俄文）</br>857:土耳其文</br>860:葡萄牙文</br>861:冰島文</br>863:加拿大法文</br>865:北歐</br>866:俄文</br>869:現代希臘文|
|codepage|必要。 顯示的數字的程式碼頁面 （如果有的話） 所指定的裝置。|
|/status|顯示目前的字碼頁，選取指定之裝置的數字。 您可以將縮寫 **/status**要 **/sta**。 您指定是否 **/status**，**模式的字碼頁**顯示所指定之裝置的字碼頁的數字。|
|/?|在命令提示字元顯示說明。|

## <a name="BKMK_5"></a>若要變更命令提示字元螢幕緩衝區的大小

### <a name="syntax"></a>語法

```
mode con[:] [cols=<C>] [lines=<N>]
```

### <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|con[:]|必要。 指出此變更會套用至命令提示字元視窗。|
|cols=\<C>|在命令提示字元螢幕緩衝區中指定資料行的數目。|
|lines=\<N>|在命令提示字元螢幕緩衝區中指定行的數。|
|/?|在命令提示字元顯示說明。|

## <a name="BKMK_6"></a>若要設定鍵盤輸入速率

### <a name="syntax"></a>語法

```
mode con[:] [rate=<R> delay=<D>]
```

### <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|con[:]|必要。 指的是鍵盤。|
|rate=\<R>|當您按住按鍵時，請在畫面上指定字元的重複的速率。|
|delay=\<D>|指定在您按下並按住按鍵之前的字元輸出會重複, 之後經過的時間量。|
|/?|在命令提示字元顯示說明。|

### <a name="remarks"></a>備註

-   輸入速率是您可以按住該字元的索引鍵時，字元重複的速率。 輸入速率會有兩個元件、 速度和延遲。 有些鍵盤無法辨識此命令。
-   使用 **速率 = * * * R*

    有效值為 1 到 32 的範圍中。 這些值會等於大約每秒 2 到 30 個字元。 預設值是 IBM 在相容的鍵盤，20 和 21 的 IBM PS/2 相容的鍵盤。 如果您設定的速率，您也必須設定延遲。
-   使用**延遲**=*D*

    有效值*D*是 1、 2、 3 和 4 （代表 0.25，0.50，0.75 和 1 秒）。 預設值為 2。 如果您設定 [延遲] 時，您也必須設定的速率。
