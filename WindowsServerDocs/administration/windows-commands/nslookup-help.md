# <a name="nslookup-help"></a>nslookup help

顯示子命令解說文字。

## <a name="syntax"></a>語法

```
help
```

```
?
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
| --------- | ----------- |
| /? | 在命令提示字元顯示說明。 |
| /help | 在命令提示字元顯示說明。 |
