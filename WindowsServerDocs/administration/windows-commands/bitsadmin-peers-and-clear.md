# <a name="bitsadmin-peers-and-clear"></a>bitsadmin 同儕和清除



清除對等電腦清單。

## <a name="syntax"></a>語法

```
bitsadmin /Peers /Clear 
```

## <a name="BKMK_examples"></a>範例

下列範例會清除對等電腦清單。
```
C:\>bitsadmin /Peers /Clear
```
