# <a name="bitsadmin-peers-and-clear"></a>bitsadmin 對等和清除



清除對等清單。

## <a name="syntax"></a>語法

```
bitsadmin /Peers /Clear 
```

## <a name="BKMK_examples"></a>典型

下列範例會清除對等清單。
```
C:\>bitsadmin /Peers /Clear
```
