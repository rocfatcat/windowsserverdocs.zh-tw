# <a name="rundll32-printuidllprintuientry"></a>rundll32.exe printui.dll .dll、PrintUIEntry

>適用於：Windows Server （半年通道）、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

自動化許多印表機設定工作。 printui.dll 是可執行檔，其中包含 [印表機設定] 對話方塊所使用的功能。 您也可以從腳本或命令列批次檔中呼叫這些函式，也可以從命令提示字元以互動方式執行這些函數。 如需如何使用此命令的範例，請參閱[範例](#BKMK_Examples)。  
## <a name="syntax"></a>語法  
```  
rundll32 printui.dll PrintUIEntry [BaseParameter] [ModificationParameter1] [ModificationParameter2] [ModificationParameterN]  
```  
您也可以使用下列替代語法，雖然本主題中的範例使用先前的語法：  
```  
rundll32 printui.dll,PrintUIEntry [BaseParameter] [ModificationParameter1] [ModificationParameter2] [ModificationParameterN]  
```  
```  
rundll32 printui PrintUIEntry [BaseParameter] [ModificationParameter1] [ModificationParameter2] [ModificationParameterN]  
```  
```  
rundll32 printui,PrintUIEntry [BaseParameter] [ModificationParameter1] [ModificationParameter2] [ModificationParameterN]  
```  
## <a name="parameters"></a>參數

## <a name="remarks"></a>備註  
- **PrintUIEntry**關鍵字會區分大小寫，而且您必須使用本主題中的範例所示的確切大小寫來輸入此命令的語法。  
- 如需一些常見工作的語法，請參閱本檔中的[範例](#BKMK_Examples)。 如需更多範例，請在命令提示字元中輸入： **rundll32.exe printui.dll .dll、PrintUIEntry/？**  
  ## <a name="BKMK_Examples"></a>典型  
  若要為電腦新增新的遠端印表機 printer1，請 Client1，這是執行此命令的使用者帳戶可以看見的，請輸入：  
  ```  
  rundll32 printui.dll PrintUIEntry /in /n\\client1\printer1  
  ```  
  若要使用「新增印表機」 wizard 來新增印表機，並使用 InfFile 位於磁片磁碟機 c： at Infpath 的 .inf 檔案，請輸入：  
  ```  
  rundll32 printui.dll PrintUIEntry /ii /f c:\Infpath\InfFile.inf  
  ```  
  若要刪除現有的印表機，請 printer1 在電腦上 Client1，輸入：  
  ```  
  rundll32 printui.dll PrintUIEntry /dn /n\\client1\printer1  
  ```  
  若要為電腦的所有使用者新增每台電腦的印表機連線，請 printer2，Client2，輸入（當使用者登入時會套用連線）：  
  ```  
  rundll32 printui.dll PrintUIEntry /ga /n\\client2\printer2  
  ```  
  若要刪除每台電腦的印表機連線，請 printer2，針對電腦的所有使用者，Client2，輸入（當使用者登入時將會刪除連線）：  
  ```  
  rundll32 printui.dll PrintUIEntry /gd /n\\client2\printer2  
  ```  
  若要查看列印伺服器的屬性，請 printServer1，輸入：  
  ```  
  rundll32 printui.dll PrintUIEntry /s /t1 /c\\printserver1  
  ```  
  若要查看印表機的屬性，請 printer3，輸入：  
  ```  
  rundll32 printui.dll PrintUIEntry /p /n\\printer3  
  ```
