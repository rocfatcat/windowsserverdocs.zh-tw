# <a name="the-start-server-command"></a>啟動伺服器 命令

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

Windows 部署服務伺服器上啟動多點傳送的傳輸、 命名空間、 傳輸伺服器或所有服務。
## <a name="subcommands"></a>子命令
|子命令|描述|
|-------|--------|
|[子命令： /start-multicasttransmission](subcommand-start-multicasttransmission.md)|開始傳輸。|
|[子命令： 開始命名空間](subcommand-start-namespace.md)|啟動命名空間。|
|[Subcommand: start-Server](subcommand-start-server.md)|Windows 部署服務伺服器上啟動所有服務。|
|[Subcommand: start-TransportServer](subcommand-start-transportserver.md)|Transport Server 上啟動所有服務。|
