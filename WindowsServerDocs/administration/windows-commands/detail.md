# <a name="detail"></a>資訊



顯示所選磁片、磁碟分割、磁片區或虛擬硬碟（VHD）的相關資訊。

## <a name="syntax"></a>語法

```
detail disk
detail partition
detail volume 
detail vdisk
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|[詳細資料磁片](detail-disk.md)|顯示所選磁碟的內容以及該磁碟上的磁碟區。|
|[詳細資料分割](detail-partition.md)|顯示選取之分割區的屬性。|
|[詳細資料量](detail-volume.md)|顯示目前磁片區所在的磁片。|
|[詳細資料 vdisk](detail-vdisk.md)|顯示所選取 VHD 的屬性。|
