# <a name="detail"></a>詳細資料



顯示選取的磁碟、 磁碟分割、 磁碟區或虛擬硬碟 (VHD) 的相關資訊。

## <a name="syntax"></a>語法

```
detail disk
detail partition
detail volume 
detail vdisk
```

## <a name="parameters"></a>參數

|參數|描述|
|---------|-----------|
|[詳細資料磁碟](detail-disk.md)|顯示所選磁碟的內容以及該磁碟上的磁碟區。|
|[詳細資料的磁碟分割](detail-partition.md)|顯示選取的資料分割的屬性。|
|[詳細資料的磁碟區](detail-volume.md)|會顯示目前的磁碟區所在的磁碟。|
|[詳細資料 vdisk](detail-vdisk.md)|顯示選取的 VHD 的屬性。|
