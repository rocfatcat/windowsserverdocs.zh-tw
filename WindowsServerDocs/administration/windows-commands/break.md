# <a name="break"></a>break

> [!IMPORTANT]
> 此命令已不再使用。 而僅是為了維持與現有 MS-DOS 檔案的相容性才將此命令包含在內，而且因為功能是自動的，它在命令列並不會有任何作用。

設定或清除 MS-DOS 系統上的擴充 CTRL + C 檢查。 如果使用時不含參數， **break** 會顯示現有的設定值。

如果命令延伸模組已在 Windows 平臺上啟用和執行，將 **break** 命令插入批次檔中，就會在偵錯工具進行調試時進入硬式編碼的中斷點。

## <a name="syntax"></a>語法

```
break=[on|off]
```
