# <a name="automount"></a>自動掛接



啟用或停用自動掛接功能。 啟用 （預設值），Windows 會在新增至系統，，然後將磁碟機代號指派給磁碟區時，自動會掛接新的基本或動態磁碟區的檔案系統。
