# <a name="bitsadmin-peers-and-discover"></a>bitsadmin 對等和探索

再次探索對等。

## <a name="syntax"></a>語法

```
bitsadmin /peers /discover
```

## <a name="examples"></a><a name=BKMK_examples></a>典型

下列範例會探索對等。

```
C:\>bitsadmin /peers /discover
```
