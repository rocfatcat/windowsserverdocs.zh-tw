# <a name="bitsadmin-peers-and-discover"></a>bitsadmin 讓對等互連，並探索



再次探索對等。

## <a name="syntax"></a>語法

```
bitsadmin /Peers /Discover 
```

## <a name="BKMK_examples"></a>範例

下列範例會探索對等。
```
C:\>bitsadmin /Peers /Discover
```
