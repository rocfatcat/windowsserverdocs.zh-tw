# <a name="using-the-disable-command"></a>使用停用 命令

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

停用 Windows 部署服務的所有服務。
## <a name="subcommands"></a>子命令
|子命令|描述|
|-------|--------|
|[使用停用伺服器命令](using-the-disable-server-command.md)|停用指定的伺服器 （部署伺服器） 上的所有 Windows 部署服務服務。|
|[使用停用 TransportServer 命令](using-the-disable-transportserver-command.md)|停用指定的傳輸伺服器上的所有 Windows 部署服務服務。|
