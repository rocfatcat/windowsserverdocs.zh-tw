# <a name="chdir"></a>chdir

> 適用于： Windows Server (半年通道) 、Windows Server 2019、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

顯示目前目錄的名稱，或變更目前的目錄。 如果只搭配磁碟機號使用 (例如， `chdir C:`) ， **chdir** 會顯示指定磁片磁碟機中目前的目錄的名稱。 如果使用時不含 **參數，則** 會顯示目前的磁片磁碟機和目錄。

> > [!NOTE]
> 此命令已由 **cd 命令**取代。 如需詳細資訊，包括語法和參數詳細資料，請參閱 [cd 命令](cd.md)。
