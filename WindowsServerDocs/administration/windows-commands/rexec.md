# <a name="rexec"></a>rexec

在遠端主機上執行指定的命令。 遠端主機必須執行 rexecd 服務 (或背景程式) ，讓 rexec 連接至。

> [!IMPORTANT]
> **Rexec**命令已被取代，不保證會在 Windows 中支援。
