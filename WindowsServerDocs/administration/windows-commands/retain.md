# <a name="retain"></a>幻燈片



準備要當做開機或系統磁碟區使用的現有動態簡單磁片區。

## <a name="syntax"></a>語法

```
retain
```

## <a name="remarks"></a>備註

-   在主開機記錄（MBR）動態磁碟上，此命令會在主開機記錄中建立磁碟分割專案。
-   在 GUID 磁碟分割表格（GPT）動態磁碟上，此命令會在 GUID 磁碟分割表格中建立磁碟分割專案。
