# <a name="pagefileconfig"></a>pagefileconfig

> 適用于： Windows Server 2003、Windows Server 2003 R2、Windows Server 2003 SP1、Windows Server 2003 含 SP2

>[!IMPORTANT]
> 此命令已被取代。

可讓系統管理員顯示並設定系統的分頁檔案虛擬記憶體設定。 如需說明和使用方式資訊，請參閱 [pagefileconfig](/previous-versions/orphan-topics/ws.10/cc772827(v=ws.10))。
