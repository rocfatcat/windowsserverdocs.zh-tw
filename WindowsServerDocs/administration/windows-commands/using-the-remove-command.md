# <a name="using-the-remove-command"></a>使用移除 命令

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

## <a name="subcommands"></a>子命令
|Subommand|描述|
|-------|--------|
|[使用移除映像命令](using-the-remove-image-command.md)|從伺服器中移除的開機或安裝映像。|
|[使用移除 ImageGroup 命令](using-the-remove-imagegroup-command.md)|從伺服器移除映像群組。|
|[使用 /remove-multicasttransmission 命令](using-the-remove-multicasttransmission-command.md)|停用映像的多點傳送的的傳輸。|
|[使用移除命名空間命令](using-the-remove-namespace-command.md)|從伺服器移除命名空間。|
|[使用移除 DriverGroupPackage 命令](using-the-remove-drivergrouppackage-command.md)|在伺服器上的驅動程式群組中移除的驅動程式套件。|
|[使用移除 DriverGroupPackages 命令](using-the-remove-drivergrouppackages-command.md)|移除驅動程式群組的伺服器上的驅動程式套件。|
|[使用移除 DriverPackage 命令](using-the-remove-driverpackage-command.md)|從伺服器移除驅動程式套件。|
|[使用移除 DriverPackages 命令](using-the-remove-driverpackages-command.md)|從伺服器移除驅動程式套件。|
|[使用移除 DriverGroup 命令](using-the-remove-drivergroup-command.md)|從伺服器移除驅動程式群組。|
|[使用移除 DriverGroupFilter 命令](using-the-remove-drivergroupfilter-command.md)|移除驅動程式群組的伺服器上的篩選規則。|
