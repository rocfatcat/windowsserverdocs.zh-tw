# <a name="ftpdebug"></a>ftp:debug

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

切換偵錯模式。   
## <a name="syntax"></a>語法  
```  
debug  
```  
### <a name="parameters"></a>參數  
無  
## <a name="remarks"></a>備註  
-   根據預設，偵錯已關閉。  
-   偵錯開啟時，會傳送到遠端電腦的每個命令會顯示，前面加上大於 (>) 字元。  
## <a name="BKMK_Examples"></a>範例  
切換偵錯模式開啟及關閉。  
```  
debug  
```
