# <a name="rcp"></a>rcp

>[!IMPORTANT]
> 此命令已被取代。

您可以使用 [**新增功能] [Wizard]** 來安裝 UNIX 應用程式的子系統。 如需詳細資訊和下載檔案，請參閱[Microsoft Windows 7 和 Windows Server 2008 R2 中 UNIX 應用程式子系統的公用程式和 SDK](https://www.microsoft.com/download/details.aspx?id=2391)。

安裝之後，您可以 (csh 或 tcsh) 或 KornShell 命令視窗中開啟 C Shell，並執行**rcp**。 如需詳細資訊，請在 [C Shell] 或 [KornShell] 提示字元中輸入「**男人**」。
