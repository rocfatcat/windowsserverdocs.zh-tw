# <a name="the-stop-server-command"></a>停止伺服器命令

>適用於：Windows Server （半年通道），Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

停止 Windows 部署服務的服務。
## <a name="subcommands"></a>子命令
|子命令|描述|
|-------|--------|
|[子命令： 停止伺服器](subcommand-stop-server.md)|Windows 部署服務伺服器 （部署伺服器） 上，會停止所有服務。|
|[Subcommand: stop-TransportServer](subcommand-stop-transportserver.md)|Transport Server 上，會停止所有服務。|
