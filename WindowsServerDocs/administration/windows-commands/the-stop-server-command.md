# <a name="stop-server"></a>停止-伺服器
> 適用于： Windows Server (半年通道) 、Windows Server 2019、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

停止 Windows 部署服務的服務。

## <a name="subcommands"></a>子
|子命令|描述|
|-------|--------|
|[子命令： stop-Server](subcommand-stop-server.md)|停止 Windows 部署服務 server (部署伺服器上的所有服務) 。|
|[子命令： stop-TransportServer](subcommand-stop-transportserver.md)|停止傳輸伺服器上的所有服務。|
