# <a name="stop-server"></a>停止-伺服器
>適用於：Windows Server (半年通道)、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

停止 Windows 部署服務的服務。

## <a name="subcommands"></a>子
|來|描述|
|-------|--------|
|[子命令： stop-Server](subcommand-stop-server.md)|停止 Windows 部署服務伺服器上的所有服務（部署伺服器）。|
|[子命令： stop-TransportServer](subcommand-stop-transportserver.md)|停止傳輸伺服器上的所有服務。|
