Sysocmgr.exe 已被取代，在未來的 Windows 版本中不保證會受到支援。

此工具組含在 Windows Server 2003 中。 如需詳細資訊，請參閱[sysocmgr.exe](https://technet.microsoft.com/library/cc773290(v=ws.10).aspx)。
