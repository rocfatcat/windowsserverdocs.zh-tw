# <a name="bitsadmin-cache-and-list"></a>bitsadmin 快取與清單

列出所有快取專案。

## <a name="syntax"></a>語法

```
bitsadmin /cache /list [/verbose]
```

## <a name="examples"></a><a name=BKMK_examples></a>典型

下列範例會列出詳細資訊格式的所有快取專案。

```
C:\> bitsadmin /cache /list /verbose
```
