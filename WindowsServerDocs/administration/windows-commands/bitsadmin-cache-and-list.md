# <a name="bitsadmin-cache-and-list"></a>bitsadmin 快取和清單



列出所有快取項目。

## <a name="syntax"></a>語法

```
bitsadmin /Cache /List [/Verbose] 
```

## <a name="BKMK_examples"></a>範例

下列範例會列出所有的快取項目詳細資訊的格式。
```
C:\> bitsadmin /Cache /List /Verbose
```
