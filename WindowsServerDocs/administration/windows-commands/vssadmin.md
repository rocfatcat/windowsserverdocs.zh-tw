# <a name="vssadmin"></a>Vssadmin

>適用於： Windows 10、 Windows 8.1、 Windows Server 2016、 Windows Server 2012 R2、 Windows Server 2012、 Windows Server 2008 R2、 Windows Server 2008

會顯示目前的磁碟區陰影複製備份所有已安裝的陰影複製作者和提供者。 下列表格檢視中其命令語法選取命令名稱。

|命令|說明|
|---|---|
|[Vssadmin 新增 shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788051(v%3dws.11))|新增磁碟區陰影複製儲存關聯。|
|[Vssadmin 建立陰影](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788055(v%3dws.11))|會建立新的磁碟區陰影複製。|
|[Vssadmin 刪除陰影](vssadmin-delete-shadows.md)|會刪除磁碟區陰影複製。|
|[Vssadmin 刪除 shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc785461(v%3dws.11))|會刪除磁碟區陰影複製儲存關聯。|
|[Vssadmin 清單提供者](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788108(v%3dws.11))|列出已登錄的磁碟區陰影複製提供者。|
|[Vssadmin 清單陰影](vssadmin-list-shadows.md)|現有的磁碟區陰影複製的清單。|
|[Vssadmin 清單 shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788045(v%3dws.11))|列出在系統上所有的陰影複製儲存關聯。|
|[Vssadmin 清單磁碟區](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788064(v%3dws.11))|列出已可供陰影複製的磁碟區。|
|[Vssadmin 清單作者](vssadmin-list-writers.md)|列出所有已訂閱的磁碟區陰影複製作者在系統上。|
|[Vssadmin 調整大小 shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788050(v%3dws.11))|調整大小的陰影複製儲存關聯的大小上限。|
