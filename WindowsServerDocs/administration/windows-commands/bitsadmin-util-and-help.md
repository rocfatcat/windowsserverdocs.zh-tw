# <a name="bitsadmin-util-and-help"></a>bitsadmin util 與協助

顯示**util**參數的命令列使用方式。

## <a name="syntax"></a>語法

```
bitsadmin /util /help
```

## <a name="examples"></a>範例

下列命令會顯示**Util**參數的命令列說明。

```
C:\>bitsadmin /util /help
```
