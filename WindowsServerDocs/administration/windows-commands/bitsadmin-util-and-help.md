# <a name="bitsadmin-util-and-help"></a>bitsadmin util and help

顯示 **/util**參數的命令列使用方式。

## <a name="syntax"></a>語法

```
bitsadmin /util /help
```

## <a name="examples"></a>範例

若要顯示 **/util**參數的命令列說明：

```
bitsadmin /util /help
```
