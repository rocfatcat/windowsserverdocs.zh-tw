# <a name="bitsadmin-util-and-help"></a>bitsadmin util 與協助

顯示**Util**參數的命令列使用方式。

## <a name="syntax"></a>語法

```
Bitsadmin /Util /Help 
```

## <a name="examples"></a><a name=BKMK_examples></a>典型

下列命令會顯示**Util**參數的命令列說明。
```
Bitsadmin /Util /Help
```
