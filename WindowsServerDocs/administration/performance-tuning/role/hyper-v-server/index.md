# <a name="performance-tuning-hyper-v-servers"></a>Hyper-V 伺服器的效能調整


Hyper-V 是 Windows Server 2016 中的虛擬機器伺服器角色。 虛擬機器伺服器可以裝載多部與彼此隔離但共用底層硬體資源 (利用處理器、記憶體與 I/O 裝置虛擬化功能) 的虛擬機器。 透過將多部伺服器合併到單一機器，虛擬化可以改善資源使用率與能源效率，並降低伺服器作業與維護成本。 此外，虛擬機器與管理 API 提供更多彈性讓您管理資源、進行負載平衡與佈建系統。
