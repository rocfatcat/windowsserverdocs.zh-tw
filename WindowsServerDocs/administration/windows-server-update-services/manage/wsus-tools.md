 ## <a name="wsus-reporting-rollup-sample-toolhttpsdownloadmicrosoftcomdownload339339ac5ee-ae9a-44a4-b09c-483736294433wsusrollupsampleexe"></a>[WSUS 報告匯總範例工具](https://download.microsoft.com/download/3/3/9/339ac5ee-ae9a-44a4-b09c-483736294433/WSUSRollupSample.EXE)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    
 > 此工具會使用 WSUS 應用程式開發介面（API）來示範 WSUS 的集中式監視和報告。 它會從 WSUS 伺服器建立一份「更新」和「電腦狀態」報告到您的 WSUS 環境。 範例套件也包含範例來源檔案，可自訂或擴充工具的工具功能以符合特定需求。 WSUS 報告匯總範例工具和檔案是依原樣提供。 此工具或範例檔案不提供產品支援。 如需詳細資訊，請參閱[讀我檔案](https://download.microsoft.com/download/8/1/a/81a41962-cff5-4396-a567-0d2f87d8f67a/Readme.htm)。
