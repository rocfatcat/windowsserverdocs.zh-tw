# <a name="windows-server-update-services-tools-and-utilities"></a>Windows Server Update Services 工具和公用程式

>適用於：Windows Server (半年通道)、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

 在此頁面上，您可以找到適用于 WSUS 及其元件的可下載工具和公用程式。

 ## <a name="wsus-reporting-rollup-sample-tool"></a>[WSUS 報告匯總範例工具](https://download.microsoft.com/download/3/3/9/339ac5ee-ae9a-44a4-b09c-483736294433/WSUSRollupSample.EXE)

 > 此工具使用 WSUS 應用程式開發介面 (API) 來示範 WSUS 的集中式監視和報告。 它會從 WSUS 伺服器建立一份「更新」和「電腦狀態」報告到您的 WSUS 環境。 範例套件也包含範例來源檔案，可自訂或擴充工具的工具功能以符合特定需求。 WSUS 報告匯總範例工具和檔案是依原樣提供。 此工具或範例檔案不提供產品支援。 如需詳細資訊，請參閱 [讀我檔案](https://download.microsoft.com/download/8/1/a/81a41962-cff5-4396-a567-0d2f87d8f67a/Readme.htm)。

## <a name="client-diagnostics-tool"></a>[用戶端診斷工具](https://download.microsoft.com/download/9/7/6/976d1084-d2fd-45a1-8c27-a467c768d8ef/WSUS%20Client%20Diagnostic%20Tool.EXE)

 > 下載這項工具，其設計目的是要協助 WSUS 系統管理員針對無法回報給 WSUS 伺服器的用戶端電腦進行疑難排解。 此工具會進行初步檢查，並測試 WSUS 伺服器與用戶端電腦之間的通訊。 工具完成測試之後，就會在主控台視窗中顯示結果。 Windows Server Update Services 用戶端診斷工具的提供方式如下。 此工具不提供產品支援。 如需詳細資訊，請參閱 [讀我檔案](https://download.microsoft.com/download/e/4/b/e4bc4153-be1f-460f-800e-69c6a1857d68/readme.htm)。
