## <a name="microsoft-windows-admin-center---license-terms-for-extensions"></a>MICROSOFT WINDOWS ADMIN CENTER - 擴充功能授權條款規定
________________________________________

此軟體可能包含 Microsoft 所開發的軟體擴充功能。 這些 Microsoft 擴充功能可能受個別合約及其各自支援原則所規範，並不受本軟體授權條款規定約束。 您可以按一下適用的擴充功能，檢視 [擴充管理員] 中 [已安裝的擴充功能] 下方的授權資訊。 本軟體也會提供選項讓您從 Microsoft 的擴充功能摘要或其他由協力廠商所管理的存放庫，下載其他 Microsoft 及協力廠商擴充功能。 這些擴充功能是受個別合約規範，不受本軟體授權條款規定約束。 您可以按一下適用的擴充功能，檢視 [擴充管理員] 中 [可用的擴充功能] 下方的授權資訊。 Microsoft 並不散發任何協力廠商套件，也未對任何協力廠商套件提供任何授權或保證。
