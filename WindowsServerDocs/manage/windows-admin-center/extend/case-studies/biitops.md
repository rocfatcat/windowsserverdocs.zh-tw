# <a name="biitops-changes-extension"></a>BiitOps 變更延伸模組

BiitOps Windows Admin Center 變更可讓您深入了解變更您的 Windows Server 實體/虛擬機器上的硬體、 軟體和組態設定。 BiitOps 變更延伸模組可讓您快速且符合成本效益地判斷造成合規性、 可靠性或安全性相關問題的變更。 它會顯示精確的新功能、 變更和已刪除單一-虛擬管理--介面中。 它適用於伺服器和 Microsoft Windows Server 上所選的服務。

瞭解詳細資訊，請造訪[BiitOps 產品網站](http://www.biitops.com/solutions/changes-for-wac/)或訂閱[免費試用](http://www.biitops.com/solutions/register-changes-for-wac/)。
