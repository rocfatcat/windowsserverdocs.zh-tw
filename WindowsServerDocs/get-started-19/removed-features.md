# <a name="features-removed-or-planned-for-replacement-in-windows-server"></a>Windows Server 中已移除或計劃取代的功能

> 適用於：Windows Server 2019、Windows Server 2016、Windows Server (半年通道)

您可以找到每個這些 Windows Server 版本已移除和已淘汰的功能。  

長期維護通道 (LTSC) 版本：

- [Windows Server 2019 中已移除或計劃移除的功能](removed-features-19.md)
- [Windows Server 2016 中已移除或取代的功能](../get-started/Deprecated-Features.md)

半年通道版本：

- [從 Windows Server 版本 1903 開始移除或計劃取代的功能](removed-features-1903.md)
- [從 Windows Server 版本 1803 開始移除或計劃取代的功能](../get-started/windows-server-1803-removed-features.md)
- [從 Windows Server 版本 1709 開始移除或計劃取代的功能](../get-started/Removed-Features-1709.md) (自 2019 年 4 月 9 日起支援終止)

Windows Server 半年通道發行以更快的速度提供新的作業系統功能。 每六個月推出新版本，且支援 18 個月。 檢查 [Windows Server 版本資訊](../get-started/windows-server-release-info.md)和[生命週期支援頁面](https://support.microsoft.com/lifecycle)的支援日期，並盡可能一律使用最新版本。
