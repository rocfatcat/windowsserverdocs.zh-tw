# <a name="add-host-information-to-the-hgs-configuration"></a>將主機資訊新增至 HGS 設定

>適用於：Windows Server 2019，Windows Server （半年通道），Windows Server 2016

從網狀架構系統管理員取得主機的資訊，並將它新增至 HGS。 此步驟會因證明模式而有所不同：

- [新增系統管理信任證明的主機資訊](guarded-fabric-add-host-information-for-admin-trusted-attestation.md)
- [新增 TPM 信任證明的主機資訊](guarded-fabric-add-host-information-for-tpm-trusted-attestation.md)
