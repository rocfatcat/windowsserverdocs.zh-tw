# <a name="windows-defender-antivirus-for-windows-server"></a>Windows Server 的 Windows Defender 防毒軟體

>適用於：Windows Server 2016

Windows Server 2016 現在會包含 Windows Defender 防毒軟體。 Windows Defender 防毒軟體是立即主動針對已知的惡意程式碼保護 Windows Server 2016 及可以定期更新反惡意程式碼定義透過 Windows Update 的惡意程式碼防護。

查看[在 Windows 10 中的 Windows Defender 防毒軟體](https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/windows-defender-antivirus-in-windows-10)如需詳細資訊的文件] 媒體櫃。


時的功能、設定及管理大量相同的 Windows Defender 防毒軟體是在 Windows 10 或 Windows Server 2016 上，有幾個重要不同：

- Windows Server 2016 中[自動排除項目](https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/configure-server-exclusions-windows-defender-antivirus)會套用依據您的定義伺服器角色。
- 在 Windows Server 2016、Windows Defender 防毒軟體並不會關閉本身如果您執行其他防毒軟體 product。

[在 Windows Server 2016 上的 Windows Defender 防毒軟體](https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/windows-defender-antivirus-on-windows-server-2016)主題包含設定好與 Windows Server 2016 的特定設定資訊如何包括：

-   [讓介面](https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/windows-defender-antivirus-on-windows-server-2016#BKMK_UsingDef)

-   [確認已在執行 Windows Defender 防毒軟體]( https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/windows-defender-antivirus-on-windows-server-2016#BKMK_DefRun)

-   [反惡意程式碼定義的更新]( https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/windows-defender-antivirus-on-windows-server-2016#BKMK_UpdateDef)

-   [提交範例]( https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/windows-defender-antivirus-on-windows-server-2016#BKMK_DefSamples)

-   [設定自動排除項目]( https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/windows-defender-antivirus-on-windows-server-2016#BKMK_DefExclusions)
