## <a name="guidance-on-disabling-system-services-on-windows-server-2016-with-desktop-experience"></a>停用桌面體驗與 Windows Server 2016 上的系統服務指導方針

適用於： Windows Server 2016

Windows 作業系統包含許多系統服務提供重要的功能。 其他服務了不同的預設開機原則： 一些會開始預設 （自動），部分時所需 （手動），某些都預設停用，必須在之前可以執行明確支援。 仔細選擇預設值是每個服務來針對一般的平衡效能、 功能和安全性。

不過，有些企業針對可能會想更多安全性焦點款項他們的 Windows 電腦和降低絕對最小值，其攻擊，因此可能會想要完全停用所有服務，不需要在他們的特定環境中的伺服器。 適用於這些針對，Microsoft® 提供相關的服務可以放心地停用為這個項目的隨附指導方針。

Windows Server 2016 的桌面體驗 （除非終端使用者使用的傳統型取代） 是指導方針。 系統每個服務分類，如下所示：

-   **必須停用：**安全性焦點企業很可能會想要停用這項服務，不執行它的功能 （看到下方的其他詳細資料）。
- **確定要停用：**這項服務提供的部分，但並非所有企業，而且不使用的安全性相關的焦點企業安全地可以停用它。
- **不要停用：**停用此服務會影響不可或缺的功能或避免角色特定或功能無法正常運作。 因此它應該不會停用。
-  **（不指導方針）：**停用這些服務的影響不完整評估。 因此，不應該變更的預設設定，這些服務。


針對可以設定，來停用安全性範本使用他們的群組原則中或 PowerShell 自動化選取的服務他們的 Windows 電腦和伺服器。 有時候，指南包含特定的群組原則設定的另一種服務本身停用直接，停用服務的功能。

Microsoft 建議，針對停用下列服務及桌面體驗與 Windows Server 2016 上他們各自排定的工作：

服務： 
1. Xbox Live 驗證管理員
2. Xbox Live 重大儲存

排程的工作： 
1. \Microsoft\XblGameSave\XblGameSaveTask
2. \Microsoft\XblGameSave\XblGameSaveTaskLogon

(您也可以存取的資訊來檢視附加的 Microsoft Excel 試算表詳細本文中的所有服務：[系統服務停用桌面體驗與 Windows Server 2016 上的指導方針](https://msdnshared.blob.core.windows.net/media/2017/05/Service-management-WS2016.xlsx))

<br />

### <a name="disabling-services-not-installed-by-default"></a>停用預設尚未安裝的服務。

Microsoft 建議針對套用原則來停用預設不會安裝的服務。
-  如果已安裝的功能，通常需要服務。 安裝該功能或服務的需要系統管理員權限。 不允許功能安裝，而不服務開機。
-  安裝類似第三方相同，可能是一個更高的安全性風險從系統管理員 （或非系統管理員合計），不會停止封鎖 Microsoft Windows 服務。
-  基準或停用非預設 Windows 服務 (例如，W3SVC) 的基準可讓某些員蓄意的印象技術 (例如，IIS) 基本上就不安全，並不應該使用。
-  如果永遠不會安裝的功能 （和服務），這只是將不必要大量基準和驗證的工作。
