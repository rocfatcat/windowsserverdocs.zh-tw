# <a name="overview-of-disk-management"></a>磁碟管理概觀

> **適用於：**Windows 10、Windows 8.1、Windows Server (半年度管道)、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

[磁碟管理] 是管理硬碟、磁碟區或其內含之磁碟分割的系統公用程式。 您可以使用 [磁碟管理] 來初始化磁碟、建立磁碟區，以及利用 FAT、FAT32 或 NTFS 檔案系統來格式化磁碟區。 [磁碟管理] 可讓您執行大部分磁碟相關工作，而不需重新啟動系統或干擾使用者。 大部分的設定變更都會立即生效。

如需詳細資訊，請參閱下列主題：

-   [管理磁碟](manage-disks.md)
-   [管理基本磁碟區](manage-basic-volumes.md)
-   [疑難排解磁碟管理](troubleshooting-disk-management.md)
