# <a name="setting-file-server-resource-manager-options"></a>設定檔案伺服器資源管理員選項

> 適用于： Windows Server 2019、Windows Server 2016、Windows Server (半年通道) 、Windows Server 2012 R2、Windows Server 2012、Windows Server 2008 R2

您可以在 **\[檔案伺服器資源管理員選項\]** 對話方塊中設定檔案伺服器資源管理員的一般選項。 這些設定會使用在整個節點上，當您使用配額、檢測檔案或產存放裝置報告時，可以修改其中一些設定。

本節包含下列主題：

-   [設定電子郵件通知](configure-email-notifications.md)
-   [設定通知限制](configure-notification-limits.md)
-   [設定存放裝置報告](configure-storage-reports.md)
-   [設定檔案檢測稽核](configure-file-screen-audit.md)
