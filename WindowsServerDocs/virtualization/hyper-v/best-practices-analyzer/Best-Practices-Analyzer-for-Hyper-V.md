# <a name="best-practices-analyzer-for-hyper-v"></a>適用於 HYPER-V 的最佳做法分析程式

>適用於：Windows Server 2016
  
在 Windows 管理中，*最佳做法*是專家定義在一般情況下最理想的伺服器設定方式的指導方針。 最佳做法違規、 甚至是重大的、 可能不一定會造成問題。 但是，它們可以表示伺服器設定，可能會導致效能不佳、 可靠性差、 未預期的衝突、 安全性風險增加或其他潛在的問題。  
  
最佳做法分析程式掃描您的電腦使用這些最佳作法為基礎的規則，並報告結果。 每個最佳作法規則包括如何符合規則的詳細資料。 本節將記載這些規則，以協助您了解適用於 HYPER-V 的最佳作法以及掃描尋找不符合最佳作法的條件時解譯結果。  
  
如需最佳做法分析程式的詳細資訊，請參閱[Best Practices Analyzer](https://go.microsoft.com/fwlink/?LinkId=122786)。  
  
## <a name="about-hyper-v"></a>關於 Hyper-V  
HYPER-V 可讓您在上一部實體電腦，同時執行多個作業系統執行虛擬機器中的每一個。 虛擬機器可協助您更有效率又有彈性地使用您的運算資源。 如需有關 HYPER-V 的詳細資訊，請參閱[Windows Server 2016 上的 HYPER-V](../Hyper-V-on-Windows-Server.md)。
