# <a name="enable-or-disable-checkpoints-in-hyper-v"></a>啟用或停用 HYPER-V 中的檢查點

>適用於：Windows 10，Windows Server 2016、 Microsoft HYPER-V Server 2016、windows Server 2019，Microsoft HYPER-V Server 2019
  
您可以選擇啟用或停用每個虛擬機器的檢查點。  
  
1.  在  **HYPER-V 管理員**，以滑鼠右鍵按一下 虛擬機器，然後按一下**設定**。  
  
2.  底下**管理**區段中，選取**檢查點**。  
  
3.  若要讓此虛擬機器要採取的檢查點，請確定**啟用檢查點**已選取。 若要停用檢查點，請清除**啟用檢查點**核取方塊。  
  
4.  按一下 **\[套用\]** 以套用變更。 完成之後，請按一下 **\[確定\]** 以關閉對話方塊。
