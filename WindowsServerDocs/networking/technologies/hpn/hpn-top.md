# <a name="high-performance-networking-hpn"></a>高效能的網路功能 (HPN)

高效能網路 (HPNs) 即時資料處理需求中扮演的角色。 比方說，活動，例如資料中心複寫、 資料中心嚴重損壞修復，以及高效能分散式運算都需要大量資料傳輸和低網路延遲。 HPNs 動態連接功能進行高效能的網路資源，可更容易管理。

  - [軟體和硬體 (SH) 整合功能與技術](hpn-software-hardware-features.md)

  - [硬體只 （主機） 功能與技術](hpn-hardware-only-features.md)
