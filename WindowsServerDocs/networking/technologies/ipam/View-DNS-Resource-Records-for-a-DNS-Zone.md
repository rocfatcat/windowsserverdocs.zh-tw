# <a name="view-dns-resource-records-for-a-dns-zone"></a>檢視 DNS 資源記錄 DNS 區域

>適用於：Windows Server（以每年次管道）、Windows Server 2016

若要檢視 DNS 區域 DNS 資源記錄 IPAM client 主控台中，您可以使用本主題。  
  
資格在**系統管理員**，或相當於，才能執行此程序最小值。  
  
### <a name="to-view-dns-resource-records-for-a-zone"></a>若要檢視 DNS 資源記錄區域  
  
1.  在伺服器管理員中，按一下**IPAM**。 顯示 IPAM client 主機。  
  
2.  在瀏覽窗格中，在**監視和管理**，按一下 [ **DNS 區域**。  瀏覽窗格中分為左上角的瀏覽窗格中，較低的瀏覽窗格。  
  
3.  在較低的瀏覽窗格中，按一下 [**向前查詢**，然後展開網域和區域清單以尋找並選取您想要檢視的區域。 例如，如果您安裝名為都柏林區域，按一下**都柏林**。  
  
    ![選取您想要檢視的區域](../../media/View-DNS-Resource-Records-for-a-DNS-Zone/ipam_DNSzones_01a.jpg)  

  
4.  在 [顯示] 窗格中，預設檢視是區域的 DNS 伺服器。 若要變更檢視中，按一下**目前的檢視**，然後按一下 [**資源記錄**。  
  
    ![資源記錄變更檢視](../../media/View-DNS-Resource-Records-for-a-DNS-Zone/ipam_Zone_RR_02.jpg)  
  
5.  區 DNS 資源錄會顯示。 若要篩選記錄，輸入您想要尋找中的文字**篩選**。  
  
    ![篩選來輸入文字](../../media/View-DNS-Resource-Records-for-a-DNS-Zone/ipam_DNSzones_01c.jpg)  
  
6.  若要篩選資源記錄使用碼表進行輸入，存取範圍，或其他條件，請按一下**[新增條件**，並進行條件清單中的選取，然後按一下 [**新增**。  
  
    ![使用用以篩選](../../media/View-DNS-Resource-Records-for-a-DNS-Zone/ipam_DNSzones_01d.jpg)
