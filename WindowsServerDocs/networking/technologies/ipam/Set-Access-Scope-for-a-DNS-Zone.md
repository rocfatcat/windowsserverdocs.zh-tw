# <a name="set-access-scope-for-a-dns-zone"></a>將存取範圍 DNS 區域的設定

>適用於：Windows Server（以每年次管道）、Windows Server 2016

若要使用 IPAM client 主機設定存取範圍 DNS 區域的您可以使用此主題。  
  
資格在**系統管理員**，或相當於，才能執行此程序最小值。  
  
### <a name="to-set-the-access-scope-for-a-dns-zone"></a>設定存取 DNS 區域的領域  
  
1.  在伺服器管理員中，按一下**IPAM**。 顯示 IPAM client 主機。  
  
2.  在瀏覽窗格中，按一下**DNS 區域**。 在 [顯示] 窗格中，以滑鼠右鍵按一下您要變更存取範圍。，然後再按一下 [DNS 區域**設定存取範圍**。  
  
    ![將存取範圍設定](../../media/Set-Access-Scope-for-a-DNS-Zone/ipam_SetAccessScopeOfZone_02.jpg)  
  
3.  **設定存取範圍**對話方塊。 如果您需要為您的部署，按一下 [取消選取**繼承存取範圍家長的**。 在**選擇存取範圍**，選取項目，然後按一下 [ **[確定]**。  
  
    ![選取存取領域](../../media/Set-Access-Scope-for-a-DNS-Zone/ipam_SetAccessScopeOfZone_03.jpg)  
  
4.  在 IPAM client 主機顯示窗格中，請確認存取範圍區域的變更。  
  
    ![請確認存取範圍區域的變更](../../media/Set-Access-Scope-for-a-DNS-Zone/ipam_SetAccessScopeOfZone_04.jpg)
