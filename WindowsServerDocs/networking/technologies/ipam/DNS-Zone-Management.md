# <a name="dns-zone-management"></a>管理 DNS 區域

>適用於：Windows Server（以每年次管道）、Windows Server 2016

本主題提供使用 IPAM client 主控台管理 DNS 區域的相關資訊。  
  
> [!NOTE]  
> 本主題中，除了下列的 DNS IPAM 區域管理主題可在本區段中。  
>   
> -   [建立 DNS 區域](../../technologies/ipam/Create-a-DNS-Zone.md)  
> -   [編輯 DNS 區域](../../technologies/ipam/Edit-a-DNS-Zone.md)  
> -   [檢視 DNS 資源記錄 DNS 區域](../../technologies/ipam/View-DNS-Resource-Records-for-a-DNS-Zone.md)  
> -   [檢視 DNS 區域](../../technologies/ipam/View-DNS-Zones.md)  
  
在 Windows Server 2016 中的 IPAM 部署時，您可以使用 IPAM 管理 DNS 區域。  
  
在 IPAM 主控台中，您可以檢視 DNS 資源特定 DNS 區域，記錄及篩選記錄根據類型、IP 位址、資源記錄資料，以及其他篩選的選項。 此外，您可以編輯 DNS 資源記錄特定區域
