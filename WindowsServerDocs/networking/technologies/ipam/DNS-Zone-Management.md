# <a name="dns-zone-management"></a>DNS 區域管理

>適用於：Windows Server （半年通道），Windows Server 2016

本主題提供有關使用 IPAM 用戶端主控台來管理 DNS 區域的資訊。  
  
> [!NOTE]  
> 本主題中，除了下列 IPAM DNS 區域管理主題有這一節。  
>   
> -   [建立 DNS 區域](../../technologies/ipam/Create-a-DNS-Zone.md)  
> -   [編輯 DNS 區域](../../technologies/ipam/Edit-a-DNS-Zone.md)  
> -   [檢視 DNS 區域的 DNS 資源記錄](../../technologies/ipam/View-DNS-Resource-Records-for-a-DNS-Zone.md)  
> -   [檢視 DNS 區域](../../technologies/ipam/View-DNS-Zones.md)  
  
當您部署 Windows Server 2016 中的 IPAM 時，您可以使用 IPAM 來管理 DNS 區域。  
  
在 IPAM 主控台中，您可以檢視特定的 DNS 區域的 DNS 資源記錄，並篩選依類型、 IP 位址、 資源記錄資料和其他篩選選項的記錄。 此外，您可以編輯特定區域的 DNS 資源記錄
