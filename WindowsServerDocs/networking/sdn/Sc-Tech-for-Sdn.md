# <a name="system-center-technologies-for-sdn"></a>SDN 的 System Center 技術

>適用於：Windows Server （半年通道），Windows Server 2016

System Center 包含下列技術，用於使用軟體定義網路 (SDN):  
  
-   [System Center Operations Manager](#bkmk_scom)  
  
-   [System Center Virtual Machine Manager](#bkmk_scvmm)  
  
  
## <a name="bkmk_scom"></a>System Center Operations Manager  
System Center 2016 Operations Manager 提供基礎結構監視功能，有彈性且符合成本效益、 協助確保可預測的效能和重要的應用程式的可用性和可全面監視您的資料中心和雲端私用和公用。  
  
如需詳細資訊，請參閱 < [Operations Manager](https://technet.microsoft.com/library/hh205987.aspx)。  
  
## <a name="bkmk_scvmm"></a>System Center Virtual Machine Manager  
使用 System Center 2016 Virtual Machine Manager (VMM) 中，您可以：

- 佈建和管理大規模的虛擬網路。
- 部署和管理 SDN 基礎結構，包括網路控制站、 軟體負載平衡器和閘道。 
- 定義和集中控制虛擬網路原則，並將它們連結至您的應用程式或工作負載。 

  部署或移動工作負載時，網路設定本身會自動調整。 這是重要的，因為這可免除手動重新設定網路硬體的必要，藉此降低操作的複雜性，並節省您寶貴的資源以運用於較需要的工作。 
- 可協助您控制虛擬網路，包括定義關鍵應用程式和工作負載的保證的頻寬的能力之間的流量。  
  

如需詳細資訊，請參閱 <<c0> [ 設定 VMM 光纖中軟體定義網路 (SDN) 基礎結構](https://technet.microsoft.com/system-center-docs/vmm/scenario/sdn-overview)。
