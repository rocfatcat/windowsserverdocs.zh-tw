# <a name="enable-hash-publication-for-file-servers"></a>支援檔案伺服器 Hash 的發行

>適用於：Windows Server（以每年次管道）、Windows Server 2016

您可以讓 BranchCache hash 發行一個檔案伺服器或多個檔案伺服器上。  
  
-   要 hash 發行一個檔案伺服器使用群組原則的本機電腦上，請查看[適用於非網域成員檔案伺服器讓 Hash 發行](../../branchcache/deploy/Enable-Hash-Publication-for-Non-Domain-Member-File-Servers.md)。  
  
-   若要使用群組原則網域多個檔案伺服器 hash 發行，請查看[讓 Hash 發行網域成員檔案伺服器的](../../branchcache/deploy/Enable-Hash-Publication-for-Domain-Member-File-Servers.md)。  
  
> [!NOTE]  
> 如果您有多個檔案伺服器和您想要讓共用，請依據 hash 發行，而不是所有共用啟用 hash 發行，您可以使用指示主題中的[適用於非網域成員檔案伺服器讓 Hash 發行](Enable-Hash-Publication-for-Non-Domain-Member-File-Servers.md)。
