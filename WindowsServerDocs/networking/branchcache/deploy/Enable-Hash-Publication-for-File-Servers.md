# <a name="enable-hash-publication-for-file-servers"></a>啟用檔案伺服器的雜湊發行

>適用於：Windows Server （半年通道），Windows Server 2016

您可以啟用 BranchCache 的雜湊發行一個檔案伺服器或多個檔案伺服器上。  
  
-   若要啟用使用本機電腦群組原則的一個檔案伺服器上的雜湊發行集，請參閱[啟用非網域成員的檔案伺服器的雜湊發行](../../branchcache/deploy/Enable-Hash-Publication-for-Non-Domain-Member-File-Servers.md)。  
  
-   若要啟用雜湊發行集上使用網域群組原則的多個檔案伺服器，請參閱[啟用網域成員的檔案伺服器的雜湊發行](../../branchcache/deploy/Enable-Hash-Publication-for-Domain-Member-File-Servers.md)。  
  
> [!NOTE]  
> 如果您有多個檔案伺服器，而且您想要啟用雜湊發行集，每個共用，而不是啟用的雜湊發行集之所有共用，您可以使用指示 > 主題中[啟用非網域成員的檔案伺服器的雜湊發行](Enable-Hash-Publication-for-Non-Domain-Member-File-Servers.md).
