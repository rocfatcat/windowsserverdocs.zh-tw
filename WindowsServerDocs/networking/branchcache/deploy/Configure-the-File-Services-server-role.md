# <a name="configure-the-file-services-server-role"></a>設定檔服務伺服器角色

>適用於：Windows Server（以每年次管道）、Windows Server 2016

您可以部署 BranchCache 檔案伺服器端內容伺服器上的電腦是執行 Windows Server 2016 和檔案服務伺服器角色與**網路的檔案 BranchCache**安裝角色服務。  
  
-   已經不會有安裝檔案服務的電腦上安裝 BranchCache 內容伺服器，查看[安裝新的檔案伺服器 Content 伺服器以](../../branchcache/deploy/Install-a-New-File-Server-as-a-Content-Server.md)。  
  
-   已將檔案服務伺服器角色的電腦上安裝 BranchCache 內容伺服器，查看[設定為 Content 伺服器現有檔案伺服器](../../branchcache/deploy/Configure-an-Existing-File-Server-as-a-Content-Server.md)。
