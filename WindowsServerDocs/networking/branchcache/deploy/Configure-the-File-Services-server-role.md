# <a name="configure-the-file-services-server-role"></a>設定檔案服務伺服器角色

>適用於：Windows Server （半年通道），Windows Server 2016

您可以部署 BranchCache 的檔案伺服器型內容伺服器上執行 Windows Server 2016 和與檔案服務伺服器角色的電腦**網路檔案的 BranchCache**安裝角色服務。  
  
-   若要安裝 BranchCache 內容伺服器還沒有安裝檔案服務的電腦上，請參閱[安裝新的檔案伺服器做為內容伺服器](../../branchcache/deploy/Install-a-New-File-Server-as-a-Content-Server.md)。  
  
-   若要在已設定檔案服務伺服器角色的電腦上安裝 BranchCache 內容伺服器，請參閱[設定為內容伺服器的現有檔案伺服器](../../branchcache/deploy/Configure-an-Existing-File-Server-as-a-Content-Server.md)。
