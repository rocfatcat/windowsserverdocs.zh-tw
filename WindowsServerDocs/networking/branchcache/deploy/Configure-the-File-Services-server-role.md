# <a name="configure-the-file-services-server-role"></a>設定檔案服務伺服器角色

>適用於：Windows Server (半年通道)、Windows Server 2016

您可以在執行 Windows Server 2016 的電腦上，以及已安裝網路檔案的**branchcache**角色服務的檔案服務伺服器角色上，部署 branchcache 檔案伺服器內容伺服器。  
  
-   若要在尚未安裝檔案服務的電腦上安裝 BranchCache 內容伺服器，請參閱[安裝新的檔案伺服器做為內容伺服器](../../branchcache/deploy/Install-a-New-File-Server-as-a-Content-Server.md)。  
  
-   若要在已設定檔案服務伺服器角色的電腦上安裝 BranchCache 內容伺服器，請參閱[將現有的檔案伺服器設定為內容伺服器](../../branchcache/deploy/Configure-an-Existing-File-Server-as-a-Content-Server.md)。
