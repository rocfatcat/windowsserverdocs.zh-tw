# <a name="branchcache-network-shell-and-windows-powershell-commands"></a>殼層 BranchCache 網路及 Windows PowerShell 命令

>適用於：Windows Server（以每年次管道）、Windows Server 2016

在 Windows Server 2016，您可以設定和管理 BranchCache BranchCache 使用 Windows PowerShell 或命令網路 Shell (Netsh)。  
  
在未來的 Windows 版本，Microsoft 可能會移除 netsh 功能 BranchCache。 如果您目前使用 netsh 設定及管理 BranchCache 和其他網路技術 Microsoft 建議您轉換到 Windows PowerShell。  
  
Windows PowerShell 和 netsh 命令參照位於在下列位置。 雖然這兩個命令參照已更早版本與 Windows Server 2016 發行的作業系統，這些參考是此作業系統的正確的。  
  
-   [在 Windows Server 2008 R2 BranchCache Netsh 命令](https://technet.microsoft.com/library/dd979561(v=ws.10))  
  
-   [Windows PowerShell 中的 BranchCache Cmdlet](https://technet.microsoft.com/library/hh848392.aspx)適用於 Windows Server 2012。  
  
> [!TIP]  
> Windows PowerShell 命令提示字元中，檢視 BranchCache Windows PowerShell 命令的清單，輸入`Get-Command -Module BranchCache`在 Windows PowerShell 命令提示字元中，，然後按下 ENTER。
