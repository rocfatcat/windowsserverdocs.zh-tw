# <a name="azure-active-directory-connect"></a>Azure Active Directory Connect


Azure AD Connect 會將您的內部部署目錄與 Azure Active Directory 整合。 這可讓您為您的使用者提供與 Azure AD 整合之 Office 365、Azure 和 SaaS 應用程式的通用身分識別。 .  
  
如需詳細資訊，請參閱[整合您的內部部署身分識別與 Azure Active Directory](https://azure.microsoft.com/documentation/articles/active-directory-aadconnect/)。
