# <a name="azure-active-directory-connect"></a>Azure Active Directory Connect

>適用於：Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

Azure AD Connect 會整合您的內部部署目錄與 Azure Active Directory。 這可讓您的 Office 365 和 SaaS 應用程式與 Azure AD 整合的使用者提供通用身分識別。 .  
  
如需詳細資訊，請參閱[整合內部部署身分識別與 Azure Active Directory](https://azure.microsoft.com/documentation/articles/active-directory-aadconnect/)。
