# <a name="windows-server-ad-fs-deployment-guide"></a>Windows Server AD FS 部署指南


您可以使用 Windows Server 2016 和 2012 R2 作業系統的 Active Directory 同盟服務 \(AD FS\) 來建立同盟身分識別管理解決方案，將分散式識別、驗證和授權服務延伸到跨組織和平臺界限的 Web\-應用程式。 藉由部署 AD FS，您可以將組織現有的身分識別管理功能延伸至網際網路。
