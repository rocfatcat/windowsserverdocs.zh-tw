# <a name="ad-fs-overview"></a>AD FS 概觀

Active Directory 同盟服務（AD FS）可安全地跨安全性和企業界限共用數位身分識別和存取權，藉以啟用同盟身分識別和存取管理。 AD FS 擴充了使用單一登入功能的功能，可在單一安全性或企業界限內提供給網際網路面向的應用程式，讓客戶、合作夥伴和供應商在存取組織的 web 應用程式。
