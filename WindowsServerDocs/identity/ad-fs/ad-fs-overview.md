# <a name="ad-fs-overview"></a>AD FS 概觀

Active Directory 同盟服務（AD FS）可安全地跨安全性和企業界限共用數位身分識別和存取權，藉以啟用同盟身分識別和存取管理。 AD FS 擴充了使用單一登入功能（可在單一安全性或企業界限內取得）以進行網際網路對向應用程式的能力，讓客戶、合作夥伴和供應商能夠在存取組織的 web 應用程式時，提供簡化的使用者體驗。
