# <a name="evaluating-ad-ds-deployment-strategy-examples"></a>正在評估 AD DS 部署策略範例

>適用於：Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

請參考下列虛構公司，以 Contoso 醫藥在其環境中部署 Active Directory Domain Services (AD DS) 的範例。 環境以 Contoso 網域四個所組成。 森林功能層級是 Windows Server 2003。 下圖顯示目前的網域結構 Contoso 組織。  
  
![AD DS 部署策略](media/Evaluating-AD-DS-Deployment-Strategy-Examples/3dd79e00-48f8-4927-989c-c55a79caf1be.gif)  
  
檢視之後，它是現有的環境並找出部署目標，以 Contoso 建立下列 AD DS 部署策略的使用者：  
  
-   升級到 Windows Server 2008 網域的 Windows Server 2003 網域。  
  
-   進階的 AD DS 功能可讓提高以 Windows Server 2008 的網域和森林功能層級。  
  
-   重新建構樹系整合與 emea.concorp.contoso.con 網域網域中的 africa.concorp.contoso.com 網域。  
  
提高以 Windows Server 2008 的樹系功能層級會讓 Contoso 充分利用 AD DS 新功能。 如下所示重建網域中的樹系，將會減少的所需的管理網域管理。  
  
![AD DS 部署策略](media/Evaluating-AD-DS-Deployment-Strategy-Examples/1c061755-413d-452d-b121-6910f8555327.gif)
