# <a name="finding-additional-resources-for-enabling-advanced-features"></a>尋找啟用進階功能的其他資源

>適用於：Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

您可以在 Windows Server 2008 Technical Library 中找到有關 Active Directory 網域服務 (AD DS) 的下列文件：  
  
-   如需部署 Windows Server 2008 樹系根網域的詳細資訊，請參閱[部署 Windows Server 2008 樹系根網域\[LH\]](assetId:///92406e8d-dc1c-4740-a00a-2c4032896dd1)。  
  
-   如需有關如何升級到 Windows Server 2008 的 Active Directory 網域的詳細資訊，請參閱[至 Windows Server 2008 AD DS 網域升級 Active Directory 網域\[LH\]](assetId:///9c91be5f-df14-40b2-b176-2b1852a51e61)。  
  
-   如需部署 AD DS 的詳細資訊，請參閱逐步指南的 Windows Server 2008 Active Directory 網域服務安裝和移除 ([https://go.microsoft.com/fwlink/?LinkId=100492](https://go.microsoft.com/fwlink/?LinkId=100492))。
