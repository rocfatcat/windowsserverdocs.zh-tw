# <a name="domain-controller-location"></a>網域控制站位置

>適用於：Windows Server 2016 中，Windows Server 2012 R2 中，Windows Server 2012

用戶端會使用網域名稱系統 (DNS)，找出完成的作業，例如處理登入要求，或已發行的資源在目錄中搜尋的網域控制站。 網域控制站可協助用戶端的 DNS 中登錄的各種不同的記錄和其他電腦找到它們。 這些記錄會統稱為定位器記錄。  
  
尋找其他網域控制站，以及執行工作，例如複寫網域控制站也會使用 DNS。 網域控制站找到其他網域控制站的程序是供用戶端找到網域控制站的程序相同。
