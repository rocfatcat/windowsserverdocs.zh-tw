# <a name="identifying-your-ad-ds-design-and-deployment-requirements"></a>找出您 AD DS 設計和部署需求

>適用於：Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

執行高階評估的您目前的環境並正確地找出您 Active Directory Domain Services (AD DS) 部署工作務必 AD DS 部署策略的成功。  
  
AD DS 部署策略您現有的網路設定而定。 例如，如果您的組織目前執行 Windows Server 2003，您可以升級您的作業系統以 Windows Server 2008。 您的部署程序可能需要重建現有 Active Directory 樹系或 Active Directory 樹系之間的網域。 您可能要部署的 Windows Server 2008 AD DS 之後，或變更組織或企業數之後重建您現有的網域。  
  
-   [AD DS 設計需求](../../ad-ds/plan/AD-DS-Design-Requirements.md)  
  
-   [AD DS 部署需求](../../ad-ds/plan/AD-DS-Deployment-Requirements.md)
