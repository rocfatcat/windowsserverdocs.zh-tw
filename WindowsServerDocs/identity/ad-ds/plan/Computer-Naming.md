# <a name="computer-naming"></a>電腦命名

>適用於：Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

當執行 Windows 2000、Windows XP、Windows Server 2003、Windows Server 2008 或 Windows Vista 作業系統的電腦加入網域，預設電腦指定本身的名稱。 指派本身名稱包含主機電腦的名稱（也就是，在系統屬性電腦名稱）和 Active Directory 網域的電腦所加入的網域名稱系統」(DNS) 名稱（也就是主要 DNS 尾碼系統屬性）。 主機名稱和 DNS 網域名稱的全名連續稱為的完整的網域名稱 (FQDN)。 例如，主機名稱 Server1 電腦加入網域 corp.contoso.com，FQDN 的電腦 server1.corp.contoso.com。  
  
如果電腦已經有不同的 DNS 網域名稱靜態輸入 DNS 區域或登記整合 DNS/動態主機設定通訊協定」(DHCP) 伺服器服務，電腦的 FQDN 是不同於之前登記完畢名稱。 可以將電腦參考任一名稱。  
  
如需命名規格 Active Directory Domain Services (AD DS)，查看針對電腦、網域、網站及組織單位 Active Directory 命名規格 ([https://go.microsoft.com/fwlink/?LinkID=106629](https://go.microsoft.com/fwlink/?LinkID=106629))。
