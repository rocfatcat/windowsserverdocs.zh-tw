# <a name="understanding-active-directory-site-topology"></a>了解 Active Directory 站台拓撲

>適用於：Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

您的網站拓撲會大幅影響網路的效能，以及您的使用者存取網路資源的能力。 在您開始設計您的網站拓撲之前，請先熟悉 Windows Server 2008 中的網站功能、組織通常使用的不同網路拓撲、網站拓撲擁有者的角色，以及一些 Active Directory 複寫概念。
