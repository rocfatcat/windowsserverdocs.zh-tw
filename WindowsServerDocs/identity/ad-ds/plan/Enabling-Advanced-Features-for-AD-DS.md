# <a name="enabling-advanced-features-for-ad-ds"></a>進階的功能讓 AD ds

>適用於：Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

Active Directory Domain Services (AD DS) 可讓您以您的環境進階的功能引入提高網域或森林功能層級。 若要使用進階的 AD DS 功能，您必須找出作業系統網域控制站在您的環境中執行。   
  
您必須也會根據您現有的基礎結構組織判斷的最佳功能層級，然後提高網域或森林功能等級視。 您可以提升功能時網域或森林中的所有網域控制站都執行適當的 Windows 版本。 提升功能等級可讓您要的新功能，但它也會限制的 Windows 作業系統，您可以在網域控制站在您的環境中執行的版本。
