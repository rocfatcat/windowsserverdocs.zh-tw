# <a name="ad-ds-troubleshooting"></a>對 AD DS 進行疑難排解

>適用於：Windows Server 2016、Windows Server 2012 R2

本節包含診斷和修正 Active Directory 複寫可能發生之問題的疑難排解建議和程式。

本內容主要著重于對目錄服務事件記錄檔訊息的回應，以及可能由 Repadmin 和 Dcdiag.exe 工具報告的工具型錯誤訊息。 這些工具可在執行 Windows Server 2016 或 2012 R2 的所有網域控制站上使用。 您也可以在執行 Windows 10 的成員伺服器上安裝遠端伺服器管理工具（RSAT）。

如需安裝 RSAT 的詳細資訊，請參閱[遠端伺服器管理工具](https://docs.microsoft.com/windows-server/remote/remote-server-administration-tools)一文。
