# <a name="multipoint-services"></a>MultiPoint 服務
MultiPoint 服務解決方案，可讓多個使用者，各有自己的獨立且熟悉的 Windows 體驗，同時共用一部電腦。

由監視器、鍵盤和滑鼠組成的使用者站台，可透過 USB 或視訊纜線直接連接到主機電腦。 取得適當的授權時，MultiPoint 服務會是正版的 Microsoft 已發行的軟體產品，因為您有資格獲得由 Microsoft 或授權的合作夥伴的支援。 這可讓您完整功能的 Windows，存取所有最新的更新和信賴度，您會達成您所預期的體驗。

MultiPoint 服務可讓多個使用者共用一部電腦，因為它可以提供傳統運算案例，其中每位使用者都有自己的電腦較低成本的替代方案。 MultiPoint 服務還提供了讓您輕鬆管理解決方案為 MultiPoint 服務系統管理，又稱為 MultiPoint 管理員，並讓您輕鬆管理解決方案，一天\-至\-天內，又稱為 MultiPoint儀表板。
