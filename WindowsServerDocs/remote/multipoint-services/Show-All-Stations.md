# <a name="show-all-stations"></a>顯示所有站台
身為 MultiPoint 服務系統上的管理員，您可能想要檢視所有站台連線到您的系統，包括非\-作用中或已登出的站台。 比方說，每個授權合約，您可能需要檢查連接 MultiPoint 服務系統的站台的數目。 您可以檢視這項資訊使用 MultiPoint 管理員。  
  
1.  在標準模式中，按一下 [桌面] 索引標籤。  
  
2.  在 [General Tasks (一般工作)] 下，按一下 [Show all stations (顯示所有站台)]。
