# <a name="show-all-stations"></a>顯示所有站台
身為 MultiPoint 服務系統的系統管理員，您可能會想要查看所有連線到您系統的工作站，包括非作用中 \- 或已登出的工作站。 例如，根據您的授權合約，您可能需要檢查已連線到 MultiPoint 服務系統的工作站數目。 您可以使用 MultiPoint 管理員來查看此資訊。

1.  在標準模式中，按一下 [桌面]**** 索引標籤。

2.  在 [General Tasks (一般工作)]**** 下，按一下 [Show all stations (顯示所有站台)]****。
