# <a name="view-user-connection-status"></a>檢視使用者連線狀態
使用 [**工作站**] 索引標籤，判斷標準或其他系統管理使用者與 MultiPoint 服務站的線上狀態。  
  
狀態值包括：  
  
-   **已登入**：使用者工作階段在站台上為使用中  
  
-   **已暫停**：使用者工作階段已暫停，但在電腦上仍為使用中。 使用者的桌面會話會保留到使用者重新登入為止  
  
-   **已登出**︰已登出的使用者不會顯示在 [站台] 索引標籤上  
  
若要查看站狀態，請開啟 [電臺模式] 中的 [MultiPoint 管理員]，然後按一下 [**工作站**]。
