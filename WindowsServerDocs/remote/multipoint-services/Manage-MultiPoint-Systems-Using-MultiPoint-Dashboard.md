# <a name="manage-multipoint-systems-using-multipoint-dashboard"></a>使用 MultiPoint 儀表板管理 MultiPoint 系統
在 MultiPoint 儀表板的 [**系統**] 索引標籤上，您可以執行下列動作：

- 重新開機或關閉選取的系統您可以重新開機或關閉選取的 MultiPoint 服務系統。 如需重新啟動或關閉所選系統的資訊，請參閱[重新啟動或關閉 MultiPoint 系統](Restart-or-Shut-Down-MultiPoint-Systems.md)主題。
- 重新對應選取的系統您可以重新對應選取的 MultiPoint 服務系統。 如需重新對應所選系統的詳細資訊，請參閱[重新對應選取的 MultiPoint 系統](Remap-Selected-MultiPoint-Systems.md)主題。
