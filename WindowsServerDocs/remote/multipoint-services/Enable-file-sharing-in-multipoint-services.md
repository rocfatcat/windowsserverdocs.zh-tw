# <a name="enable-file-sharing-in-multipoint-services"></a>啟用 MultiPoint 服務的檔案共用
您可以透過兩種方式讓 MultiPoint 工作站上的使用者共用檔案：  
  
-   **如果您的網路上有檔案伺服器，** 建議您在檔案伺服器上建立共用資料夾。  
  
-   **如果您有一部 2-3 MultiPoint 伺服器的小型網路，而沒有專用的檔案伺服器，** 則其中一部 MultiPoint 伺服器可作為所有執行 MultiPoint 服務之其餘電腦的檔案伺服器。 在該伺服器上建立共用資料夾，然後為該伺服器上的所有使用者建立本機使用者帳戶。 共用資料夾可以位於原始的內部磁片磁碟機上，或者您也可以將額外的內部或外部磁片磁碟機連接到電腦。
