# <a name="enable-file-sharing-in-multipoint-services"></a>啟用 MultiPoint 服務的檔案共用
您可以允許使用者在共用檔案有兩種您 MultiPoint 站台上：  
  
-   **如果您在網路上，檔案伺服器**建議您在檔案伺服器上建立共用的資料夾。  
  
-   **如果您有 2 到 3 MultiPoint 伺服器使用，但沒有專用的檔案伺服器的小型網路**其中一部 MultiPoint 伺服器可做為執行 MultiPoint 服務的所有其餘電腦的檔案伺服器。 在該伺服器上，建立共用的資料夾，然後建立該伺服器上的 所有使用者的本機使用者帳戶。 共用的資料夾可在原始的內部磁碟機，或您可以將額外的內部或外部磁碟機附加至電腦。
