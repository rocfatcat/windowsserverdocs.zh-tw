# <a name="set-the-date-time-and-time-zone"></a>設定日期、時間與時區
安裝裝置驅動程式完成之後，請 MultiPoint 伺服器上設定的日期、 時間和時區。  
  
1.  從**開始**MultiPoint 伺服器、 開啟的畫面**控制台**。  
  
2.  底下**時鐘、 語言和地區**，按一下**設定的時間和日期**。  
  
3.  在 **日期和時間**索引標籤上，確認日期和時間。 如果不正確，按一下**日期和時間變更**，更新的日期和時間，然後按一下 **確定**。  
  
4.  底下**時區**，確認時區。 如果您不正確，請按一下**變更時區**，選取正確的時區，然後按一下**確定**。  
  
5.  按一下 **確定**以儲存您的設定，然後關閉對話方塊。
