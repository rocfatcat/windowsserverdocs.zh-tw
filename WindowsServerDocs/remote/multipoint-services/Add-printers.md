# <a name="add-printers"></a>新增印表機
使用本主題中的程序，可讓 MultiPoint 服務系統上的所有使用者使用本機印表機。  
  
> [!NOTE]  
> 如果您使用 MultiPoint 服務使用網域帳戶，使用者可以使用任何網路印表機，從其站台。  
  
1.  印表機連接到 Multipoint server。  
  
2.  設定共用印表機的印表機：  
  
    1.  以系統管理員身分登入 MultiPoint Server 的電腦。  
  
    2.  從 [開始]  畫面，開啟 [控制台] 。  
  
    3.  在控制台中，按一下**硬體**，然後按一下**裝置和印表機**。  
  
    4.  底下**印表機和傳真**，以滑鼠右鍵按一下 [印表機]，然後按一下**印表機內容**。  
  
    5.  按一下 [**共用**] 索引標籤。  
  
    6.  按一下 **共用這個印表機**，指定印表機的共用名稱，然後按一下**確定**。  
  
登入任何已連線到 Multipoint 服務電腦的站台的使用者將能夠看到並使用印表機。
