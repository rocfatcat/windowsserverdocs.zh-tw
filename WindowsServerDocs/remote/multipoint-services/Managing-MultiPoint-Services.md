# <a name="managing-multipoint-services"></a>管理 MultiPoint 服務
MultiPoint &trade; 服務可讓多個使用者，各自擁有各自獨立的 Windows 體驗，同時共用一部電腦。 使用者站台是由監視器、鍵盤和滑鼠組成，可透過 USB、視訊纜線或網路直接連接到主機電腦。
