# <a name="remap-all-stations"></a>重新對應所有站台
重新對應站台可讓您在鍵盤及滑鼠與監視器之間建立關聯。 當您重新對應所有工作站時，會清除原始設定，例如名稱和自動登入資訊。 所有本機使用者站台在進行重新對應時都會暫止。  
  
1.  以站模式開啟 MultiPoint 管理員，然後按一下 [**首頁**] 索引標籤。  
2.  在 [工作] 下，按一下 [Remap all stations] (重新對應所有站台)。  
3. 遵循站台畫面上的指示，在鍵盤與您系統中的站台之間建立關聯。
