# <a name="take-control-of-a-user-session"></a>接管使用者工作階段的控制權
身為 MultiPoint 儀表板使用者，您可以使用 [接管控制權] 功能，從遠端存取其他使用者的桌面來提供協助。  
  
1.  MultiPoint 儀表板上**首頁**索引標籤上，按一下您想要協助使用者桌面的縮圖影像。  
  
2.  在 [Assist (協助)] 索引標籤上，按一下 [Take Control (接管控制權)]。 您的桌面上隨即開啟使用者的桌面，您可使用您的鍵盤和滑鼠來瀏覽其桌面。  
  
3.  當您結束協助使用者之後，請按一下**停止**。  
  
    > [!NOTE]  
    > 您可能需要最小化使用者的桌面，以查看您的 MultiPoint 儀表板。
