# <a name="install-software-on-your-multipoint-services-system"></a>在 MultiPoint 服務系統上安裝軟體
當您以系統管理使用者身分登入時，您可以在主控台模式中或從工作站以工作站模式安裝新的程式。 不過，我們建議您以主控台模式安裝程式。  
  
您可以在執行 MultiPoint Server 的電腦上安裝新軟體，讓所有使用者都可以執行軟體，或只有您可以使用軟體，視軟體的安裝和授權選項而定。  
   
1.  以系統管理員身分登入 MultiPoint 服務電腦。  
  
2.  開啟 [MultiPoint 管理員]。  
  
3.  按一下 [**首頁**] 索引標籤，然後按一下 [**切換至主控台模式**]。  
  
4.  以系統管理員身分登入，並安裝您的應用程式。  
  
5.  在您完成安裝應用程式之後，請將電腦切換回站模式。 若要這麼做，請在 [**首頁**] 索引標籤上，按一下 [**切換到工作站模式**]。
