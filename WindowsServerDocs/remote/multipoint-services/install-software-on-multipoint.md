# <a name="install-software-on-your-multipoint-services-system"></a>在 MultiPoint 服務系統上安裝軟體
當您以系統管理使用者身分登入時，您就可以安裝新的程式，在主控台模式或，從站台，站台模式中。 不過，我們建議您在主控台模式下安裝程式。  
  
您可以安裝新軟體的電腦上執行 MultiPoint Server，讓所有使用者都可以都執行軟體，或使只有您可以在其中使用根據安裝的軟體，且授權的軟體的選項。  
   
1.  以系統管理員身分登入 MultiPoint 服務電腦。  
  
2.  開啟 MultiPoint 管理員。  
  
3.  按一下 **首頁**索引標籤，然後再按一下**切換至 主控台模式**。  
  
4.  以系統管理員身分，登入，並安裝您的應用程式。  
  
5.  完成安裝應用程式之後，請回到站台模式切換電腦。 若要這樣做，請在**首頁**索引標籤上，按一下**切換至 站台模式**。
