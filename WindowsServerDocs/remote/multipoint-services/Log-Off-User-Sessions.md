# <a name="log-off-user-sessions"></a>登出使用者工作階段
標準使用者 」、 「 MultiPoint 儀表板使用者和 「 系統管理使用者可以登入並登出他們的桌面工作階段，如同處理任何 Windows 工作階段。 此外，系統管理使用者和 MultiPoint 儀表板使用者可以結束所有 MultiPoint 服務系統上受監視的工作階段的使用者工作階段。  
  
1.  在 MultiPoint 儀表板中，按一下**首頁** 索引標籤。  
  
2.  執行下列其中一項：  
  
    -   若要登出單一使用者工作階段或選取的工作階段，按一下您想要結束，工作階段的縮圖影像，然後按一下左上方的下拉式選單。 按一下 [登出使用者]，然後按一下 [登出選取的使用者]。 您也可以以滑鼠右鍵按一下選取的縮圖，即可看到這個選項。
  
    -   若要登出所有使用者工作階段，請按一下左上方的下拉式選單，按一下**登出使用者**，然後按一下**登出所有使用者**。
