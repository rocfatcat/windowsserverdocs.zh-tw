# <a name="configure-stations-for-automatic-logon"></a>為站台設定自動登入
如果您想要可供任何人 – 您站台，而且您的使用者不需要私人資料夾來儲存其個人資料或個人化的桌面，您可以設定自動登入的站台。 自動登入自動登入 MultiPoint 服務啟動時，自動登入設定中具有指定的使用者帳戶。  
  
1.  從**開始**畫面上，開啟**MultiPoint 管理員**。  
  
2.  按一下 **站台**索引標籤，然後再按一下您想要設定自動登入的站台名稱。  
  
3.  在右窗格中，按一下**設定自動登入**。  
  
    設定自動登入頁面隨即開啟。  
  
4.  選取 **自動登入使用下列資訊**核取方塊，，然後輸入 使用者帳戶和密碼，使用自動登入。 按一下 [確定] 。  
  
    > [!NOTE]  
    > 您要用於自動登入的使用者帳戶必須有密碼。  
  
> [!NOTE]  
> 若要暫時登入已設定為使用不同的使用者帳戶自動登入的站台，停留在右上角的螢幕顯示垂直的功能表，按一下 [設定] 快速鍵，按一下 [電源] 圖示，然後按住 SHIFT 鍵並按一下 **</c0>中斷<spanclass="notranslate">**。</span> 登入提示出現之前，請按住 SHIFT 鍵。
