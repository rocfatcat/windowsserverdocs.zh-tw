# <a name="restart-or-shut-down-multipoint-systems"></a>重新啟動或關閉 MultiPoint 系統
您可以重新啟動或關閉 MultiPoint 服務系統中或在 MultiPoint 儀表板中的多個 MultiPoint 服務系統。  
  
## <a name="restart-a-multipoint-services-system-or-multiple-systems"></a>重新啟動 MultiPoint 服務系統或多個系統  
  
1.  按一下 **系統**。  
  
2.  按一下您要重新啟動的伺服器，然後在縮圖影像**硬體**索引標籤上，按一下**重新啟動**。  
  
## <a name="to-shut-down-a-multipoint-services-system-or-multiple-systems"></a>關閉一或多個 MultiPoint 服務系統  
  
1.  按一下 [**系統**] 索引標籤。  
  
2.  按一下您要重新啟動的伺服器，然後在縮圖影像**硬體**索引標籤上，按一下**關機**。
