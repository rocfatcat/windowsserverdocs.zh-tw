# <a name="project-a-station-to-other-stations"></a>將站台投影到其他站台
身為 MultiPoint 儀表板使用者，您可以將桌面投影到單一使用者的站台或所有的使用者 （非系統管理員） 站台。 當您想要向一位使用者或一組使用者示範工作時，此功能會很有用。  
  
## <a name="to-project-your-desktop-to-a-standard-users-station"></a>將您的桌面投影到標準使用者的站台  
  
1.  在 MultiPoint 儀表板中，按一下您要投影到站台之桌面的縮圖影像。  
2.  在上**首頁**索引標籤上，按一下**您的桌面**，然後按一下**專案選取 Desktop(s) 到您的桌面**。  
3.  若要結束投影，請按一下**停止**(在**投影** 索引標籤或功能區下方右手邊角)。  
  
## <a name="to-project-your-desktop-to-all-stations"></a>將您的桌面投影到所有站台  
  
1.  MultiPoint 儀表板上**首頁**索引標籤上，按一下**您的桌面**，然後按一下**專案所有 Desktop(s) 到您的桌面**。  
  
2.  若要結束投影，請按一下**停止**(在**投影** 索引標籤或功能區下方右手邊角)。  
  
## <a name="to-project-a-different-desktop-to-all-desktops"></a>將不同的桌面投影到所有桌面  
  
1.  按一下您要投影到所有桌面之桌面的縮圖影像。  
  
2.  在上**首頁**索引標籤上，按一下**Selected Desktop**，然後按一下**Selected Desktop**。 選取的桌面隨即顯示在所有桌面上。  
  
3.  若要結束投影，請按一下**停止**(在**投影** 索引標籤或功能區下方右手邊角)。
