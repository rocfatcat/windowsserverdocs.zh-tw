# <a name="install-updates"></a>安裝更新
我們建議您安裝更新（如果有的話）。 安裝更新需要網際網路連線。  

1.  從 [開始] 畫面，開啟 [控制台]。  
  
2.  在 [控制台] 中輸入**updates**，然後按一下 [**檢查更新**]。  
  
3.  如果 Windows Update 網站列出電腦上所需的任何更新，請安裝更新。
