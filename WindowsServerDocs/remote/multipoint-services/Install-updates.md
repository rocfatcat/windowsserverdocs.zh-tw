# <a name="install-updates"></a>安裝更新
我們建議您安裝更新，如果有的話。 安裝更新需要網際網路連線。  

1.  從 [開始]  畫面，開啟 [控制台] 。  
  
2.  在 [控制台]，鍵入**更新**，然後按一下**檢查是否有更新**。  
  
3.  如果 Windows Update 網站列出您的電腦所需的任何更新，安裝更新。
