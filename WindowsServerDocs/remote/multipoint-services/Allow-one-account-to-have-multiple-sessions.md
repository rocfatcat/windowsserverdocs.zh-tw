# <a name="allow-one-account-to-have-multiple-sessions"></a>允許一個帳戶有多個工作階段
若要啟用一群使用者使用多個站台上的共用的帳戶在此同時，MultiPoint 伺服器設定為允許一個帳戶來登入多個站台同時。 根據預設，如果使用者登入時使用共用的使用者帳戶，第二個站台的使用者帳戶被登出的第一個站台。  
  
1.  從**開始**畫面上，開啟**MultiPoint 管理員**。  
  
2.  按一下 [**首頁**] 索引標籤。  
  
3.  在 **電腦**資料行中，按一下 MultiPoint Server 電腦的名稱，然後在右窗格中，按一下**編輯伺服器設定**。  
  
4.  選取 **允許一個帳戶有多個工作階段**核取方塊，然後按一下**確定**。
