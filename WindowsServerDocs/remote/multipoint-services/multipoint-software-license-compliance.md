# <a name="important-information-about-software-license-compliance"></a>軟體使用權規範遵循的相關重要資訊
您必須遵守授權條款，您使用 MultiPoint 服務系統上的程式。 您也必須擁有適當的授權數目的數字*站台*您新增至您的 MultiPoint 服務系統，下列各節中所述。  
  
## <a name="software-license-compliance"></a>軟體的授權合規性  
如果您使用其他 Microsoft 程式或任何協力廠商程式與此 Microsoft 軟體或服務，然後將套用至您使用這些條款隨附這類程式。 Microsoft 可能會認定執行遵守這些條款也是您從 Microsoft 的授權或任何違反任何失敗。  
  
## <a name="client-access-license-cal-compliance"></a>用戶端存取使用權 (CAL) 合規性  
如果您想要將站台新增至您的 MultiPoint 服務系統，您必須保護的適當數目*用戶端存取使用權 (Cal)* 所有站台連接到 MultiPoint 服務。
