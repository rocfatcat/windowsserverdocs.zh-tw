# <a name="protecting-the-system-volume-with-disk-protection"></a>使用磁碟保護保護系統磁碟區
MultiPoint 服務會提供選項，立即清除每次啟動電腦的系統磁碟區的任何變更。 如果您啟用磁碟保護] 功能的 [磁碟機，例如組態損毀或引進的惡意程式碼的任何修改都會復原在下次電腦重新啟動。 這是方便使用的系統管理員想要確保每次載入的已知的 「 良好 」 或 「 黃金 」 軟體映像的功能。 自動更新或修補軟體可排程，比方說，在夜間。 規劃考量就是您要有能夠進行變更，例如從網際網路安裝軟體的使用者。 以啟用此功能，如果您希望使用者能夠儲存檔案，檔案共用必須是系統磁碟區之外。
