# <a name="step-1-plan-an-advanced-single-server-deployment"></a>步驟 1 規劃進階的單一伺服器部署

>適用於：Windows Server （半年通道），Windows Server 2016

規劃使用單次密碼 (OTP) 用戶端驗證部署遠端存取的第一個步驟是規劃和設定進階的單一伺服器部署。  
  
## <a name="plan-a-single-server-deployment"></a>計劃單一伺服器部署  
部署使用 OTP 進行遠端存取之前，請確定您已完成部署單一遠端存取伺服器的所有步驟。 請參閱[部署單一 DirectAccess 伺服器使用進階設定](https://technet.microsoft.com/windows-server-docs/networking/remote-access/directaccess/single-server-advanced/deploy-a-single-directaccess-server-with-advanced-settings)。
