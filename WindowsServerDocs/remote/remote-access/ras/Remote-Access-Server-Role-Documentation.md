# <a name="remote-access-server-role-documentation"></a>遠端存取伺服器角色文件

>適用於：Windows Server （半年通道），Windows Server 2016

遠端存取伺服器角色文件會提供個別或在相同伺服器上部署的任何三個角色服務 （一律開啟 」 VPN、 路由及遠端存取服務，Web 應用程式 Proxy） 的指示。 比方說，這些文件適用於您已在其中部署的三個角色服務，例如將 RRAS 和 DirectAccess 部署在相同伺服器上的任意組合的情況。
