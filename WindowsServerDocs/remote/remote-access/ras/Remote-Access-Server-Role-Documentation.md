# <a name="remote-access-server-role-documentation"></a>遠端存取伺服器角色文件

>適用於：Windows Server (半年通道)、Windows Server 2016

遠端存取服務器角色檔提供指示，分別部署三個角色服務（Always On VPN、路由及遠端存取服務、Web 應用程式 Proxy）的任何一項，或是在相同的伺服器上。 例如，這些檔適用于您已部署三個角色服務組合的情況，例如在同一部伺服器上部署 RRAS 和 DirectAccess。
