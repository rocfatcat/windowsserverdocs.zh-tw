# <a name="plan-deployment-for-remote-management-of-directaccess-clients"></a>規劃部署 DirectAccess 用戶端的遠端管理

>適用於：Windows Server （半年通道），Windows Server 2016

下列主題提供的部署執行的單一遠端存取伺服器的規劃步驟可用來遠端管理 DirectAccess 用戶端。  
  
[步驟 1：規劃遠端存取基礎結構](Step-1-Plan-the-Remote-Access-Infrastructure.md):本主題將協助您規劃網路拓樸和伺服器設定、 防火牆需求、 憑證需求、 網域名稱系統需求、 DirectAccess 網路位置伺服器和 management server 的組態，Active Directory需求，以及群組原則物件的建立。  
  
[步驟 2：規劃遠端存取部署](Step-2-Plan-the-Remote-Access-Deployment.md):規劃用戶端和伺服器部署策略和基礎結構伺服器的組態。
