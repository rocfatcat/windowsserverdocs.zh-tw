本主題介紹部署單一遠端存取服務器所需的設定步驟，以用於 DirectAccess 用戶端的遠端系統管理。  
  
-   [步驟 1：設定遠端存取基礎結構 @ no__t-0：本主題說明如何設定網路和伺服器設定、憑證需求、DNS 設定、網路位置伺服器部署、DirectAccess 管理伺服器、Active Directory 設定，以及群組原則物件。  
  
-   [步驟 2：設定遠端存取服務器 @ no__t-0：本主題說明如何設定 DirectAccess 用戶端電腦、伺服器設定，以及基礎結構和應用程式伺服器。  
  
-   [步驟 3：確認部署 @ no__t-0：本主題說明如何驗證部署。
