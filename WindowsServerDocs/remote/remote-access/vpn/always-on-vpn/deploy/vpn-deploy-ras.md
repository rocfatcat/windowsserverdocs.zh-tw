# 步驟 3. 設定 Always On VPN 的遠端存取伺服器

>適用於： Windows Server （半年度管道）、 Windows Server 2016、 Windows Server 2012 R2、 Windows 10

& #171; [**先前：** 步驟 2。設定伺服器基礎結構](vpn-deploy-server-infrastructure.md)<br>
& #187; [**先前：** 步驟 4。安裝與設定網路原則伺服器 (NPS)](vpn-deploy-nps.md)


RRAS 被設計來執行也為路由器與遠端存取伺服器，因為它支援各種功能。 基於此部署的目的，您需要這些功能一小部分： IKEv2 VPN 連線和 LAN 路由的支援。

IKEv2 是 VPN 通道通訊協定的註解 7296 網際網路工程的工作力要求中所述。 IKEv2 的主要優點是它可容許基礎的網路連線中斷。 例如，如果連線暫時中斷或使用者的用戶端電腦從一個網路移到另一個，IKEv2 自動還原 VPN 連線時重新建立網路連線 — 完全不需要使用者介入。

設定 RRAS 伺服器來支援 IKEv2 連線時停用未使用的通訊協定，這也可以減少伺服器的安全性使用量。 此外，設定伺服器以指派給 VPN 用戶端的地址，從靜態位址集區。 您可以從集區或 DHCP 伺服器; 遇到了將指派位址不過，使用 DHCP 伺服器增加設計的複雜性，而且會提供最少好處。


>[!Important]
>請務必：
>- 在實體伺服器上安裝兩個乙太網路介面卡。 如果您在 VM 上安裝的 VPN 伺服器，您必須建立兩個外部虛擬交換器，一個用於每個實體網路介面卡;然後再建立兩個虛擬網路介面卡的 VM，與每個網路介面卡連接至一個虛擬交換器。
>
>- 安裝您的邊緣與內部防火牆，之間周邊網路上含有一個網路介面卡連接至外部適用於周邊網路，以及一個網路介面卡連接至內部周邊網路伺服器。


>[!Warning]
>在您開始之前，請確定在 VPN 伺服器上啟用 IPv6。 否則，無法建立的連線，並顯示錯誤訊息。

1.  在 VPN 伺服器上，在伺服器管理員中，按一下**通知**旗標。

2.  在 [**工作**] 功能表中，按一下 [**開啟啟動精靈 」**。<p>設定遠端存取精靈會開啟。

3.  按一下**部署只 VPN**。<p>路由及遠端存取 Microsoft Management Console (MMC) 會開啟。

4.  VPN 伺服器，以滑鼠右鍵按一下，然後按一下 [**設定和啟用路由及遠端存取**。<p>路由及遠端存取伺服器安裝精靈會開啟。

5.  在 [歡迎使用路由及遠端存取伺服器安裝精靈，按一下 [**下一步**。

6.  在**設定**中，按一下 [**自訂組態**中，，然後按一下 [**下一步**。

7.  在 [**自訂組態**中，按一下**VPN 存取**]，然後按一下 [**下一步**。<p>完成 [路由及遠端存取伺服器安裝精靈便會開啟。

8.  按一下 [**完成 \]** 以關閉精靈，並按一下 **[確定]** 以關閉的路由及遠端存取 」 對話框。

9.  按一下 [開始遠端存取的**啟動服務**。

10. 在遠端存取 MMC，以滑鼠右鍵按一下 VPN 伺服器，然後按一下**屬性**。

11. 在內容中，按一下 [**安全性**] 索引標籤並執行：

    c. 按一下 **\[新增\]**。<p>新增 RADIUS 伺服器對話方塊隨即開啟。
    
    d. **伺服器名稱**時，輸入您的組織/公司網路上的完全完整網域名稱 (FQDN) NPS 伺服器。<p>例如，如果您的 NPS 伺服器的 NetBIOS 名稱是 NPS1，且您的網域名稱是 corp.contoso.com，輸入**NPS1.corp.contoso.com**。
    
    e. 在**共用密碼**，按一下 [**變更**。<p>變更密碼對話方塊隨即開啟。
    
    f. 在**新的密碼**，輸入文字字串。
