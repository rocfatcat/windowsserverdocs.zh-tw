-   [步驟 1：設定基本 DirectAccess 基礎結構 @ no__t-0。 這個步驟包括設定網路和伺服器設定、DNS 設定及 Active Directory 設定。  
  
-   [步驟 2：設定基本 DirectAccess 伺服器 @ no__t-0。 此步驟包括設定 DirectAccess 用戶端電腦與伺服器設定。
