# <a name="plan-a-basic-directaccess-deployment"></a>規劃基本 DirectAccess 部署

>適用於：Windows Server (半年通道)、Windows Server 2016

本主題說明使用基本功能部署執行 Windows Server 2016、Windows Server 2012 R2 或 Windows Server 2012 的單一 DirectAccess 伺服器所需的規劃步驟：  
  
1.  [步驟1：規劃基本 DirectAccess 基礎結構](da-basic-plan-s1-infrastructure.md)-規劃網路和伺服器拓撲、防火牆設定、憑證需求、DNS 和 Active Directory。  
  
2.  [步驟2：規劃基本 DirectAccess 部署](da-basic-plan-s2-deployment.md)-規劃用戶端和伺服器部署。  
  
## <a name="next-step"></a>後續步驟  
在您完成這些規劃步驟之後，即可開始進行伺服器部署。 如需指示，請參閱[安裝和設定基本 DirectAccess](Install-and-Configure-Basic-DirectAccess.md)。
