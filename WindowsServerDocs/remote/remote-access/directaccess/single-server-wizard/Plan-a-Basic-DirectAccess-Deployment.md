# <a name="plan-a-basic-directaccess-deployment"></a>規劃基本 DirectAccess 部署

>適用於：Windows Server （半年通道），Windows Server 2016

本主題說明部署單一 DirectAccess 伺服器執行 Windows Server 2016、 Windows Server 2012 R2 或 Windows Server 2012 的基本功能所需的規劃步驟：  
  
1.  [步驟 1：規劃基本 DirectAccess 基礎結構](da-basic-plan-s1-infrastructure.md)-規劃網路和伺服器拓撲、 防火牆設定、 憑證需求、 DNS 及 Active Directory。  
  
2.  [步驟 2：規劃基本 DirectAccess 部署](da-basic-plan-s2-deployment.md)-規劃用戶端和伺服器部署。  
  
## <a name="next-step"></a>後續步驟
