# <a name="directaccess-cluster-nlb-test-lab-configuration-requirements"></a>DirectAccess Cluster-NLB 測試實驗室案例設定需求

>適用於：Windows Server （半年通道），Windows Server 2016

下列元件所需的測試實驗室中設定 DirectAccess:  
  
-   產品光碟或 Windows Server 2016、 Windows Server 2012 R2 或 Windows Server 2012 的檔案。  
  
-   六個的電腦或符合 Windows Server 2016、 Windows Server 2012 R2 或 Windows Server 2012; 的最低硬體需求的虛擬機器這些電腦的兩個已安裝的兩個網路介面卡。  
  
-   產品光碟或檔案，適用於 Windows 10&reg;或 Windows&reg; 8。  
  
-   兩個電腦或虛擬機器符合最低硬體需求適用於 Windows 10&reg;或 Windows&reg; 8; 一部電腦已安裝的兩個網路介面卡。
