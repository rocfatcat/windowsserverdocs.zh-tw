# <a name="step-8-snapshot-the-directaccess-cluster-nlb-configuration"></a>步驟 8 建立 DirectAccess 叢集 NLB 設定的快照

>適用於：Windows Server （半年通道），Windows Server 2016

如此即完成 DirectAccess 測試實驗室。 若要儲存這個設定，讓您能夠快速返回使用 DirectAccess，您可以從此處測試其他 DirectAccess 模組測試實驗室指南的 NLB 叢集設定，測試實驗室指南的延伸模組，或您自己的實驗和學習，請執行下列：  
  
1.  在測試實驗室的所有實體電腦或虛擬機器上，關閉所有的視窗，然後執行正常關機。  
  
2.  如果您的實驗室以虛擬機器為基礎，儲存每部虛擬機器的快照和快照集的 DirectAccess 叢集和 NLB 命名。 如果您的實驗室使用實體電腦，請建立磁碟映像，以儲存 DirectAccess 測試實驗室設定。
