# <a name="step-3-install-and-configure-edge2"></a>步驟 3 的安裝和設定 EDGE2

>適用於：Windows Server （半年通道），Windows Server 2016

EDGE2 是遠端存取叢集的第二個成員。 EDGE2 安裝和設定後，再啟用叢集組態。

執行下列步驟來設定 EDGE2:

## <a name="installOS"></a>EDGE2 上安裝作業系統  
  
1.  在上 EDGE2，啟動 Windows Server 2016、 Windows Server 2012 R2 或 Windows Server 2012 的安裝。  
  
2.  請依照下列指示完成安裝，指定 Windows Server 2016、 Windows Server 2012 R2 或 Windows Server 2012 （完整安裝） 和本機系統管理員帳戶的強式密碼。 使用本機 Administrator 帳戶登入。  
  
3.  連線 EDGE2 至具有網際網路存取的網路，並執行 Windows Update，以針對 Windows Server 2016、 Windows Server 2012 R2 或 Windows Server 2012，安裝最新的更新，並再中斷網際網路連線。  
  
4.  連線到公司網路子網路或代表公司網路子網路和其他網際網路子網路或代表網際網路子網路的虛擬交換器的虛擬交換器的一張網路介面卡。  
  
## <a name="TCP"></a>設定 TCP/IP 內容  
  
1.  在 伺服器管理員 主控台中，按一下**本機伺服器**，然後在**屬性**區域中下, 一步**有線乙太網路連線**，按一下連結。  
  
2.  在 **網路連線**視窗，以滑鼠右鍵按一下 連線到公司網路子網路或虛擬交換器的網路連線，，然後按一下**重新命名**。  
  
3.  型別**Corpnet**，然後按 ENTER 鍵。  
  
4.  以滑鼠右鍵按一下**Corpnet**，然後按一下**屬性**。  
  
5.  按一下 [網際網路通訊協定第 4 版 (TCP/IPv4)]  ，然後按一下 [內容]  。  
  
6.  按一下 [使用下列的 IP 位址]  。 在  **IP 位址**，型別**10.0.0.8**。 在 [子網路遮罩]  中，輸入 **255.255.255.0**。  
  
7.  按一下 [使用下列的 DNS 伺服器位址]  。 在 [慣用 DNS 伺服器]  中，輸入 **10.0.0.1**。  
  
8.  按一下 [進階]  ，然後按一下 [DNS]  索引標籤。  
  
9. 在 **此連線的 DNS 尾碼**，型別**corp.contoso.com**，按一下  **確定**兩次。  
  
10. 選取 **[網際網路通訊協定第 6 版 (TCP/IPv6)]** ，然後按一下 **[屬性]** 。  
  
11. 按一下 **使用下列 IPv6 位址**。 在  **IPv6 位址**，型別**2001:db8:1::8**。 在 **子網路首碼長度**，型別**64**。  
  
12. 按一下 [使用下列的 DNS 伺服器位址]  。 在 **慣用 DNS 伺服器**，型別**2001:db8:1::1**。  
  
13. 按一下 [進階]  ，然後按一下 [DNS]  索引標籤。  
  
14. 在 **此連線的 DNS 尾碼**，型別**corp.contoso.com**，按一下**確定**兩次，然後按一下 **關閉**。  
  
15. 在 **網路連線** 視窗中，以滑鼠右鍵按一下 連線到網際網路子網路的網路連線，然後按一下 **重新命名**。  
  
16. 型別**網際網路**，然後按 ENTER 鍵。  
  
17. 在 [網際網路]  按滑鼠右鍵，然後按一下 [內容]  。  
  
18. 按一下 [網際網路通訊協定第 4 版 (TCP/IPv4)]  ，然後按一下 [內容]  。  
  
19. 按一下 [使用下列的 IP 位址]  。 在  **IP 位址**，輸入**131.107.0.8**。 在 **子網路遮罩**，輸入**255.255.255.0**。  
  
20. 按一下 [ **DNS** ] 索引標籤  
  
21. 在**此連線的 DNS 尾碼**，型別**isp.example.com**，然後按一下 **[確定]** 兩次，然後按一下**關閉**。  
  
22. 關閉 [網路連線]  視窗。  
  
23. 若要檢查 EDGE2 與 DC1 之間的網路通訊，請按一下**開始**，型別**cmd**，然後按 ENTER 鍵。  
  
24. 在 [命令提示字元] 視窗中，輸入**ping dc1.corp.contoso.com**按 ENTER 鍵。 請確認有四個來自 10.0.0.1 的回應，或 IPv6 位址 2001:db8:1::1  
  
25. 關閉 [命令提示字元] 視窗。  
  
## <a name="rename"></a>重新命名 EDGE2，並將它加入網域  
  
1.  在 伺服器管理員 主控台中，在**本機伺服器**，請在**屬性**區域中下, 一步**電腦名稱**，按一下連結。  
  
2.  在 [系統內容]  的 [電腦名稱]  索引標籤上，按一下 [變更]  。  
  
3.  在 **電腦名稱/網域變更**對話方塊中，於**電腦名稱**方塊中，輸入**EDGE2**。 中**的成員**區域中，按一下**網域**，然後在文字方塊中，輸入**corp.contoso.com**，然後按一下 **[確定]** 。  
  
4.  當系統提示您輸入使用者名稱和密碼時，輸入 **User1** 和其密碼，然後按一下 [確定]  。  
  
5.  出現歡迎您進入 corp.contoso.com 網域的對話方塊時，按一下 [確定]  。  
  
6.  當提示您必須重新啟動電腦時，按一下 [確定]  。  
  
7.  按一下 [系統內容]  對話方塊中的 [關閉]  。  
  
8.  當提示您重新啟動電腦時，請按一下 [立即重新啟動]  。  
  
9. 重新啟動之後，以 corp\user1 的身分登入。  
  
## <a name="IPHTTPSCert"></a>安裝 IP-HTTPS 憑證  
  
1.  在 **開始**畫面上，輸入**mmc.exe**，然後按 ENTER 鍵。 如果出現 [**使用者帳戶控制**] 對話方塊，請確認它所顯示的動作就是您所需的動作，然後按一下 [**是**]。  
  
2.  在 MMC 主控台中，按一下 [檔案]  功能表上的 [新增/移除嵌入式管理單元]  。  
  
3.  在上**新增或移除嵌入式管理單元** 對話方塊中，按一下**憑證**，按一下**新增**，按一下 **電腦帳戶**，按一下  **下一步**，按一下**完成**，然後按一下**確定**。  
  
4.  在主控台的左窗格中，瀏覽至**Certificates (Local Computer) \Personal\Certificates**。 以滑鼠右鍵按一下**憑證**節點，指向**所有工作**，然後按一下**要求新憑證**。  
  
5.  在 [憑證註冊精靈] 中，按一下 [**下一步]** 兩次。  
  
6.  在 **要求憑證**頁面上，選取**Web 伺服器**核取方塊，然後按一下**的詳細資訊，才能註冊此憑證**。  
  
7.  在上**憑證內容**對話方塊的 **主旨**索引標籤**主體名稱**區域中**類型**清單中，按一下**一般名稱**。  
  
8.  在 **值**，型別**edge1.contoso.com**，然後按一下**新增**。  
  
9. 在 **別名**區域，請在**型別**清單中，按一下**DNS**。  
  
10. 在 **值**，型別**edge1.contoso.com**，然後按一下**新增**。  
  
11. 在上**一般**索引標籤中，於**易記名稱**，型別**IP-HTTPS 憑證**。  
  
12. 依序按一下 [確定]  、[註冊]  ，然後按一下 [完成]  。  
  
13. 在 [憑證] 嵌入式管理單元的詳細資料窗格中，確認新的憑證名稱 edge1.contoso.com 與已註冊使用目的伺服器 」 驗證。  
  
14. 關閉主控台視窗。 如果提示您儲存設定時，請按一下**No**。  
  
## <a name="InstallDA"></a>EDGE2 上安裝遠端存取角色  
  
1.  在 [伺服器管理員] 主控台中，在**儀表板**，按一下**新增角色及功能**。  
  
2.  按 [下一步]  3 次，進入伺服器角色選擇畫面。  
  
3.  在 [選取伺服器角色]  對話方塊上，選取 [遠端存取]  ，按一下 [新增功能]  ，然後按 [下一步]  。  
  
4.  按 [下一步]  五次。  
  
5.  在 [確認安裝選項]  對話方塊中，按一下 [安裝]  。  
  
6.  在 [安裝進度]  對話方塊中，確認安裝成功，然後按一下 [關閉]  。
