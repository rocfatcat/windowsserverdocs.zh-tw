# <a name="configuration-requirements"></a>設定需求

>適用於：Windows Server （半年通道），Windows Server 2016

若要設定遠端存取測試實驗室中需要下列元件：  
  
-   產品光碟或 Windows Server 2016、 Windows Server 2012 R2 或 Windows Server 2012 的檔案。  
  
-   九個電腦或虛擬機器符合最低硬體需求 Windows Server 2016、 Windows Server 2012 R2 或 Windows Server 2012;這些電腦的三個已安裝的兩個網路介面卡。  
  
-   產品光碟或檔案，適用於 Windows 10&reg;或 Windows&reg; 8。  
  
-   產品光碟或檔案的 Windows 7 旗艦版。  
  
-   三個電腦或虛擬機器符合最低硬體需求適用於 Windows 10&reg;，Windows 8 或 Windows 7; 一部電腦已安裝的兩個網路介面卡。
