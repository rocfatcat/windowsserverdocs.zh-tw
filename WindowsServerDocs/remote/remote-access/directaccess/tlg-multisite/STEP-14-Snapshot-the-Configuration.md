# <a name="step-14-snapshot-the-configuration"></a>步驟 14 建立設定的快照

>適用於：Windows Server （半年通道），Windows Server 2016

如此即完成 DirectAccess 多站台的測試實驗室。 若要儲存這個設定，讓您能夠快速返回工作 DirectAccess 多站台組態，您可以從此處測試其他模組化的 DirectAccess 測試實驗室指南、 測試實驗室指南的延伸模組，或您自己的實驗和學習，請執行下列:  
  
1.  在測試實驗室的所有實體電腦或虛擬機器上，關閉所有的視窗，然後執行正常關機。  
  
2.  如果您的實驗室以虛擬機器為基礎，儲存每個虛擬機器快照，然後將快照命名為 TLG DirectAccess 多站台。 如果您的實驗室使用實體電腦，請建立磁碟映像，以儲存 DirectAccess 測試實驗室設定。
