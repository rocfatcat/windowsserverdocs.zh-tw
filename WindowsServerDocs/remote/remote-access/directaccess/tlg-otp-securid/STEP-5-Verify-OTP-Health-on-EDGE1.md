# <a name="step-5-verify-otp-health-on-edge1"></a>步驟5確認 EDGE1 上的 OTP 健全狀況

>適用於：Windows Server (半年通道)、Windows Server 2016

下列程式會確認 OTP 已設定且正常運作，使用 EDGE1 上的 DirectAccess 伺服器健全狀況監視。  
  
### <a name="verify-otp-health-on-edge1-using-directaccess-server-health-monitoring"></a>使用 DirectAccess 伺服器健全狀況監視，確認 EDGE1 上的 OTP 健全狀況  
  
1.  在 EDGE1 上，開啟 [**遠端存取管理**] 主控台。  
  
2.  按一下 [**作業狀態**]。  
  
3.  確認 OTP 的狀態為 [**作用**中]。
