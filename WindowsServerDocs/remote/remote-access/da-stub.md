# <a name="directaccess"></a>DirectAccess

>適用於：Windows Server (半年度管道)、Windows Server 2016

在 Windows Server 2016 中，**DirectAccess 和 VPN** 是**遠端存取**伺服器角色的角色服務。

DirectAccess 可讓遠端使用者連線到組織網路資源，而不需要傳統虛擬私人網路（VPN）連線。 

DirectAccess 文件現在位於 Windows Server 2016 目錄的[遠端存取和伺服器管理](../index.yml)區段中，[遠端存取](./remote-access.md)下方。 如需詳細資訊，請參閱 [DirectAccess](directaccess/DirectAccess.md)。
