# <a name="recommended-settings-for-vdi-desktops"></a>建議的 VDI 桌面設定

>適用於：Windows Server （半年通道），Windows Server 2016 中，windows 10

Microsoft 桌面虛擬化會自動偵測裝置組態和網路狀況，讓使用者啟動並執行更快讓 立即安裝公司應用程式和桌面，以及它所提供以提供存取至舊版的 IT在移轉至 Windows 10 的應用程式。

Windows 10 作業系統會妥善調整現成的雖然有時候會為您要先精簡進一步專為公司的 Microsoft 虛擬桌面基礎結構 (VDI) 環境。 在 VDI 環境中，許多背景服務和工作已停用從一開始。

本主題不是藍圖，但而指南或起始點。 某些建議可能會停用您想要使用，因此您應該考慮調整任何特定的設定，在您的案例的優點與成本的功能。

這些指示和建議的設定是與 Windows 10 1607 （版本 10.0.1393）。

> [!NOTE]  
> 可以保留任何未特別說明本主題中的設定不會明顯影響 VDI 功能及其預設值 （或每個您的需求和原則集合）。

當您建立基底 VDI 部署映像時，務必使用**最新分支**。 如需有關最新分支的詳細資訊，請參閱 < [Windows 10 版本資訊](https://technet.microsoft.com/windows/release-info.aspx)。

## <a name="in-windows-settings"></a>在 Windows 設定
若要存取 Windows 設定，請按一下**開始**（在工作列上的 Windows 圖示），然後按一下**設定**（形狀像齒輪） 圖示。

### <a name="in-the-system-area-of-windows-settings"></a>在 [系統] 區域中的 Windows 設定
在 [Windows 設定] 區域中，按一下**系統**圖示可讓您存取各種系統相關的設定。 並非所有需要調整的最佳 VDI 使用--這些設定是最重要：

#### <a name="apps-and-features"></a>應用程式和功能

若要移除的應用程式，藉此將它從您的 VDI 映像，排除 應用程式，然後**解除安裝**。 如果**解除安裝**會變為灰色，您無法將其移除由這個方法，您可以使用 Windows PowerShell，將它移除，或請嘗試下列步驟：
1. 按一下 **管理選擇性功能**(正下方**應用程式和功能**標題相同頁面上)。
2. 按一下 [選擇性] 功能，然後再按一下**解除安裝**。

請考慮移除 （如果有的話） 的功能包括下列各項：
- **請連絡支援服務**
- **英文 （美國） 零售示範內容**
- **中性零售示範內容**
- **快速協助**

#### <a name="default-apps"></a>預設 App

這個區域會定義用於預設特定泛型的函式，例如電子郵件、 網頁瀏覽，以及對應的應用程式。 如果您想要用於特定函式不同的應用程式時，按一下目前的項目，然後按一下 應用程式，您想要用於 VDI 的映像。 非 Microsoft 應用程式中是可用的選擇，您必須安裝應用程式之前調整此設定。

### <a name="performance-monitor"></a>效能監視器

## <a name="microsoft-store-apps"></a>Microsoft Store 應用程式
有一些您可能想要移除 VDI 映像; 中的 Microsoft Store 應用程式移除它們，會降低 CPU 使用量，並節省磁碟空間。 移除的良好候選項目包括：

- 取得 Office
- Skype （預覽）
- 開始 （尤其是如果沒有網際網路連線）
- 意見反應中樞
- Microsoft [接龍] 的集合
- 付費的 Wi-fi 和行動數據

若要自訂預設使用者設定檔，用來建立 VDI 映像，使用內建的 Administrator 帳戶。 如果未啟用，請使用 電腦管理 中的 本機使用者和群組。 然後登入系統管理員帳戶，以完成下列步驟。

> [!NOTE]  
> 請勿移除系統應用程式，例如市集應用程式。 它們很難進行重新安裝。 其他應用程式必須容易 reinstallable 從存放區。

### <a name="delete-unwanted-apps-from-the-administrator-user-profile"></a>從系統管理員使用者設定檔刪除不必要的應用程式
1. 在 Windows PowerShell 中執行`Get-AppxPackage | ft PackageFamilyName`若要查看已安裝的應用程式清單。
2. 針對您想要解除安裝每個應用程式封裝工具執行 cmdlet，此範例格式：

    `Get-AppxPackage *messaging* | Remove-AppxPackage`

    `Get-AppxPackage *WindowsMaps* | Remove-AppxPackage`

    `Get-AppxPackage *ZuneMusic* | Remove-AppxPackage`

### <a name="delete-the-payload-of-unwanted-store-apps"></a>刪除不必要的市集應用程式的承載
這會避免重新安裝應用程式。
1. 清單的市集應用程式和已佈建搭配這個指令程式的儲存體中資料的其他項目： `Get-AppxProvisionedPackage -Online`。
2. 移除指定的套件與`Remove-AppxProvisionedPackage -Online -PackageName MyAppPackage`，使用適當的 MyAppPackage 步驟 1 中傳回。 例如，若要移除的 Zune 相關的封裝，您會執行`Remove-AppxProvisionedPackage -Online -PackageName Microsoft.ZuneMusic_2019.17012.10311.0_neutral_~_8wekyb3d8bbwe`。

## <a name="removing-other-items"></a>移除其他項目
您可以移除 OneDrive 圖示和應用程式、 關閉 系統圖示和刪除下載的更新。

### <a name="remove-onedrive-icon-and-app"></a>移除 [OneDrive] 圖示和應用程式
1. 按一下 **開始**，然後捲動至**OneDrive**圖示。
2. 以滑鼠右鍵按一下**OneDrive**圖示，指向**詳細**，然後按一下**開啟檔案位置**。
3. 以滑鼠右鍵按一下**OneDrive**圖示，在其檔案位置，然後按一下**刪除**。

若要移除的 OneDrive 應用程式：
1. 按一下 **開始**，然後捲動至**OneDrive**圖示。
2. 以滑鼠右鍵按一下**OneDrive**圖示，，然後按一下**解除安裝**。 程式和功能會開啟。
3. 在 程式和功能，以滑鼠右鍵按一下**Microsoft OneDrive**然後按一下**解除安裝**。

### <a name="programs-and-features-from-previous-versions-of-control-panel"></a>程式和功能 （從控制台中的舊版）
1. 推播**開始**按鈕，輸入*控制*，然後按 ENTER 鍵。
2. 點選或按兩下**程式和功能**。
3. 最左邊底下**主控制台**，請點選或按一下**開啟或關閉開啟的 Windows 功能**。 會開啟新的使用者介面。
4. 清除核取方塊的任何項目，您不想要或不需要在基底映像，例如：**SMB 1.0/CIFS 檔案共用支援**。

### <a name="turn-system-icons-off"></a>關閉 系統圖示
1. 推播，或按一下**開始**，然後按一下**設定**（齒輪圖示）。
2. 在 **尋找設定**文字區域中，輸入*工作列*，然後按一下**工作列設定**。
3. 底下**工作列**區段中，捲動或向下撥動**通知區域**一節。
4. 按一下或點選**開啟或關閉系統圖示**，然後開啟每個系統 圖示開啟或關閉以您想映像。

### <a name="delete-downloaded-updates"></a>刪除下載的更新
1. 使用檔案總管 中，瀏覽至**C:\Windows\Software Distribution\Download**。
2. 刪除所有檔案和資料夾，在該目錄中。
