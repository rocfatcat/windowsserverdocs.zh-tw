# <a name="additional-remote-desktop-resources"></a>其他遠端桌面資源

除了 Windows Server 2016 文件庫中的資訊以外，您也可以使用下列資源來了解遠端桌面服務及取得相關協助：

- 使用 Twitter 嗎？ 我們也是：[@RDS4U](https://twitter.com/RDS4U)
- 在 [RDS TechNet 論壇](https://aka.ms/technetforum-rds)加入關於遠端桌面服務的一般討論。
- 如需適用於 Windows、Android、iOS 和 Mac 的遠端桌面應用程式/用戶端的相關討論，請瀏覽[遠端桌面用戶端 TechNet 論壇](https://aka.ms/technetforum-rdc)。
- 如需 MultiPoint 的相關資訊，請參閱 [MultiPoint TechNet 論壇](https://aka.ms/multipoint-forum)。

如果您有遠端桌面服務方面的想法要與我們分享，請在[我們的 UserVoice 論壇](https://aka.ms/uservoice-rds)發表相關主題。
