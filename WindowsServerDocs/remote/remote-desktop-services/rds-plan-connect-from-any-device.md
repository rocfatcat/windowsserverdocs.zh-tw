# <a name="remote-desktop-services---connect-from-any-device"></a>遠端桌面服務 - 從任何裝置連線

>適用於：Windows Server (半年通道)、Windows Server 2019、Windows Server 2016

從任何 Windows、Apple 或 Android 電腦、平板電腦或手機存取公司資源。 讓使用者能夠輕鬆地從任何裝置透過 RD Web 摘要查看其可用的桌面和應用程式。

深入了解 [Microsoft 遠端桌面用戶端](clients/remote-desktop-clients.md)。
