# 什麼是新的遠端桌面用戶端上 macOS？

我們會定期更新[的 macOS 遠端桌面用戶端](remote-desktop-mac.md)，新增新的功能，並修正問題。 請查看最新的更新。

如果您遇到任何問題，您一律可以連絡我們透過**協助 > 報告問題**。

## 更新版本，10.2.9
*發行日期： 3/6/2019年*

- 我們已修正 RD 閘道連線問題發生時伺服器重新導向採用此版本中放置。
- 我們也會解決因 10.2.8 RD 閘道迴歸更新。

## 更新版本，10.2.8
*發行日期： 3/1/2019年*

- 解析提出使用 RD 閘道時的連線問題。
- 連線時所顯示的固定不正確的憑證警告。
- 解決某些情況下，其中的功能表列和擴充座就會無端地隱藏時啟動遠端 app。
- 重新作業位址當機和有已侵犯某些使用者的停止回應的剪貼簿重新導向程式碼。
- 修正造成連線中心會無端地捲動啟動連線時的錯誤。

## 更新版本，10.2.7
*發行日期： 2/6/2019年*

- 在此版本中，我們解決圖形 mispaints （伺服器編碼的錯誤所造成） 出現時使用 AVC444 模式。

## 更新版本，10.2.6
*發行日期： 1/28/2019年*

- 新增的 AVC （420 和 444） 轉碼器，連線到目前版本的 Windows 10 時，可用以支援。
- 在調整成視窗模式，視窗重新整理現在會發生之後立即調整大小，以確保該內容會呈現在正確的內插補點層級。
- 修正造成某些使用者重疊的摘要標頭的版面配置錯誤。
- 清除應用程式的喜好設定 UI。
- 優雅編輯新增桌面 UI。
- 做了許多的調整與完成連線中心磚和清單檢視，針對桌上型電腦和同步發佈摘要，來調整。

>[!NOTE]
>沒有錯誤 macOS 10.14.0 和可能會造成 「.com.microsoft.rdc.application-data_SUPPORT/_EXTERNAL_DATA 」 的 10.14.1 資料夾 （~/Library 資料夾內的深層巢狀結構） 中取用大量磁碟空間。 若要解決此問題，刪除資料夾的內容，升級至 macOS 10.14.2。 請注意刪除資料夾內容的副作用是指派給書籤快照映像將會刪除。 重新連線到遠端電腦時，將產生這些映像。

## 更新版本，10.2.4
*發行日期： 2018/12/18*

- MacOS Mojave 10.14 夜間模式中新增的支援。
- 如果它是空的來從 Microsoft 遠端桌面 8 現在匯入的選項會出現在連線中心。
- 解決與某些協力廠商企業應用程式的資料夾重新導向相容性。
- 從何處使用者已取得 0x30000069 遠端桌面閘道錯誤，因為安全性通訊協定後援問題解決的問題。
- 某些使用者已經遇到使用固定的漸進式轉譯問題容納視窗模式。
- 修正錯誤，導致檔案複製和貼上從檔案的最新版本複製。
- 已改善滑鼠為基礎的小型捲動差異捲動。

## 更新版本，10.2.3
*發行日期： 2018/11/06*

- 遠端 app 案例的 「 remoteapplicationcmdline 」 RDP 檔案設定已新增的支援。
- 標題的工作階段視窗現在包含 RDP 檔案 （和伺服器名稱） 的名稱從 RDP 檔案啟動時。
- 修正回報的 RD 閘道效能問題。
- 當機報告固定的 RD 閘道。
- 固定位置連線會停止回應的 RD 閘道透過連線問題。
- 全螢幕的遠端應用程式的更清楚地處理明智地隱藏的功能表列和擴充座。
- 固定的案例，其中遠端應用程式仍留隱藏啟動之後。
- 停用硬體加速上使用 「 調整視窗大小 」 時，會解決較慢的轉譯更新。
- 處理不正確的權限所造成的用戶端啟動時的資料庫建立錯誤。 
- 修正問題之前，用戶端所在以一致的方式在啟動損毀，並針對某些使用者無法啟動。
- 固定位置連線已正確匯入做為全螢幕從遠端桌面 8 的案例。

## 更新版本，10.2.2
*發行日期： 10/09/2018年*

- 全新的連線中心支援拖放，桌上型電腦、 可調整大小的資料行清單檢視模式、 欄排序，及更簡單群組管理中手動排列方式。
- 連線中心現在會記住最後一個使用中的樞紐 （桌上型電腦或同步發佈摘要） 時關閉應用程式。
- 檢修認證提示 UI 與流程。
- RD 閘道意見反應現已連接的狀態 UI 的一部分。
- 已改進版本 8 用戶端從設定匯入。
- 指向 RemoteApp 端點 RDP 檔案現在可以匯入連線中心。
- 針對單一監視器遠端桌面案例視網膜顯示最佳化。
- 用於指定的圖形內插補點層級 （這會影響 blurriness） 支援不使用視網膜最佳化時。
- 若要啟用連線至 Windows 2000 256 色的支援。
- 固定裁剪的右邊和底部邊緣的畫面時，連線到 Windows 7、 Windows Server 2008 R2 和較舊版本。
- 現在將本機檔案複製到 Outlook （遠端工作階段中執行） 作為附件加入的檔案。
- 修正的問題。 如果檔案是從網路共用，已慢繪圖板為基礎的檔案傳輸。
- 解決錯誤會造成停止回應時重新導向的資料夾上的檔案儲存至 Excel （遠端工作階段中執行）。
- 修正問題，造成沒有可用的空間來報告為重新導向的資料夾。
- 修正造成取用太多磁碟上的儲存空間 macOS 10.14 縮圖的錯誤。
- 強制使用 RD 閘道裝置重新導向原則已新增的支援。
- 固定的工作階段 windows 防止關閉時中斷連線使用 RD 閘道的問題。
- 如果伺服器不會強制執行網路層級驗證 (NLA)，您將會現在會路由傳送到登入畫面如果您的密碼已過期。
- 解決效能問題，發現時已透過網路傳輸大量資料。
- 智慧卡重新導向修正程式。
- 如果要支援的 「 EnableCredSspSupport 」 和 「 驗證層級 」 的所有可能的值為 RDP 檔案設定 ClientSettings.EnforceCredSSPSupport 使用者預設金鑰 （com.microsoft.rdc.macos 在網域中） 設定為 0。
- 當未交涉 NLA 「 提示的認證上用戶端 」 RDP 檔案設定的支援。
- 智慧卡為基礎的登入透過 Winlogon 命令提示字元中時未交涉 NLA 智慧卡重新導向的支援。
- 修正問題，導致無法下載摘要 URL 中有空格的資源。

## 更新版本，10.2.1
*發行日期： 08/06/2018年*

- 已啟用的連線至 Azure Active Directory (AAD) 加入的電腦。 若要連線到已加入的 AAD 的電腦，您的使用者名稱必須是下列格式的其中一種: 「 AzureAD\user 」 或 「 AzureAD\user@domain 」。
- 解決影響遠端工作階段中的智慧卡的使用方式的一些問題。

## 更新版本，10.2.0
*發行日期： 2018/07/24*

- 合併的 GDPR 合規性更新。
- MicrosoftAccount\username@domain 現在接受為有效的使用者名稱。
- 共用剪貼簿經過重寫速度較快和支援更多的格式。
- 複製和貼上文字、 影像或檔案現在工作階段之間會略過本機電腦的剪貼簿。
- （如果您接受警告提示），您現在可以連線透過 RD 閘道伺服器使用未受信任的憑證。
- 裸機硬體加速現在用 （若有支援） 以加快轉譯與最佳化的電池使用情況。
- 當使用我們嘗試運作，讓某些魔術裸機硬體加速工作階段圖形會顯示清晰。
- 了解 windows 會停止回應後關閉某些情況下，移除。
- 已防止在某些情況下 RemoteApp 程式的啟動的修正的錯誤。
- 固定 0x204 錯誤所導致的 RD 閘道通道同步處理錯誤。
- 滑鼠游標圖形現在會更新正確移出的工作階段或 RemoteApp 視窗時。
- 固定資料夾重新導向的錯誤會造成資料遺失時複製和貼上的資料夾。
- 修正造成不正確報告的資料夾大小資料夾重新導向問題。
- 固定導致無法登入使用本機帳戶加入 AAD 的電腦迴歸。
- 修正的錯誤所造成的工作階段視窗內容被裁剪。
- 新增含有橢圓曲線非對稱金鑰的 RD 端點憑證以支援。
- 固定導致無法下載的受管理的資源，在某些情況下的錯誤。
- 解決釘選的連線中心裁剪問題。
- 若要搭配使用時，顯示屬性工作表中修正核取方塊。
- 動態顯示變更生效時現在停用鎖定外觀比例。
- 解決 F5 基礎結構的相容性問題。
- 更新的空白密碼，以確保正確的訊息會顯示在連線時處理。
- 固定的滑鼠捲動 MapInfra 專業版的相容性問題。
- Mojave 上執行時，請在連線中心固定對齊的一些問題。

## 更新版本，10.1.8
*發行日期： 05/04/2018年*

- 已新增的工作階段調整視窗大小的變更遠端解析度的支援 ！

## 更新版本，10.1.7
*發行日期： 2018/04/05*

- 進行納入 CredSSP 加密 oracle 補救更新 CVE-2017-2018年-0886年中所述的安全性問題修正。
- 改善的 RemoteApp 圖示和滑鼠游標呈現來解決報告的 mispaints。

- 當您準備全螢幕檢視中時，CMD + M 現在正確地對應至 WIN + M。
- 連線中心、 喜好設定，以及關於 windows 現在回應 CMD + M。
- 您現在可以開始探索同步發佈摘要，藉由按下 ENTER**新增遠端資源**頁面上。
- 其中新的遠端資源摘要顯示空連線中心，直到您重新整理之後修正問題。

- 修正問題 RemoteApp windows 會重新排序本身。

- 我們已錯誤修正的錯誤訊息，您會看到正確的訊息現在如果我們找不到您的閘道。
- 結束捷徑 （⌘ + Q） 現在會以一致的方式顯示在 UI 中。
- 改進時自動縮放 」 調整至視窗大小 」 模式中的影像品質。
- 修正造成在遠端工作階段中顯示的主資料夾的多個執行個體迴歸。
- 更新桌面磚的預設圖示。
