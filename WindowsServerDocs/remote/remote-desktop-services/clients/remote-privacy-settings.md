# 適用於受管理的應用程式和桌面的隱私權設定

>適用於： Windows 10、 Windows 8.1、 Windows Server 2012 R2、 Windows Server 2016

存取受管理的資源時 （應用程式或桌上型電腦） 提供您的 IT 系統管理員，已經預先設定的遠端系統的隱私權設定您的 IT 系統管理員。 這些設定可能會不同於您的本機系統上的隱私權設定。 

>[!NOTE]
>在美國以外地區中使用受管理的資源可能會導致資料傳輸到美國。

以下是一些您可以在您受管理的桌面設定的 Windows 10 隱私權設定：
- [語音辨識](https://go.microsoft.com/fwlink/?linkid=874646)
- [尋找我的裝置](https://go.microsoft.com/fwlink/?linkid=533063)
- [筆跡 & 鍵入](https://go.microsoft.com/fwlink/?linkid=874646)
- [廣告識別碼](https://go.microsoft.com/fwlink/?linkid=838419)
- [位置](https://go.microsoft.com/fwlink/?linkid=529987)
- [診斷資料](https://go.microsoft.com/fwlink/?linkid=614828)
- [量身訂做的體驗](https://go.microsoft.com/fwlink/?linkid=614828)

您一律可以檢閱收集並透過存取您的[隱私權儀表板](https://go.microsoft.com/fwlink/?linkid=864206)傳送給 Microsoft 的資訊。

## 深入了解隱私權設定

如果您的 IT 系統管理員已提供您與受管理的桌面，您可以依照深入了解這些設定，以及變更任何由您的 IT 系統管理員未鎖定的設定在下一節的指示。

### 如何變更隱私權設定，在 Windows 10 的遠端桌面

若要變更在 Windows 10 的遠端桌面的隱私權設定：

1. 從遠端桌面，選取 [Windows] 按鈕，在工作列上，或按 Windows 鍵以開啟 [開始] 功能表鍵盤上。
2. 選取的齒輪圖示以開啟 [設定。
3. 搜尋的可設定的隱私權設定列出稍早在本主題中深入了解它的名稱。

>[!NOTE]
> 如果您的 IT 系統管理員已設定受管理的桌面，到不會保留使用者組態設定之間的連線，不會儲存您進行這些設定的任何變更。
