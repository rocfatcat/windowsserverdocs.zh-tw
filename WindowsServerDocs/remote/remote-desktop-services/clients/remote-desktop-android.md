# 開始使用 Android 上的遠端桌面

>適用於： Windows 10、 Windows 8.1、 Windows Server 2019、 Windows Server 2016、 Windows Server 2012 R2

您可以使用遠端桌面用戶端，以使用 Android 直接從您的 Android 裝置搭配 Windows 應用程式和桌面。

若要開始使用下列資訊。 請務必查看[常見問題集](remote-desktop-client-faq.md)，如果您有任何問題。

> [!NOTE]
> - 好奇 Android 用戶端的新發行版本？ 請查看[的 Android 上的遠端桌面的新功能？](android-whatsnew.md)
> 您可以執行用戶端，以及使用 ChromeOS 53 安裝和 Chromebook Android 4.1 和較新的裝置上。 深入了解 Android 應用程式上 Chrome[以下](https://sites.google.com/a/chromium.org/dev/chromium-os/chrome-os-systems-supporting-android-apps)。

## 取得 RD 用戶端，並開始使用它

請依照下列步驟來開始使用遠端桌面 Android 裝置上：

1. 從[Google Play](https://play.google.com/store/apps/details?id=com.microsoft.rdc.android)中下載遠端桌面用戶端。 
2. [設定您的電腦以接受遠端連線](remote-desktop-allow-access.md)。
3. 新增遠端桌面連線或遠端資源。 您要直接連線到 Windows 電腦和遠端資源使用 RemoteApp 程式、 工作階段為基礎的桌面，使用連線或虛擬桌面發行的內部部署。 
4. 建立 widget，所以您可以快速取得遠端桌面。

> [!NOTE]
> 如果您想要更早版本飛行的新功能建議從 Google Play 市集下載我們的[Microsoft 遠端桌面搶鮮版](https://play.google.com/store/apps/details?id=com.microsoft.rdc.android.beta)應用程式。 

### 新增遠端桌面連線

若要建立遠端桌面連線：

1. 在連線中心點選**+**，然後點選 [**桌面**。
2. 輸入您想要連線到下列資訊的電腦：
  - **電腦名稱**– 電腦的名稱。 這可以是 Windows 的電腦名稱、 網際網路網域名稱或 IP 位址。 您也可以附加到電腦名稱 （例如， **MyDesktop:3389**或**10.0.0.1:3389**） 的連接埠資訊。
  - **使用者名稱**– 要用來存取遠端電腦的使用者名稱。 您可以使用下列格式：*使用者名稱*、 *domain\user_name*或*user_name@domain.com*。 您也可以指定是否要提示您輸入使用者名稱和密碼。
3. 您也可以設定下列選項：
  - **易記名稱**– 容易記住您連線到電腦的名稱。 您可以使用任何字串，但如果您未指定的易記名稱，會顯示電腦名稱。
  - **閘道**– 您想要用來連線到虛擬桌面、 RemoteApp 程式及工作階段式桌面內部的公司網路上的遠端桌面閘道。 從您的系統管理員取得閘道的相關資訊。
    需要設定遠端桌面閘道嗎？
  - **音效**– 選取裝置要在遠端工作階段期間用於音訊。 您可以選擇播放聲音在本機裝置，在遠端裝置上，或完全不。
  - **自訂顯示器解析度**-透過啟用此設定以設定自訂連線的解析度。 當套用解析度關閉您的應用程式的全域設定中定義的。
  - **交換滑鼠按鈕**– 使用此選項，以滑鼠右鍵按鈕交換滑鼠左鍵的按鈕函式。 （這是特別有用，如果遠端電腦已設定為慣用左手的使用者，但您使用慣用右手的滑鼠）。
  - **連線到系統管理員工作階段**-使用此選項以連線到主控台工作階段來管理 Windows server。
  - **重新導向到本機存放區**– 會掛接您本機存放區做為遠端檔案系統上的遠端電腦。
4. 點選 [**儲存**]。

需要編輯這些設定？ 點選****（...） 的桌面，名稱旁邊的溢位功能表，然後點選 [**編輯**。

想要刪除的連接嗎？ 同樣地，點選溢位功能表 （**...**），然後點選 [**移除**。

>[!TIP]
> 如果您收到錯誤 0xf07 錯誤的密碼 （「 我們無法連線到遠端電腦因為使用者帳戶相關聯的密碼已過期 」），您變更密碼，並再試一次。

### 新增遠端資源
遠端資源是 RemoteApp 程式、 工作階段為基礎的桌面，以及發佈 RemoteApp 和桌面連線使用虛擬桌面。

若要新增遠端資源：

1. 在連線中心畫面上，點選**+**，然後點選 [**遠端資源摘要**。 
2. 輸入資訊的遠端資源：
   - **電子郵件或 URL** -RD Web 存取伺服器的 URL。 這會告訴搜尋與您的電子郵件地址相關聯的 RD Web 存取伺服器用戶端您也可以在此欄位 – 輸入您公司的電子郵件帳戶。
   - **使用者名稱**-使用 RD Web 存取伺服器，您要連線的使用者名稱。
   - **密碼**-使用您在連線到 RD Web 存取伺服器的密碼。
3. 點選 [**儲存**]。

遠端資源將會顯示在連線中心。


若要刪除遠端資源：

1. 在連線中心中，點選溢位功能表****（...） 旁邊的遠端資源。
2. 點選 \ [**移除**。
3. 確認刪除。

### Widget – 針腳已儲存的桌面，到您的主畫面

遠端桌面應用程式使用 Android widget 功能，支援釘選到您的主畫面的連線。 您將新增 widget 的方式取決於您使用的 Android 裝置和其作業系統的類型。 以下是最常見的方法，來新增 widget: 

1. 點選 \ [**應用程式**啟動應用程式] 功能表。
2. 點選**小工具**。
3. 撥動透過小工具，並尋找的遠端桌面圖示與描述，「 Pin 遠端桌面 」。
4. 點選按住該遠端桌面 widget 並將它移到主畫面。
5. 當您放開圖示時，您會看到已儲存的遠端桌面。 選擇您想要儲存到您的主畫面的連線。

現在您可以直接從您的主畫面開始遠端桌面連線，藉由點選它。

> [!NOTE]
> 如果您重新命名的桌面連線的遠端桌面應用程式中，不會更新這個釘選的遠端桌面的標籤。

## 連線到 RD 閘道來存取內部的資產

遠端桌面閘道 （RD 閘道） 可讓您從任何地方連線至遠端電腦上的公司網路上網際網路。 您可以建立和管理您的閘道使用遠端桌面用戶端。

若要設定新的閘道︰

1. 在連線中心中，點選 \ [**設定 > 閘道**。 點選**+** 以新增新的閘道。
2. 輸入下列資訊：
  - **伺服器名稱**– 您想要為閘道使用的電腦的名稱。 這可以是 Windows 的電腦名稱、 網際網路網域名稱或 IP 位址。 您也可以新增連接埠資訊的伺服器名稱 (例如： **RDGateway:443**或**10.0.0.1:443**)。
  - **使用者名稱**-使用者名稱和密碼，以做為您在連線到遠端桌面閘道。 您也可以選取**要使用桌面的使用者帳戶**使用那些遠端桌面連線使用相同的認證。

## 管理您的使用者帳戶

當您連接到桌面或遠端資源時，您可以儲存一次從選取的使用者帳戶。 您也可以在用戶端本身，而不是您連線到桌面時，儲存使用者資料中定義的使用者帳戶。

若要建立新的使用者帳戶：

1. 在連線中心中，點選 [**設定**]，然後點選**使用者帳戶**。
2. 點選**+** 以新增新的使用者帳戶。
3. 輸入下列資訊：
   - **使用者名稱**-使用者儲存與遠端連線搭配使用的名稱。 您可以在任何下列格式輸入使用者名稱： 使用者名稱，domain\user_name，或 user_name@domain.com。
   - **密碼**-您指定的使用者的密碼。 您想要用來進行遠端連線儲存每個使用者帳戶必須具備與其關聯的密碼。
4. 點選 [**儲存**]。

若要刪除的使用者帳戶：

1. 在連線中心中，點選 \ [**設定 > 使用者帳戶**。
2. 點選和按住即可選取清單中的使用者帳戶。 您可以選取多個使用者。
3. 點選 \ [刪除選取的使用者資源回收筒。

## 瀏覽遠端桌面工作階段
當您開始遠端桌面連線時，有一些工具可用，您可以用來瀏覽工作階段。

### 啟動遠端桌面連線

1. 點選 \ [開始工作階段的遠端桌面連線。 
2. 如果系統要求您驗證遠端桌面的憑證，請點選 [**連線**。 您也可以選取 [**不詢問我再次為連線到此電腦**一律接受憑證。

### 管理通用應用程式設定

您可以在您的 Android 用戶端設定下列全域設定：

- **顯示桌面預覽**-可讓您連線到它之前，請參閱連線中心中的桌上型電腦的預覽。 根據預設，這是設定為**開啟**。
- **透過捏合進行縮放**-可讓您使用捏合進行縮放手勢。 如果您使用透過遠端桌面應用程式支援多點觸控 （於 Windows 8 引進），開啟此設定**關閉**。
- **以協助改善遠端桌面**-會傳送給 Microsoft 的匿名資料。 我們使用此資料來改善用戶端。 您可以深入了解我們如何處理此匿名、 私人資料，請參閱[遠端桌面用戶端隱私權聲明](https://www.microsoft.com/privacystatement/RemoteApp/Default.aspx)。 根據預設，此設定為**開啟**。
- **顯示**-有兩個全域設定，供您顯示：
   - **方向**-設定您的工作階段的慣用的方向 （橫向或直向）。 
   >[!NOTE]
   > 如果您連線到執行 Windows 8 或 Windows 的較舊版本的電腦，工作階段將不會正確縮放。 您的最佳做法是從電腦、 中斷連線，然後重新連接，以您想要使用的方向。 更好的選項是至少升級電腦以 Windows 8.1。

   - **解析度**-會設定您想要使用適用於桌面連線全域的解析度。 如果您已經設定自訂的個別的 app 或連線解析度，此設定將不會變更顏色。
   >[!NOTE]
   >當您變更其中一個顯示設定時，它們只適用於新連線從該點上。 若要查看您已經連線中斷連線，然後重新連線至工作階段中的變更。

### 連線列

連線列可讓您存取其他瀏覽控制項。 根據預設，連接列放在螢幕頂端中間。 點兩下和拖曳列向左或向右移動它。

- **移動瀏覽控制項**： 移動瀏覽控制項可讓螢幕，到放大和四處移動。 請注意，移動瀏覽控制項才可以使用直接觸碰。
   - 啟用 / 停用移動瀏覽控制項： 點選移動瀏覽列中的圖示連線顯示移動瀏覽控制項和縮放螢幕。 點選 \ [移動瀏覽列中的圖示連線一次可隱藏控制項並返回其原始的解析度螢幕。
   - 使用移動瀏覽控制項： 點選和按住不放移動瀏覽控制項，並在您想要移動螢幕方向然後拖曳。
   - 移動的移動瀏覽控制項： 點兩下和保留移動瀏覽控制項來移動在畫面上的控制項。
- **其他選項**： 點選來顯示工作階段選取項目列和命令列 （如下所示） 的其他選項圖示。
- **鍵盤**： 點選以顯示或隱藏鍵盤的鍵盤圖示。 顯示鍵盤時，會自動顯示移動瀏覽控制項。
- **移動的連線列**： 點選和按住不放在連線列，然後拖曳和拖放在螢幕頂端的新位置。


### 命令列

點選 \ [連線列在右邊的螢幕上顯示的命令列。 您可以在滑鼠模式 （直接觸控和滑鼠指標） 之間切換。 若要從命令列返回連線中心使用首頁按鈕。 或者您可以使用 [上一頁] 按鈕，針對相同的動作。 您使用中的工作階段將不會中斷連線。 


### 在遠端工作階段中直接使用觸控手勢與滑鼠模式

用戶端會使用標準的觸控手勢。 您也可以使用觸控手勢來複寫的遠端桌面的滑鼠動作。 下表定義可用的滑鼠模式。

> [!NOTE]
> 互動與 Windows 8 或更新版本的原生的觸控手勢支援直接觸控模式。 

| 滑鼠模式    | 滑鼠作業      | 手勢                                                               |
|---------------|----------------------|-----------------------------------------------------------------------|
| 直接觸控  | 按一下滑鼠左鍵           | 1 指點選                                                          |
| 直接觸控  | 按一下滑鼠右鍵          | 1 手指點選和按住不放                                                 |
| 滑鼠指標 | 縮放                 | 使用 2 個手指，並透過捏合進行縮放，或將手指分開的方向移動以縮小內容。 |
| 滑鼠指標 | 按一下滑鼠左鍵           | 1 指點選                                                          |
| 滑鼠指標 | 按一下滑鼠左鍵並拖曳  | double 1 手指的點選和按住不放，然後拖曳                               |
| 滑鼠指標 | 按一下滑鼠右鍵          | 2 個指點選                                                          |
| 滑鼠指標 | 按一下滑鼠右鍵並拖曳 | 2 個 double 手指的點選和按住不放，然後拖曳                               |
| 滑鼠指標 | 滑鼠滾輪          | 2 個手指點選和按住，然後拖曳向上或向下                           |
