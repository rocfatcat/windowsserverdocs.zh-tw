# <a name="get-started-with-remote-desktop-on-android"></a>開始使用在 Android 上的遠端桌面

>適用於：Windows 10，Windows 8.1、 Windows Server 2019、 Windows Server 2016、windows Server 2012 R2

您可以使用適用於 Android 的遠端桌面用戶端，才能使用 Windows 應用程式和桌面直接從您的 Android 裝置。

若要開始使用下列資訊。 請務必查看[常見問題集](remote-desktop-client-faq.md)如果您有任何問題。

> [!NOTE]
> - 想知道的 Android 用戶端的新版本嗎？ 請參閱[在 Android 上的遠端桌面最新消息？](android-whatsnew.md)
> 您可以執行用戶端，以及與安裝的 ChromeOS 53 Chromebook Android 4.1 和更新版本的裝置上。 深入了解 Android 應用程式在 Chrome[此處](https://sites.google.com/a/chromium.org/dev/chromium-os/chrome-os-systems-supporting-android-apps)。

## <a name="get-the-rd-client-and-start-using-it"></a>取得 RD 用戶端，並開始使用

請遵循下列步驟來開始使用遠端桌面 Android 裝置上：

1. 下載遠端桌面用戶端，從[Google Play](https://play.google.com/store/apps/details?id=com.microsoft.rdc.android)。 
2. [您的電腦設定成接受遠端連接](remote-desktop-allow-access.md)。
3. 將遠端桌面連線或存取遠端資源。 您可以使用連線來直接連接至 Windows 電腦和遠端資源，若要使用的 RemoteApp 程式、 工作階段為基礎的桌面，或虛擬桌面上發佈內部部署。 
4. 建立小工具，以便快速取得遠端桌面。

> [!NOTE]
> 如果您想要稍早飛行的新功能建議下載我們[Microsoft 遠端桌面 Beta](https://play.google.com/store/apps/details?id=com.microsoft.rdc.android.beta)應用程式從 Google Play 商店。 

### <a name="add-a-remote-desktop-connection"></a>新增遠端桌面連線

若要建立遠端桌面連線：

1. 在 「 連線中心 」 點選 **+** ，然後點選**桌面**。
2. 輸入您想要連接到下列資訊的電腦：
   - **電腦名稱**– 電腦的名稱。 這可以是 Windows 電腦名稱、 網際網路網域名稱或 IP 位址。 您也可以在電腦名稱加連接埠資訊 (例如**MyDesktop:3389**或是**10.0.0.1:3389**)。
   - **使用者名稱**– 要用來存取遠端電腦的使用者名稱。 您可以使用下列格式： *user_name*， *domain\user_name*，或<em>user_name@domain.com</em>。 您也可以指定是否要提示輸入使用者名稱和密碼。
3. 您也可以設定下列其他選項：
   - **易記名稱**– 您要連接到電腦以易記名稱。 您可以使用任何字串，但如果您未指定好記的名稱，會顯示電腦名稱。
   - **閘道**– 您想要用於連線到虛擬桌面、 RemoteApp 程式以及以工作階段為基礎的桌面，在公司內部網路上的遠端桌面閘道。 從您的系統管理員取得閘道的相關資訊。
    需要設定遠端桌面閘道？
   - **音效**– 選擇要用於遠端工作階段期間的音訊裝置。 您可以選擇播放音效，在本機裝置，也就是遠端裝置上，或者完全不要。
   - **自訂顯示解析度**-藉由啟用此設定中設定連線的自訂解決方案。 套用解決方案關閉時您必須定義在應用程式的通用設定。
   - **交換滑鼠按鈕**– 使用此選項可為滑鼠右按鈕交換滑鼠左的按鈕函式。 （這是特別有用，如果遠端電腦已設定為慣用左手的使用者，但您有使用右手的滑鼠）。
   - **連線到系統管理員工作階段**-使用此選項來連線到主控台工作階段管理的 Windows 伺服器。
   - **重新導向至本機儲存體**– 做為遠端檔案系統掛接您的本機儲存體，在遠端電腦上。
4. 點選**儲存**。

需要編輯這些設定嗎？ 點選 溢位功能表 ( **...** ) 以及與的桌面，然後點選名稱旁邊**編輯**。

想要刪除連線嗎？ 同樣地，請點選 溢位功能表 ( **...** )，然後點選**移除**。

>[!TIP]
> 如果您收到的錯誤 0xf07 （「 我們無法連接到遠端電腦因為使用者帳戶相關聯的密碼已過期 」） 的密碼不正確，就會變更密碼，並再試一次。

### <a name="add-a-remote-resource"></a>新增遠端資源
遠端資源是 RemoteApp 程式、 工作階段為基礎的桌面及發佈使用 RemoteApp 與桌面連線的虛擬桌面。

若要新增的遠端資源：

1. 在 「 連線中心 」 畫面上，點選 **+** ，然後點選**遠端資源摘要**。 
2. 輸入遠端資源的資訊：
   - **電子郵件或 URL** -RD Web 存取伺服器的 URL。 您也可以輸入您的公司電子郵件帳戶，在此欄位 – 這會告訴用戶端搜尋與您的電子郵件地址相關聯的 RD Web 存取伺服器。
   - **使用者名稱**-要用於您要連接到 RD Web 存取伺服器的使用者名稱。
   - **密碼**-針對您要連接到 RD Web 存取伺服器使用的密碼。
3. 點選**儲存**。

遠端資源將會顯示在 「 連線中心。


若要刪除遠端資源：

1. 在 「 連線中心 」 中，點選 溢位功能表 ( **...** ) 旁邊的遠端資源。
2. 點選**移除**。
3. 確認刪除。

### <a name="widgets--pin-a-saved-desktop-to-your-home-screen"></a>Widget – 釘選到主畫面已儲存的桌面

遠端桌面應用程式使用 Android 的小工具的功能來支援釘選到主畫面的連線。 您將新增小工具的方式取決於您使用的 Android 裝置和其作業系統的類型。 以下是最常見的方式加入的 widget: 

1. 點選**應用程式**以啟動應用程式功能表。
2. 點選**widget**。
3. 透過 widget 撥動，並尋找說明，「 釘選遠端桌面 」 使用遠端桌面圖示
4. 點選 按住該遠端桌面小工具，並將它移到主畫面。
5. 當您釋放圖示時，您會看到已儲存的遠端桌面。 選擇您想要儲存到主畫面連接。

現在您可以開始直接從您的主畫面的遠端桌面連線，藉由點選它。

> [!NOTE]
> 如果您重新命名的桌面的連線，遠端桌面應用程式中，將不會更新這個固定的遠端桌面的標籤。

## <a name="connect-to-an-rd-gateway-to-access-internal-assets"></a>連接至 RD 閘道存取內部資產

遠端桌面閘道 （RD 閘道） 可讓您從任何地方連線到公司網路上的遠端電腦在網際網路上。 您可以建立和管理您使用遠端桌面用戶端的閘道。

若要設定新的閘道：

1. 在 「 連線中心 」 中，點選**設定 > 閘道**。 點選 **+** 新增新的閘道。
2. 輸入下列資訊：
   - **伺服器名稱**– 您想要作為閘道使用的電腦名稱。 這可以是 Windows 電腦名稱、 網際網路網域名稱或 IP 位址。 您也可以新增連接埠資訊的伺服器名稱 (例如：**RDGateway:443**或是**10.0.0.1:443**)。
   - **使用者名稱**-使用者名稱和密碼用於您要連接到遠端桌面閘道。 您也可以選取**使用桌面的使用者帳戶**所使用的遠端桌面連線使用相同的認證。

## <a name="manage-your-user-accounts"></a>管理您的使用者帳戶

當您連線到桌面或遠端資源時，您可以儲存一次從選取的使用者帳戶。 您也可以定義在用戶端本身，而不是儲存使用者資料，當您連線到桌面的使用者帳戶。

若要建立新的使用者帳戶：

1. 在 「 連線中心 」 中，點選**設定**，然後點選**使用者帳戶**。
2. 點選 **+** 來新增新的使用者帳戶。
3. 輸入下列資訊：
   - **使用者名稱**-要儲存以供使用，透過遠端連線的使用者名稱。 您可以輸入使用者名稱，任何下列格式： user_name domain\user_name，或user_name@domain.com。
   - **密碼**-您指定的使用者的密碼。 您想要將儲存到使用遠端連線的每個使用者帳戶必須具有與其相關聯的密碼。
4. 點選**儲存**。

若要刪除的使用者帳戶：

1. 在 「 連線中心 」 中，點選**設定 > 使用者帳戶**。
2. 點選並按住以選取清單中的使用者帳戶。 您可以選取多個使用者。
3. 點選垃圾筒來刪除選取的使用者。

## <a name="navigate-the-remote-desktop-session"></a>瀏覽遠端桌面工作階段
當您啟動遠端桌面連線時，有一些工具可用，您可以使用瀏覽工作階段。

### <a name="start-a-remote-desktop-connection"></a>啟動遠端桌面連線

1. 點選 遠端桌面連線，啟動工作階段。 
2. 如果系統要求您驗證遠端桌面憑證時，點選**Connect**。 您也可以選取**不要再詢問我連線到此電腦的**一律接受憑證。

### <a name="manage-global-app-settings"></a>管理全域應用程式設定

在您的 Android 用戶端中，您可以設定下列全域設定：

- **顯示桌面預覽**-可讓您連線到它之前，請參閱在 「 連線中心桌面的預覽。 根據預設，這設定為**上**。
- **捏合縮放**-可讓您使用捏合以縮放筆勢。 如果您使用透過遠端桌面應用程式支援多點觸控 （在 Windows 8 中引進），開啟這項設定**關閉**。
- **改善遠端桌面說明**-傳送給 Microsoft 的匿名資料。 我們可以使用這項資料來改善用戶端。 您可以深入了解我們如何處理此匿名、 私用的資料，請參閱[遠端桌面用戶端 Privacy Statement](https://www.microsoft.com/privacystatement/RemoteApp/Default.aspx)。 根據預設，此設定是**上**。
- **顯示**-有顯示兩個全域設定：
  - **方向**-設定您的工作階段的慣用的方向 （橫向或縱向）。 
    >[!NOTE]
    > 如果您連接到執行 Windows 8 或較舊版本的 Windows 電腦，工作階段不會正確縮放。 最好是在電腦中斷連線，然後再重新連接以您想要使用的方向。 更好的選擇是至少升級電腦 Windows 8.1。

  - **解析**-設定您想要使用桌面連接的全域的解析度。 如果您已設定個別的應用程式或連接的自訂解決方案，不會變更此設定的。
    >[!NOTE]
    >當您變更顯示設定的其中一個時，而只適用於新的連線從該點上。 若要查看您已連線到中斷連線，然後再重新連線工作階段中的變更。

### <a name="connection-bar"></a>連線列

連接列可讓您存取其他導覽控制項。 根據預設，連線列位於畫面頂端的中間。 點選和拖曳列向左或右移動它。

- **移動瀏覽控制項**:取景位置調整控制項讓螢幕可以放大及四處移動。 請注意取景位置調整控制項只會使用直接觸控。
   - 啟用/停用移動瀏覽控制項：點選 [移動瀏覽列中的圖示連線顯示取景位置調整控制項中與縮放] 畫面。 點選 移動瀏覽列中的圖示連線一次，若要隱藏控制項，並傳回其原始的解決方式的畫面。
   - 使用移動瀏覽控制項：點選 按住移動瀏覽控制項，然後在您想要移動螢幕的方向拖曳。
   - 移動取景位置調整控制項：Double 點選並按住 取景位置調整控制項移動螢幕上的控制項。
- **其他選項**:點選 [其他選項] 圖示，以顯示工作階段選取列和命令列 （如下所示）。
- **鍵盤**:點選 [鍵盤] 圖示以顯示或隱藏鍵盤。 顯示鍵盤時，會自動顯示取景位置調整控制項。
- **移動連線列**:點選並按住 [連線] 列中，然後拖放到新位置上方的螢幕。


### <a name="command-bar"></a>命令列

點選 連線列，以顯示在畫面右手邊的命令列。 您可以切換滑鼠模式 （直接觸控及滑鼠指標）。 使用 [首頁] 按鈕以返回 「 連線中心 」，從命令列。 或者您可以使用 [上一頁] 按鈕相同的動作。 將不會中斷您的使用中工作階段。 


### <a name="use-direct-touch-gestures-and-mouse-modes-in-a-remote-session"></a>在遠端工作階段中使用直接觸控筆勢和滑鼠模式

用戶端會使用標準的觸控筆勢。 您也可以使用觸控筆勢來複寫的遠端桌面的滑鼠動作。 下表中定義可用的滑鼠模式。

> [!NOTE]
> 互動 Windows 8 或更新版本的原生的觸控筆勢支援直接觸控模式。 

| 滑鼠模式    | 滑鼠作業      | 手勢                                                               |
|---------------|----------------------|-----------------------------------------------------------------------|
| 直接觸控  | 按一下滑鼠左鍵           | 1 的手指點選                                                          |
| 直接觸控  | 按一下滑鼠右鍵          | 1 手指點選並按住                                                 |
| 滑鼠指標 | 縮放                 | 使用 2 個手指，並縮小來放大或縮小移動手指分開的。 |
| 滑鼠指標 | 按一下滑鼠左鍵           | 1 的手指點選                                                          |
| 滑鼠指標 | 按一下滑鼠左鍵並拖曳  | 1 的手指 double 點選並按住，然後拖曳                               |
| 滑鼠指標 | 按一下滑鼠右鍵          | 2 個手指點選                                                          |
| 滑鼠指標 | 按一下滑鼠右鍵並拖曳 | 2 個手指 double 點選並按住，然後拖曳                               |
| 滑鼠指標 | 滑鼠滾輪          | 2 個手指點選並按住不放，然後向上或向下拖曳                           |
