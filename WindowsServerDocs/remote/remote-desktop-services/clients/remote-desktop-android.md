# <a name="get-started-with-remote-desktop-on-android"></a>在 Android 上開始使用遠端桌面

>適用於：Windows 10、Windows 8.1、Windows Server 2019、Windows Server 2016、Windows Server 2012 R2

您可以使用適用於 Android 的遠端桌面用戶端，直接從 Android 裝置使用 Windows 應用程式和桌面。

請使用下列資訊立即開始。 如果您有任何問題，請務必查看[常見問題集](remote-desktop-client-faq.md)。

> [!NOTE]
> - 想知道 Android 用戶端的新版本嗎？ 請參閱 [Android 上的遠端桌面有什麼新功能？](android-whatsnew.md)
> 您可以在 Android 4.1 和更新版本的裝置上，以及已安裝 ChromeOS 53 的 Chromebook 上執行用戶端。 請在[這裡](https://sites.google.com/a/chromium.org/dev/chromium-os/chrome-os-systems-supporting-android-apps)深入了解 Chrome 上的 Android 應用程式。

## <a name="get-the-rd-client-and-start-using-it"></a>取得 RD 用戶端並開始使用它

依照下列步驟，在 Android 裝置上開始使用遠端桌面：

1. 從 [Google Play 商店](https://play.google.com/store/apps/details?id=com.microsoft.rdc.android) 下載遠端桌面用戶端。 
2. [將您的電腦設定成接受遠端連線](remote-desktop-allow-access.md)。
3. 新增遠端桌面連線或遠端資源。 您可以使用連線直接連線至 Windows 電腦和遠端資源，以使用 RemoteApp 程式、工作階段型桌面，或在內部部署發佈的虛擬桌面。 
4. 建立小工具，以便快速前往遠端桌面。

> [!NOTE]
> 如果您想要提早享用新功能，建議從 Google Play 商店下載我們的 [Microsoft Remote Desktop Beta](https://play.google.com/store/apps/details?id=com.microsoft.rdc.android.beta) 應用程式。 

### <a name="add-a-remote-desktop-connection"></a>新增遠端桌面連線

建立遠端桌面連線：

1. 在 [連線中心] 內點選 **+** ，然後點選 [桌面]  。
2. 針對您想要連線的電腦，輸入下列資訊：
   - **電腦名稱** – 電腦的名稱。 這可以是 Windows 電腦名稱、網際網路網域名稱或 IP 位址。 您也可以將連接埠資訊附加到電腦名稱 (例如 **MyDesktop:3389** 或 **10.0.0.1:3389**)。
   - **使用者名稱** – 用來存取遠端電腦的使用者名稱。 您可以使用下列格式：*user_name*、*domain\user_name* 或 <em>user_name@domain.com</em>。 您也可以指定是否要提示輸入使用者名稱與密碼。
3. 您也可以設定下列其他選項：
   - **易記名稱** – 您要連線之電腦的易記名稱。 您可以使用任何字串，但如果您未指定易記名稱，會顯示電腦名稱。
   - **閘道** – 您要用來連線至公司內部網路上的虛擬桌面、RemoteApp 程式以及工作階段型桌面的遠端桌面閘道。 向您的系統管理員取得閘道的相關資訊。
    需要設定遠端桌面閘道嗎？
   - **音效** – 選擇要在遠端工作階段期間用於音訊的裝置。 您可以選擇在本機裝置、遠端裝置上播放音效，或者完全不要播放音效。
   - **自訂顯示器解析度** - 啟用此設定可為連線設定自訂解析度。 關閉時會套用您已經在應用程式的通用設定中定義的解析度。
   - **切換滑鼠按鈕** – 使用此選項可以切換左滑鼠按鈕功能以供右滑鼠按鈕使用 (如果遠端電腦是針對慣用左手的使用者而設定，但您使用右手的滑鼠，則這特別有用)。
   - **連線到系統管理員工作階段** - 使用此選項連線到主控台工作階段以管理 Windows 伺服器。
   - **重新導向至本機儲存體** – 將您的本機儲存體當作遠端檔案系統，掛接在遠端電腦上。
4. 點選 [儲存]  。

需要編輯這些設定嗎？ 點選桌面名稱旁邊的溢位功能表 ( **...** )，然後點選 [編輯]  。

想要刪除連線嗎？ 再次點選溢位功能表 ( **...** )，然後點選 [移除]  。

>[!TIP]
> 如果您收到有關密碼錯誤的錯誤 0xf07 (「無法連線到遠端電腦，因為與使用者帳戶關聯的密碼已過期」)，請變更密碼，然後再試一次。

### <a name="add-a-remote-resource"></a>新增遠端資源
遠端資源包括 RemoteApp 程式、工作階段型桌面，以及使用 RemoteApp 與桌面連線發佈的虛擬桌面。

新增遠端資源：

1. 在 [連線中心] 畫面上，點選 **+** ，然後點選 [遠端資源摘要]  。 
2. 輸入遠端資源的資訊：
   - **電子郵件或 URL** - RD Web 存取伺服器的 URL。 您也可以在此欄位中輸入您的公司電子郵件帳戶，如此會告訴用戶端搜尋與您電子郵件地址相關聯的 RD Web 存取伺服器。
   - **使用者名稱** - 用於您要連線之 RD Web 存取伺服器的使用者名稱。
   - **密碼** - 用於您要連線之 RD Web 存取伺服器的密碼。
3. 點選 [儲存]  。

遠端資源將會顯示在 [連線中心] 內。


刪除遠端資源：

1. 在 [連線中心] 內，點選遠端資源旁邊的溢位功能表 ( **...** )。
2. 點選 [移除]  。
3. 確認刪除。

### <a name="widgets--pin-a-saved-desktop-to-your-home-screen"></a>Widget – 將已儲存的桌面釘選到主畫面

遠端桌面應用程式支援使用 Android 的 Widget 功能，將連線釘選到主畫面。 您新增 Widget 的方式取決於您所使用的 Android 裝置與其作業系統的類型。 以下是新增 Widget 最常見的方式： 

1. 點選 [應用程式]  以啟動 [應用程式] 功能表。
2. 點選 [Widget]  。
3. 撥動 Widget 清單，並尋找描述為「釘選遠端桌面」的 [遠端桌面] 圖示。
4. 點選並按住該遠端桌面 Widget，並將它移到主畫面。
5. 當您放開圖示時，您將看到已儲存的遠端桌面。 選擇您想要儲存到主畫面的連線。

現在您可以點選它，直接從主畫面開始遠端桌面連線。

> [!NOTE]
> 如果您在遠端桌面應用程式中重新命名桌面連線，將不會更新這個已釘選遠端桌面的標籤。

## <a name="connect-to-an-rd-gateway-to-access-internal-assets"></a>連線至 RD 閘道以存取內部資產

遠端桌面閘道 (RD 閘道) 可讓您從網際網路上的任何地方連線到公司網路上的遠端電腦。 您可以使用遠端桌面用戶端來建立和管理您的閘道。

設定新的閘道：

1. 在 [連線中心] 內，點選 [設定] > [閘道]  。 點選 **+** 來新增閘道。
2. 輸入下列資訊：
   - **伺服器名稱** – 您想要當作閘道使用之電腦的名稱。 這可以是 Windows 電腦名稱、網際網路網域名稱或 IP 位址。 您也可以將連接埠資訊加入至伺服器名稱 (例如：**RDGateway:443** 或 **10.0.0.1:443**)。
   - **使用者名稱** - 用於您要連線之遠端桌面閘道的使用者名稱與密碼。 您也可以選取 [使用桌面使用者帳戶]  ，以使用針對遠端桌面連線所使用的相同認證。

## <a name="manage-your-user-accounts"></a>管理您的使用者帳戶

當您連線到桌面或遠端資源時，您可以儲存使用者帳戶以便再次從中選取。 您也可以在用戶端本身定義使用者帳戶，而不是在連線到桌面時儲存使用者資料。

建立新的使用者帳戶：

1. 在 [連線中心] 內，點選 [設定]  ，然後點選 [使用者帳戶]  。
2. 點選 **+** 以新增使用者帳戶。
3. 輸入下列資訊：
   - **使用者名稱** - 儲存以搭配遠端連線使用之使用者的名稱。 您可以使用下列任何一種格式輸入使用者名稱：user_name、domain\user_name 或 user_name@domain.com。
   - **密碼** - 用於指定之使用者的密碼。 您想要儲存以用於遠端連線的每個使用者帳戶都必須有與其相關聯的密碼。
4. 點選 [儲存]  。

刪除使用者帳戶：

1. 在 [連線中心] 內，點選 [設定] > [使用者帳戶]  。
2. 點選並按住清單中的使用者帳戶以選取它。 您可以選取多個使用者。
3. 點選垃圾筒以刪除選取的使用者。

## <a name="navigate-the-remote-desktop-session"></a>瀏覽遠端桌面工作階段
當您開始遠端桌面連線時，提供一些您可以用來瀏覽工作階段的工具。

### <a name="start-a-remote-desktop-connection"></a>開始遠端桌面連線

1. 點選遠端桌面連線以開始工作階段。 
2. 如果系統要求您驗證遠端桌面的憑證，請點選 [連線]  。 您也可以選取 [不要再詢問我是否要連線到這部電腦]  以一律接受憑證。

### <a name="manage-global-app-settings"></a>管理全域應用程式設定

您可以在 Android 用戶端中設定下列全域設定：

- **顯示桌面預覽** - 可讓您在連線到桌面之前，在 [連線中心] 內查看其預覽。 根據預設，這設定為 [開啟]  。
- **捏合以縮放** - 可讓您使用捏合以縮放手勢。 如果您要透過遠端桌面使用的應用程式支援多點觸控 (在 Windows 8 中引進)，請將此設定 [關閉]  。
- **協助改善遠端桌面** - 傳送匿名資料給 Microsoft。 我們將會使用此資料來改善用戶端。 您可以深入了解我們如何處理這個匿名的私人資料，請參閱[遠端桌面用戶端隱私權聲明](https://www.microsoft.com/privacystatement/RemoteApp/Default.aspx)。 根據預設，此設定為 [開啟]  。
- **顯示** - 顯示器有兩個全域設定：
  - **方向** - 為您的工作階段設定慣用的方向 (橫向或值向)。 
    >[!NOTE]
    > 如果您連線到執行 Windows 8 或舊版 Windows 的電腦，工作階段不會正確縮放。 最好是中斷與電腦的連線，然後以您想要使用的方向重新連線。 更好的選擇是將電腦升級至至少 Windows 8.1。

  - **解析度** - 設定您想要用於全域桌面連線的解析度。 如果您已經為個別的應用程式或連線設定自訂解析度，此設定不會變更該解析度。
    >[!NOTE]
    >當您變更其中一個顯示器設定時，它們只會套用到從那時之後開始的新連線。 若要查看您已連線之工作階段中的變更，請中斷連線，然後再重新連線。

### <a name="connection-bar"></a>連線列

連線列可讓您存取其他瀏覽控制項。 根據預設，連線列位於畫面頂端的中間。 點兩下，並將連線列向左或右拖曳以移動它。

- **移動瀏覽控制項**：移動瀏覽控制項可讓畫面放大及四處移動。 請注意，只有直接輕觸才可以使用移動瀏覽控制項。
   - 啟用/停用移動瀏覽控制項：點選連線列中的 [移動瀏覽] 圖示可顯示移動瀏覽控制項並縮放畫面。 再次點選連線列中的 [移動瀏覽] 圖示可隱藏控制項，並將畫面回復到其原始解析度。
   - 使用移動瀏覽控制項：點選並按住移動瀏覽控制項，然後以您想要移動畫面的方向拖曳。
   - 移動移動瀏覽控制項：點兩下並按住移動瀏覽控制項，將控制項移動到畫面上。
- **其他選項**：點選 [其他選項] 圖示以顯示工作階段選取列和命令列 (如下所示)。
- **鍵盤**：點選 [鍵盤] 圖示以顯示或隱藏鍵盤。 移動瀏覽控制項會在鍵盤顯示時自動顯示。
- **移動連線列**：點選並按住連線列，然後拖放到畫面頂端的新位置。


### <a name="command-bar"></a>命令列

點選連線列以顯示畫面右側的命令列。 您可以切換滑鼠模式 (直接輕觸與滑鼠指標)。 使用主畫面按鈕以從命令列返回連線中心。 或者，您可以使用 [返回] 按鈕來執行相同的動作。 系統將不會中斷您的使用中工作階段連線。 


### <a name="use-direct-touch-gestures-and-mouse-modes-in-a-remote-session"></a>在遠端工作階段中使用直接輕觸手勢和滑鼠模式

用戶端會使用標準的輕觸手勢。 您也可以使用輕觸手勢來複寫遠端桌面上的滑鼠動作。 下表中定義可用的滑鼠模式。

> [!NOTE]
> 與 Windows 8 或更新版本互動時，原生輕觸手勢在直接輕觸模式中受到支援。 

| 滑鼠模式    | 滑鼠操作      | 手勢                                                               |
|---------------|----------------------|-----------------------------------------------------------------------|
| 直接輕觸  | 按一下滑鼠左鍵           | 1 隻手指點選                                                          |
| 直接輕觸  | 按一下滑鼠右鍵          | 1 隻手指點選並按住                                                 |
| 滑鼠指標 | 縮放                 | 將 2 隻手指捏合以放大；將 2 隻手指分開以縮小。 |
| 滑鼠指標 | 按一下滑鼠左鍵           | 1 隻手指點選                                                          |
| 滑鼠指標 | 按一下滑鼠左鍵並拖曳  | 1 隻手指點兩下並按住，然後拖曳                               |
| 滑鼠指標 | 按一下滑鼠右鍵          | 2 隻手指點選                                                          |
| 滑鼠指標 | 按一下滑鼠右鍵並拖曳 | 2 隻手指點兩下並按住，然後拖曳                               |
| 滑鼠指標 | 滑鼠滾輪          | 2 隻手指點選並按住，然後向上或向下拖曳                           |
