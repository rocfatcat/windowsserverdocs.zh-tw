# IOS 上開始使用遠端桌面

>適用於： Windows 10、 Windows 8.1、 Windows Server 2019、 Windows Server 2016、 Windows Server 2012 R2

您可以使用適用於 iOS 的遠端桌面用戶端可搭配使用 Windows 應用程式、 資源和桌上型電腦從您的 iOS 裝置 （iPhones 和 iPads）。

若要開始使用下列資訊。 請務必查看[常見問題集](remote-desktop-client-faq.md)，如果您有任何問題。

> [!NOTE]
> - 好奇 iOS 用戶端的新發行版本？ 請查看[iOS 上新的遠端桌面？](ios-whatsnew.md)
> - IOS 用戶端支援裝置執行 iOS 6.x 和更新版本。

## 取得遠端桌面用戶端，並開始使用它

### 從 iOS 市集下載遠端桌面用戶端
請依照下列步驟來開始使用遠端桌面 iOS 裝置上：

1. 從[iTunes](https://itunes.apple.com/us/app/microsoft-remote-desktop/id714464092?mt=8)下載 Microsoft 遠端桌面用戶端。
2. [設定您的電腦以接受遠端連線](remote-desktop-client-faq.md#how-do-i-set-up-a-pc-for-remote-desktop)。
3. 新增[遠端桌面連線](#add-a-remote-desktop-connection)或[遠端資源](#add-a-remote-resource)。 您可以使用連線來連線到直接 Windows 電腦和遠端資源使用 RemoteApp 程式，工作階段為基礎的桌面或虛擬桌面發行至內部部署使用 RemoteApp 和桌面連線。 通常在公司環境中使用此功能。

### 下載遠端桌面 iOS 搶鮮版用戶端
在 iOS 裝置上，請依照[下列指示](https://aka.ms/rdiosbeta)來下載遠端桌面 iOS 搶鮮版用戶端。

### 新增遠端桌面連線

若要建立遠端桌面連線： 
1. 在連線中心點選**+**，然後點選 [**新增電腦或伺服器**。
2. 輸入遠端桌面連線的下列資訊：
  - **電腦名稱**– 電腦的名稱。 這可以是 Windows 的電腦名稱、 網際網路網域名稱或 IP 位址。 您也可以附加到電腦名稱 （例如， **MyDesktop:3389**或**10.0.0.1:3389**） 的連接埠資訊。
  - **使用者名稱**– 要用來存取遠端電腦的使用者名稱。 您可以使用下列格式：*使用者名稱*、 *domain\user_name*或*user_name@domain.com*。 您也可以指定是否要提示您輸入使用者名稱和密碼。
3. 您也可以設定下列選項：
  - **易記名稱 （選擇性）** – 容易記住您連線到電腦的名稱。 您可以使用任何字串，但如果您未指定的易記名稱，會顯示電腦名稱。
  - **閘道 （選擇性）** – 您想要用來連線到虛擬桌面、 RemoteApp 程式及工作階段式桌面內部的公司網路上的遠端桌面閘道。 從您的系統管理員取得閘道的相關資訊。
  - **音效**– 選取裝置要在遠端工作階段期間用於音訊。 您可以選擇播放聲音在本機裝置，在遠端裝置上，或完全不。
  - 它**交換滑鼠按鈕**– 每當滑鼠手勢會傳送一個命令，以滑鼠左鍵時，請改為傳送相同的命令，以滑鼠右鍵按鈕。 這是必要的如果遠端電腦設定為慣用左手的滑鼠模式。
  - **系統管理員模式**-連線到執行 Windows Server 2003 或更新版本的伺服器上的系統管理工作階段。
4. 點選 [**儲存**]。

需要編輯這些設定？ 按下和保留您想要編輯的桌面，然後點選 [設定] 圖示。 

### 新增遠端資源
遠端資源是 RemoteApp 程式、 工作階段為基礎的桌面，以及發佈 RemoteApp 和桌面連線使用虛擬桌面。

- URL 顯示連結到 RD Web 存取伺服器，讓您能夠存取 RemoteApp 和桌面連線。
- 列出已設定 RemoteApp 和桌面連線。

若要新增遠端資源：

1. 在連線中心畫面上，點選**+**，然後點選 [**新增遠端資源**。 
2. 輸入資訊的遠端資源：
   - **摘要的 URL** -RD Web 存取伺服器的 URL。 這會告訴搜尋與您的電子郵件地址相關聯的 RD Web 存取伺服器用戶端您也可以在此欄位 – 輸入您公司的電子郵件帳戶。
   - **使用者名稱**-使用 RD Web 存取伺服器，您要連線的使用者名稱。
   - **密碼**-使用您在連線到 RD Web 存取伺服器的密碼。
3. 點選 [**儲存**]。

遠端資源將會顯示在連線中心。


## 連線到 RD 閘道來存取內部的資產

遠端桌面閘道 （RD 閘道） 可讓您從任何地方連線至遠端電腦上的公司網路上網際網路。 您可以建立和管理您的閘道使用遠端桌面用戶端。

若要設定新的閘道︰

1. 在連線中心中，點選 \ [**設定 > 閘道**。 
2. 點選 \ [**新增遠端桌面閘道**。
3. 輸入下列資訊：
  - **伺服器名稱**– 您想要為閘道使用的電腦的名稱。 這可以是 Windows 的電腦名稱、 網際網路網域名稱或 IP 位址。 您也可以新增連接埠資訊的伺服器名稱 (例如： **RDGateway:443**或**10.0.0.1:443**)。
  - **使用者名稱**-使用者名稱和密碼，以做為您要連線的遠端桌面閘道。 您也可以選取 [**連線的認證**使用那些遠端桌面連線使用的相同的使用者名稱和密碼。


## 管理您的使用者帳戶 

當您連接到桌面或遠端資源時，您可以儲存一次從選取的使用者帳戶。 您可以使用遠端桌面用戶端來管理您的使用者帳戶。

若要建立新的使用者帳戶：

1. 在連線中心中，點選 [**設定**]，然後點選**使用者名稱**。
2. 點選 \ [**新增使用者帳戶**。
3. 輸入下列資訊：
   - **使用者名稱**-使用者儲存與遠端連線搭配使用的名稱。 您可以在任何下列格式輸入使用者名稱： 使用者名稱，domain\user_name，或 user_name@domain.com。
   - **密碼**-您指定的使用者的密碼。 您想要用來進行遠端連線儲存每個使用者帳戶必須具備與其關聯的密碼。
4. 點選 [**儲存**]，然後點選 [**設定**。
5. 點選 \ [**完成**儲存新的設定。

若要刪除的使用者帳戶：

1. 在連線中心中，點選 \ [**設定 > 使用者名稱**。
2. 撥動以選取使用者由右至左的資料列。
3. 點選 \ [**刪除**。



## 瀏覽遠端桌面工作階段
當您開始遠端桌面工作階段時，有一些工具可用，您可以用來瀏覽工作階段。

### 啟動遠端桌面連線

1. 點選 \ [遠端桌面連線來啟動遠端桌面工作階段。 
2. 如果系統要求您驗證遠端桌面的憑證，點選 \ [**接受]**。 您可以選擇**不要詢問我再次為連線到此電腦**的切換開關滑動至**上**一律接受。 

### 連線列

連線列可讓您存取其他瀏覽控制項。 

- **移動瀏覽控制項**： 移動瀏覽控制項可讓螢幕，到放大和四處移動。 請注意，移動瀏覽控制項才可以使用直接觸碰。
   - 啟用 / 停用移動瀏覽控制項： 點選移動瀏覽列中的圖示連線顯示移動瀏覽控制項和縮放螢幕。 點選 \ [移動瀏覽列中的圖示連線一次可隱藏控制項並返回其原始的解析度螢幕。
   - 使用移動瀏覽控制項： 點選和按住不放移動瀏覽控制項，並在您想要移動螢幕方向然後拖曳。
   - 移動的移動瀏覽控制項： 點兩下和保留移動瀏覽控制項來移動在畫面上的控制項。
- **連線名稱**： 會顯示目前的連線名稱。 點選以顯示在工作階段選取項目列連線名稱。
- **鍵盤**： 點選以顯示或隱藏鍵盤的鍵盤圖示。 顯示鍵盤時，會自動顯示移動瀏覽控制項。
- **移動的連線列**： 點選和按住不放在連線列，然後拖曳和拖放在螢幕頂端的新位置。

### 工作階段選取項目
您可以有多個連線同時開啟到不同的電腦。 點選 \ [連線列顯示在畫面左側的工作階段選取項目列。 在工作階段選取項目列可讓您檢視您開啟連線，並在它們之間切換。 

- 在開啟的遠端資源工作階段中的應用程式之間切換。

    當您連線到遠端資源時，您可以切換開啟的應用程式內點選展開工具功能表，從可用的項目清單中選擇該工作階段之間。
- 開始新的工作階段

  您可以開始新的應用程式或桌面工作階段，從您目前的連線內： 點選**新開始**，然後再從可用的項目清單中選擇。

- 中斷連線，工作階段

  若要中斷連線的工作階段點選 X 中的工作階段磚左邊。

### 命令列

命令列來取代從版本 8.0.1 列公用程式。 您可以在滑鼠模式之間切換，並從命令列回到連線中心。

## 在遠端工作階段中使用觸控手勢與滑鼠模式

用戶端會使用標準的觸控手勢。 您也可以使用觸控手勢來複寫的遠端桌面的滑鼠動作。 下表定義可用的滑鼠模式。

> [!NOTE]
> 互動與 Windows 8 或更新版本的原生的觸控手勢支援直接觸控模式。 如需有關 Windows 8 手勢，請參閱[觸控： 撥動、 點選、 和超越](https://windows.microsoft.com/en-US/windows-8/touch-swipe-tap-beyond)。

| 滑鼠模式    | 滑鼠作業      | 手勢                                                    |
|---------------|----------------------|------------------------------------------------------------|
| 直接觸控  | 按一下滑鼠左鍵           | 1 指點選                                               |
| 直接觸控  | 按一下滑鼠右鍵          | 1 手指點選和按住不放                                      |
| 滑鼠指標 | 按一下滑鼠左鍵           | 1 指點選                                               |
| 滑鼠指標 | 按一下滑鼠左鍵並拖曳  | double 1 手指的點選和按住不放，然後拖曳                    |
| 滑鼠指標 | 按一下滑鼠右鍵          | 2 個指點選                                               |
| 滑鼠指標 | 按一下滑鼠右鍵並拖曳 | 2 個 double 手指的點選和按住不放，然後拖曳                    |
| 滑鼠指標 | 滑鼠滾輪          | 2 個手指點選和按住，然後拖曳向上或向下                |
| 滑鼠指標 | 縮放                 | 捏合 2 手指來放大或散播 2 手指進行縮小 |

## 在遠端工作階段中使用鍵盤

您可以使用螢幕小鍵盤或在遠端工作階段中的實體鍵盤。

針對螢幕小鍵盤，使用的右側邊緣列在鍵盤上方的標準和其他鍵盤之間切換按鈕。

如果您的 iOS 裝置啟用藍牙時，用戶端會自動偵測藍牙鍵盤。

請注意，因為作業系統上的限制，而特殊的索引鍵，例如 Ctrl、 選項和函式將無法如預期運作使用藍牙鍵盤。 可搭配使用下列機碼：

- 英數字元鍵
- 方向鍵
- 索引標籤:] 索引標籤正常運作，但 Shift + Tab 無法運作
- 家用版 / Pos1: Alt + 向左鍵 = 家用版
- 結束： Alt + 向右 = 結束
- Page Up: Alt + 向上 = Page Up
- Page Down: Alt + 向下 = Page Down
- 選取所有： 命令 + A = Ctrl + A （在大部分的程式中選取全部）
- 剪下： 命令 + X = Ctrl + X （在大部分的程式剪下）
- 複製： 命令 + C = Ctrl + C （在大部分的程式複製）
- 貼上： 命令 + V = Ctrl + V （在大部分的程式中貼上）
- 符號： Alt + 英數字鍵就會產生不同的符號，根據所設定的語言
