# 開始使用 Windows 上的遠端桌面

>適用於： Windows 10、 Windows 8.1、 Windows Server 2019、 Windows Server 2016、 Windows Server 2012 R2

您可以使用適用於 Windows 的遠端桌面用戶端搭配 Windows 應用程式和從不同的 Windows 裝置的遠端桌面。

若要開始使用下列資訊。 請務必查看[常見問題集](remote-desktop-client-faq.md)，如果您有任何問題。

> [!NOTE]
> - 想了解新的 Windows 用戶端的版本嗎？ 請查看[新的 Windows 上的遠端桌面？](windows-whatsnew.md)
> - 您可以在任何版本的 Windows 10 上執行的用戶端。

## 取得 RD 用戶端，並開始使用它

請依照下列步驟來開始使用遠端桌面在您的 Windows 10 裝置上：

1. 從[Microsoft Store](https://www.microsoft.com/store/p/microsoft-remote-desktop/9wzdncrfj3ps)下載遠端桌面用戶端。 
2. [設定您的電腦以接受遠端連線](remote-desktop-allow-access.md)。
3. 新增遠端桌面連線或遠端資源。 您可以使用連線要直接連線到 Windows 電腦和遠端資源使用 RemoteApp 程式、 工作階段為基礎的桌面或虛擬桌面發行您的系統管理員。 
4. 釘選的項目，所以您可以快速取得遠端桌面。

### 新增遠端桌面連線

若要建立遠端桌面連線：

1. 在連線中心點選 **+ 新增**]，然後點選**桌面**。
2. 輸入您想要連線到下列資訊的電腦：
  - **電腦名稱**– 電腦的名稱。 這可以是 Windows 的電腦名稱、 網際網路網域名稱或 IP 位址。 您也可以附加到電腦名稱 （例如， **MyDesktop:3389**或**10.0.0.1:3389**） 的連接埠資訊。
  - **使用者帳戶**– 要用來存取遠端電腦的使用者帳戶。 點選**+** 新增新的帳戶，或選取現有的帳戶。 您可以使用下列格式的使用者名稱：*使用者名稱*、 *domain\user_name*或*user_name@domain.com*。 您也可以指定是否要提示您輸入使用者名稱和密碼連線期間透過選取**每次詢問我**。
3. 您也可以設定其他選項藉由點選上**顯示更多**的：
  - **顯示名稱**– 容易記住您連線到電腦的名稱。 您可以使用任何字串，但如果您未指定的易記名稱，會顯示電腦名稱。
  - **群組**– 指定一個群組，讓它更容易稍後找到您的連線。 您可以新增新的群組，點選**+** 或從清單中選取。
  - **閘道**– 您想要用來連線到虛擬桌面、 RemoteApp 程式及工作階段式桌面內部的公司網路上的遠端桌面閘道。 從您的系統管理員取得閘道的相關資訊。
  - **連線到系統管理員工作階段**-使用此選項以連線到主控台工作階段來管理 Windows server。
  - **交換滑鼠按鈕**– 使用此選項，以滑鼠右鍵按鈕交換滑鼠左鍵的按鈕函式。 （這是特別有用，如果遠端電腦已設定為慣用左手的使用者，但您使用慣用右手的滑鼠）。
  - **設為我遠端工作階段的解析度：** – 選取您想要在工作階段中使用的解析度。 **為自己的選擇**會設定解析度根據用戶端的大小。
  - **變更顯示器的大小：** – 時選取高的靜態解析工作階段，您可以選擇將項目在螢幕上的看起來更大，以提升可讀性。 注意： 這僅適用於連線至 Windows 8.1 或上方時。
  - **更新解析度上的調整大小的遠端工作階段**– 啟用時，用戶端會動態更新工作階段解析度根據用戶端的大小。 注意： 這僅適用於連線至 Windows 8.1 或上方時。
  - **剪貼簿**– 啟用時，可讓您將複製文字和影像從遠端電腦。
  - **音訊播放**– 選取裝置要在遠端工作階段期間用於音訊。 您可以選擇播放聲音上的本機裝置遠端電腦，或完全不。
  - **音訊錄製**– 啟用時，可讓您使用本機麥克風與遠端電腦上的應用程式。
4. 點選 [**儲存**]。

需要編輯這些設定？ 點選****（...） 的桌面，名稱旁邊的溢位功能表，然後點選 [**編輯**。

想要刪除的連接嗎？ 同樣地，點選溢位功能表 （**...**），然後點選 [**移除**。

### 新增遠端資源
遠端資源是 RemoteApp 程式、 工作階段為基礎的桌面及發行您的系統管理員使用遠端桌面服務的虛擬桌面。

若要新增遠端資源：

1. 在連線中心畫面上，點選 **+ 新增**]，然後點選**遠端資源**。 
2. 輸入您的系統管理員所提供的**摘要的 URL** ，然後點選 [**尋找的同步發佈摘要**。
3. 出現提示時，提供訂閱摘要所使用的認證。

遠端資源將會顯示在連線中心。


若要刪除遠端資源：

1. 在連線中心中，點選溢位功能表****（...） 旁邊的遠端資源。
2. 點選 \ [**移除**。

### 釘選到 [開始] 功能表已儲存的桌面

釘選到 [開始] 功能表的連線，點選****（...） 的桌面，名稱旁邊的溢位功能表，然後點選**釘選到 [開始] 畫面**。

現在您可以透過點選它，直接從您的 [開始] 功能表啟動遠端桌面連線。

## 連線到 RD 閘道來存取內部的資產

遠端桌面閘道 （RD 閘道） 可讓您從任何地方連線至遠端電腦上的公司網路上網際網路。 您可以建立和管理您的閘道使用遠端桌面用戶端。

若要設定新的閘道︰

1. 在連線中心中，點選 \ [**設定**。
2. 閘道，旁邊點選**+** 以新增新的閘道。 注意： 閘道您也可以新增當您新增新的連線。
3. 輸入下列資訊：
  - **伺服器名稱**– 您想要為閘道使用的電腦的名稱。 這可以是 Windows 的電腦名稱、 網際網路網域名稱或 IP 位址。 您也可以新增連接埠資訊的伺服器名稱 (例如： **RDGateway:443**或**10.0.0.1:443**)。
  - **使用者帳戶**-選取或新增使用者帳戶與您在連線到遠端桌面閘道使用。 您也可以選取**要使用桌面的使用者帳戶**使用那些遠端桌面連線使用相同的認證。
4. 點選 [**儲存**]。  

## 通用應用程式設定

您可以在您的用戶端設定下列全域設定，藉由點選**設定**：

受管理的項目
- **使用者帳戶**-可讓您將新增、 編輯及刪除儲存在用戶端的使用者帳戶。 這是個不錯的方式來變更後更新帳戶的密碼。
- **閘道**-可讓您將新增、 編輯及刪除儲存在用戶端的閘道伺服器。
- **群組**-可讓您將新增、 編輯及刪除儲存在用戶端的群組。 這些可讓您輕鬆地群組連線。

工作階段設定
- **啟動連線以全螢幕**-啟用時，每當啟動連線時，用戶端會使用目前的監視器的整個螢幕。
- **啟動新視窗中的每個連線**-啟用時，每個連線啟動時在個別的視窗中，讓您可以將它們放在不同的監視器上，並使用工作列它們之間切換。
- **大小調整應用程式時：** -可讓您可以控制用戶端視窗調整大小時，會發生什麼事。 預設為**延展式內容，並維持外觀比例**。
- **使用鍵盤命令與：** -讓我們您指定其中的鍵盤命令，例如*贏得*或*ALT + tab 鍵*會使用。 預設值是僅將它們傳送給工作階段連線時是在完整 scren。
- **防止從逾時螢幕**-可讓您讓螢幕保持從逾時在工作階段為使用中。 長時間的連線不需要任何互動時，這是很有幫助。

應用程式設定
- **顯示桌面預覽**-可讓您連線到它之前，請參閱連線中心中的桌上型電腦的預覽。 根據預設，這是設定為**開啟**。
- **有助於改善遠端桌面**-會傳送給 Microsoft 的匿名資料。 我們使用此資料來改善用戶端。 若要深入了解我們如何處理此匿名、 私人資料，請參閱[Microsoft 隱私權陳述式](https://privacy.microsoft.com/en-us/privacystatement)。 根據預設，此設定為**開啟**。

### 管理您的使用者帳戶

當您連接到桌面或遠端資源時，您可以儲存一次從選取的使用者帳戶。 您也可以在用戶端本身，而不是您連線到桌面時，儲存使用者資料中定義的使用者帳戶。

若要建立新的使用者帳戶：

1. 在連線中心中，點選 \ [**設定**。
2. 使用者帳戶旁邊點選**+** 以新增新的使用者帳戶。
3. 輸入下列資訊：
   - **使用者名稱**-使用者儲存與遠端連線搭配使用的名稱。 您可以在任何下列格式輸入使用者名稱： 使用者名稱，domain\user_name，或 user_name@domain.com。
   - **密碼**-您指定的使用者的密碼。 這可以保留空白連線期間提示的密碼。
4. 點選 [**儲存**]。

若要刪除的使用者帳戶：

1. 在連線中心中，點選 \ [**設定**。
2. 選取要刪除使用者帳戶清單中的帳戶。
3. 使用者帳戶旁邊點選 [編輯] 圖示。
4. 點選 \ [**移除此帳戶**底部刪除使用者帳戶。
5. 您也可以編輯的使用者帳戶，並點選 [**儲存**]。

## 瀏覽遠端桌面工作階段
當您開始遠端桌面連線時，有一些工具可用，您可以用來瀏覽工作階段。

### 啟動遠端桌面連線

1. 點選 \ [開始工作階段的遠端桌面連線。
2. 如果您還沒有儲存認證的連線，系統會提示您提供**使用者名稱**和**密碼**。
3. 如果系統要求您驗證遠端桌面的憑證，檢閱資訊，請確定這是您信任之前點選**連線**的電腦。 您也可以選取 [**不詢問有關此憑證再次**一律接受此憑證。

### 連線列

連線列可讓您存取其他瀏覽控制項。 根據預設，連接列放在螢幕頂端中間。 點選並拖曳軸向左或向右移動它。

- **移動瀏覽控制項**的移動瀏覽控制項可讓螢幕，到放大和四處移動。 請注意，移動瀏覽控制項只有在啟用觸控的裝置和使用直接觸控模式。
   - 啟用 / 停用移動瀏覽控制項： 點選移動瀏覽列中的圖示連線顯示移動瀏覽控制項和縮放螢幕。 點選 \ [移動瀏覽列中的圖示連線一次可隱藏控制項並返回其原始的解析度螢幕。
   - 使用移動瀏覽的控制項-輕觸並保留移動瀏覽控制項，然後在拖曳您想要移動在畫面的方向。
   - 將移動瀏覽的控制項-點兩下移動，並保留移動瀏覽控制項來移動在畫面上的控制項。
- **其他選項**-輕觸的其他選項] 圖示以顯示工作階段選擇列和命令列 （如下所示）。
- **鍵盤**點選 [鍵盤] 圖示以顯示或隱藏螢幕小鍵盤。 顯示鍵盤時，會自動顯示移動瀏覽控制項。

### 命令列

點選 **...** 連線列在右邊的螢幕上顯示的命令列上。

- **家用版**-使用 [首頁] 按鈕來從命令列傳回至連線中心。
  - 或者您可以使用 [上一頁] 按鈕，針對相同的動作。
  - 您使用中的工作階段將不會中斷連線。
  - 這可讓您啟動額外的連線。
- **中斷連線**-終止連線使用中斷連線按鈕。
  - 您的應用程式將會維持作用中，只要在遠端電腦上沒有結束工作階段。
- **全螢幕**-進入或離開全螢幕模式。
- **觸控 / 滑鼠**-您可以在滑鼠模式 （直接觸控和滑鼠指標） 之間切換。

### 在遠端工作階段中直接使用觸控手勢與滑鼠模式

互動工作階段提供兩種滑鼠模式。
- **直接觸碰**： 通過所有的觸控點來開始進行解譯從遠端工作階段。
  - 使用相同的方式，就可以使用 Windows 與觸控式螢幕。
- **滑鼠指標**： 會在本機的觸控式螢幕轉換成大型觸控板允許在工作階段中移動滑鼠指標。
  - 使用相同的方式，就可以使用 Windows 與觸控板。

> [!NOTE]
> 互動與 Windows 8 或更新版本的原生的觸控手勢支援直接觸控模式。 

| 滑鼠模式    | 滑鼠作業      | 手勢                                                               |
|---------------|----------------------|-----------------------------------------------------------------------|
| 直接觸控  | 按一下滑鼠左鍵           | 1 指點選                                                          |
| 直接觸控  | 按一下滑鼠右鍵          | 1 手指點選和按住不放                                                 |
| 滑鼠指標 | 按一下滑鼠左鍵           | 1 指點選                                                          |
| 滑鼠指標 | 按一下滑鼠左鍵並拖曳  | double 1 手指的點選和按住不放，然後拖曳                               |
| 滑鼠指標 | 按一下滑鼠右鍵          | 2 個指點選                                                          |
| 滑鼠指標 | 按一下滑鼠右鍵並拖曳 | 2 個 double 手指的點選和按住不放，然後拖曳                               |
| 滑鼠指標 | 滑鼠滾輪          | 2 個手指點選和按住，然後拖曳向上或向下                           |
| 滑鼠指標 | 縮放                 | 使用 2 個手指，並透過捏合進行縮放，或將手指分開的方向移動以縮小內容。 |
