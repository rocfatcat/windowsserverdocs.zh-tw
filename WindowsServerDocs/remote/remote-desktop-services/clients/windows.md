# <a name="get-started-with-remote-desktop-on-windows"></a>開始使用 Windows 上的遠端桌面

>適用於：Windows 10，Windows 8.1、 Windows Server 2019、 Windows Server 2016、windows Server 2012 R2

您可以使用 Windows 的遠端桌面用戶端來使用 Windows 應用程式和從不同的 Windows 裝置的遠端桌面。

若要開始使用下列資訊。 請務必查看[常見問題集](remote-desktop-client-faq.md)如果您有任何問題。

> [!NOTE]
> - 想知道 Windows 用戶端的新版本嗎？ 請參閱[有關在 Windows 上的遠端桌面的新功能？](windows-whatsnew.md)
> - 您可以在任何版本的 Windows 10 上執行的用戶端。

## <a name="get-the-rd-client-and-start-using-it"></a>取得 RD 用戶端，並開始使用

請遵循下列步驟來開始在您的 Windows 10 裝置上使用遠端桌面：

1. 下載遠端桌面用戶端，從[Microsoft Store](https://www.microsoft.com/store/p/microsoft-remote-desktop/9wzdncrfj3ps)。 
2. [您的電腦設定成接受遠端連接](remote-desktop-allow-access.md)。
3. 將遠端桌面連線或存取遠端資源。 直接連接至使用 RemoteApp 程式、 工作階段為基礎的桌面或系統管理員所發行的虛擬桌面的 Windows 電腦和遠端資源的情況下，您在使用連線 
4. 釘選的項目，以便快速取得遠端桌面。

### <a name="add-a-remote-desktop-connection"></a>新增遠端桌面連線

若要建立遠端桌面連線：

1. 在 「 連線中心 」 點選 **+ 新增**，然後點選**Desktop**。
2. 輸入您想要連接到下列資訊的電腦：
   - **電腦名稱**– 電腦的名稱。 這可以是 Windows 電腦名稱、 網際網路網域名稱或 IP 位址。 您也可以在電腦名稱加連接埠資訊 (例如**MyDesktop:3389**或是**10.0.0.1:3389**)。
   - **使用者帳戶**– 要用來存取遠端電腦的使用者帳戶。 點選 **+** 新增新的帳戶，或選取現有的帳戶。 您可以使用下列格式的使用者名稱： *user_name*， *domain\user_name*，或<em>user_name@domain.com</em>。 您也可以指定是否要提示輸入使用者名稱和密碼連線時選取**詢問我每次**。
3. 您也可以設定其他選項點選**顯示更多**:
   - **顯示名稱**– 您要連接到電腦以易記名稱。 您可以使用任何字串，但如果您未指定好記的名稱，會顯示電腦名稱。
   - **群組**– 指定群組，以方便日後尋找您的連線。 您可以加入新的群組，依序點選 **+** 或從清單中選取一個。
   - **閘道**– 您想要用於連線到虛擬桌面、 RemoteApp 程式以及以工作階段為基礎的桌面，在公司內部網路上的遠端桌面閘道。 從您的系統管理員取得閘道的相關資訊。
   - **連線到系統管理員工作階段**-使用此選項來連線到主控台工作階段管理的 Windows 伺服器。
   - **交換滑鼠按鈕**– 使用此選項可為滑鼠右按鈕交換滑鼠左的按鈕函式。 （這是特別有用，如果遠端電腦已設定為慣用左手的使用者，但您有使用右手的滑鼠）。
   - **將我的遠端工作階段解析度設定為：** – 選取您想要在工作階段中使用的解析度。 **我選擇**會設定用戶端的大小為基礎的解析。
   - **變更顯示大小：** – 當工作階段選取較高的靜態解析度，您可以選擇讓螢幕中的項目看起來更大，以提高可讀性。 注意:這只適用於連線至 Windows 8.1 或更新版本時。
   - **更新在調整大小時的遠端工作階段解析度**– 啟用時，用戶端會動態更新工作階段解析度是根據用戶端的大小。 注意:這只適用於連線至 Windows 8.1 或更新版本時。
   - **剪貼簿**– 啟用時，可讓您從遠端電腦的 複製文字和影像來回。
   - **音訊播放**– 選擇要用於遠端工作階段期間的音訊裝置。 您可以選擇播放音效，在本機裝置，也就是在遠端電腦上，或者完全不要。
   - **音訊錄製**– 啟用時，可讓您使用本機的麥克風，與在遠端電腦上的應用程式。
4. 點選**儲存**。

需要編輯這些設定嗎？ 點選 溢位功能表 ( **...** ) 以及與的桌面，然後點選名稱旁邊**編輯**。

想要刪除連線嗎？ 同樣地，請點選 溢位功能表 ( **...** )，然後點選**移除**。

### <a name="add-a-remote-resource"></a>新增遠端資源
遠端資源是 RemoteApp 程式、 工作階段為基礎的桌面及您使用遠端桌面服務的系統管理員所發行的虛擬桌面。

若要新增的遠端資源：

1. 在 「 連線中心 」 畫面上，點選 **+ 新增**，然後點選**遠端資源**。 
2. 請輸入**摘要的 URL**提供的系統管理員，點選**尋找摘要**。
3. 出現提示時，提供要用來訂閱摘要的認證。

遠端資源將會顯示在 「 連線中心。


若要刪除遠端資源：

1. 在 「 連線中心 」 中，點選 溢位功能表 ( **...** ) 旁邊的遠端資源。
2. 點選**移除**。

### <a name="pin-a-saved-desktop-to-your-start-menu"></a>釘選到 [開始] 功能表已儲存的桌面

若要釘選到開始] 功能表的連線，請點選 [溢位功能表 ( **...** ) 以及與的桌面，然後點選名稱旁邊**釘選到開始**。

現在您可以點選它，直接從 [開始] 功能表開始遠端桌面連線。

## <a name="connect-to-an-rd-gateway-to-access-internal-assets"></a>連接至 RD 閘道存取內部資產

遠端桌面閘道 （RD 閘道） 可讓您從任何地方連線到公司網路上的遠端電腦在網際網路上。 您可以建立和管理您使用遠端桌面用戶端的閘道。

若要設定新的閘道：

1. 在 「 連線中心 」 中，點選**設定**。
2. 點選 [閘道] 旁邊 **+** 新增新的閘道。 注意:加入新的連接時，也可以加入一個閘道。
3. 輸入下列資訊：
   - **伺服器名稱**– 您想要作為閘道使用的電腦名稱。 這可以是 Windows 電腦名稱、 網際網路網域名稱或 IP 位址。 您也可以新增連接埠資訊的伺服器名稱 (例如：**RDGateway:443**或是**10.0.0.1:443**)。
   - **使用者帳戶**-選取或新增使用者帳戶以搭配您要連接到遠端桌面閘道。 您也可以選取**使用桌面的使用者帳戶**所使用的遠端桌面連線使用相同的認證。
4. 點選**儲存**。  

## <a name="global-app-settings"></a>全域應用程式設定

您可以在您的用戶端中設定下列全域設定，依序點選**設定**:

受管理的項目
- **使用者帳戶**-可讓您新增、 編輯和刪除儲存在用戶端的使用者帳戶。 這是要更新帳戶的密碼變更後的好方法。
- **閘道**-可讓您新增、 編輯和刪除儲存在用戶端的閘道伺服器。
- **群組**-可讓您新增、 編輯和刪除儲存在用戶端的群組。 這些可讓您輕鬆地群組連線。

工作階段設定
- **以全螢幕開始連線**-啟用時，隨時啟動連線時，用戶端會使用目前的監視整個螢幕。
- **在新視窗中啟動每個連線**-啟用時，每個啟動的連線是在個別視窗中，可讓您將它們放在不同的監視器上，並使用工作列兩者間切換。
- **在應用程式調整大小時：** -可讓您控制用戶端視窗調整大小時，會發生什麼事。 預設值為**自動縮放其內容，維持外觀比例**。
- **使用鍵盤命令：** -讓您指定想鍵盤命令在哪裡*贏得*或*ALT + TAB*習慣。 預設為只將它們傳送到連接處於全螢幕的工作階段。
- **避免逾時的螢幕**-可讓您為了保持畫面逾時工作階段是使用中。 當連線長一段時間不需要任何互動時，這是很有幫助。

應用程式設定
- **顯示桌面預覽**-可讓您連線到它之前，請參閱在 「 連線中心桌面的預覽。 根據預設，這設定為**上**。
- **協助改進遠端桌面**-傳送給 Microsoft 的匿名資料。 我們可以使用這項資料來改善用戶端。 若要深入了解我們如何處理此匿名、 私用的資料，請參閱[Microsoft 隱私權聲明](https://privacy.microsoft.com/en-us/privacystatement)。 根據預設，此設定是**上**。

### <a name="manage-your-user-accounts"></a>管理您的使用者帳戶

當您連線到桌面或遠端資源時，您可以儲存一次從選取的使用者帳戶。 您也可以定義在用戶端本身，而不是儲存使用者資料，當您連線到桌面的使用者帳戶。

若要建立新的使用者帳戶：

1. 在 「 連線中心 」 中，點選**設定**。
2. 旁的使用者帳戶，請點選 **+** 來新增新的使用者帳戶。
3. 輸入下列資訊：
   - **使用者名稱**-要儲存以供使用，透過遠端連線的使用者名稱。 您可以輸入使用者名稱，任何下列格式： user_name domain\user_name，或user_name@domain.com。
   - **密碼**-您指定的使用者的密碼。 這可以保持空白則會提示輸入密碼，在連線期間。
4. 點選**儲存**。

若要刪除的使用者帳戶：

1. 在 「 連線中心 」 中，點選**設定**。
2. 選取要刪除使用者帳戶清單中的帳戶。
3. 旁的使用者帳戶，請點選 [編輯] 圖示。
4. 點選**移除此帳戶**底部刪除使用者帳戶。
5. 您也可以編輯使用者帳戶，然後點選**儲存**。

## <a name="navigate-the-remote-desktop-session"></a>瀏覽遠端桌面工作階段
當您啟動遠端桌面連線時，有一些工具可用，您可以使用瀏覽工作階段。

### <a name="start-a-remote-desktop-connection"></a>啟動遠端桌面連線

1. 點選 遠端桌面連線，啟動工作階段。
2. 如果您尚未儲存連接的認證，您將會提示您提供**使用者名稱**並**密碼**。
3. 如果系統要求您驗證遠端桌面憑證時，檢閱資訊，請確定這是的電腦，再點選您信任**Connect**。 您也可以選取**不要再詢問此憑證的相關**一律接受此憑證。

### <a name="connection-bar"></a>連線列

連接列可讓您存取其他導覽控制項。 根據預設，連線列位於畫面頂端的中間。 點選，拖曳列向左或右移動它。

- **移動瀏覽控制項**-取景位置調整控制項讓螢幕可以放大及四處移動。 請注意，移動瀏覽控制項只適用於觸控式裝置和使用直接觸控模式。
   - 啟用/停用移動瀏覽控制項：點選 [移動瀏覽列中的圖示連線顯示取景位置調整控制項中與縮放] 畫面。 點選 移動瀏覽列中的圖示連線一次，若要隱藏控制項，並傳回其原始的解決方式的畫面。
   - 使用移動瀏覽控制項-Tap 按住移動瀏覽控制項，然後在您想要移動螢幕的方向拖曳。
   - 移動取景位置調整控制項-連點兩下按住不放來移動螢幕上的控制項的取景位置調整控制項。
- **其他選項**-請點選 [其他選項] 圖示，以顯示工作階段選取列和命令列 （如下所示）。
- **鍵盤**-請點選 [鍵盤] 圖示以顯示或隱藏螢幕小鍵盤。 顯示鍵盤時，會自動顯示取景位置調整控制項。

### <a name="command-bar"></a>命令列

點選  **...** 右手邊的畫面上顯示的命令列的 連線 列上。

- **首頁**-使用 [首頁] 按鈕以返回 「 連線中心 」，從命令列。
  - 或者您可以使用 [上一頁] 按鈕相同的動作。
  - 將不會中斷您的使用中工作階段。
  - 這可讓您啟動其他連線。
- **中斷連線**-使用中斷連線 按鈕來終止連線。
  - 您的應用程式會持續處於作用，只要在遠端電腦上未終止工作階段。
- **全螢幕**-進入或離開全螢幕模式。
- **觸控 / 滑鼠**-您可以 （直接觸控及滑鼠指標） 的滑鼠模式之間切換。

### <a name="use-direct-touch-gestures-and-mouse-modes-in-a-remote-session"></a>在遠端工作階段中使用直接觸控筆勢和滑鼠模式

兩個滑鼠模式可與工作階段互動。
- **直接觸控**:通過所有的工作階段以遠端方式解譯的觸控接點。
  - 在相同的方式，您會使用觸控式螢幕中的 Windows 中使用。
- **滑鼠指標**:將您本機的觸控式螢幕轉換成大觸控板可讓工作階段中移動滑鼠指標。
  - 在相同的方式，您會使用觸控板中的 Windows 中使用。

> [!NOTE]
> 互動 Windows 8 或更新版本的原生的觸控筆勢支援直接觸控模式。 

| 滑鼠模式    | 滑鼠作業      | 手勢                                                               |
|---------------|----------------------|-----------------------------------------------------------------------|
| 直接觸控  | 按一下滑鼠左鍵           | 1 的手指點選                                                          |
| 直接觸控  | 按一下滑鼠右鍵          | 1 手指點選並按住                                                 |
| 滑鼠指標 | 按一下滑鼠左鍵           | 1 的手指點選                                                          |
| 滑鼠指標 | 按一下滑鼠左鍵並拖曳  | 1 的手指 double 點選並按住，然後拖曳                               |
| 滑鼠指標 | 按一下滑鼠右鍵          | 2 個手指點選                                                          |
| 滑鼠指標 | 按一下滑鼠右鍵並拖曳 | 2 個手指 double 點選並按住，然後拖曳                               |
| 滑鼠指標 | 滑鼠滾輪          | 2 個手指點選並按住不放，然後向上或向下拖曳                           |
| 滑鼠指標 | 縮放                 | 使用 2 個手指，並縮小來放大或縮小移動手指分開的。 |
