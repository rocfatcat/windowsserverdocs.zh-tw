# <a name="remote-desktop-services---gpu-acceleration"></a>遠端桌面服務 - GPU 加速

遠端桌面服務適用於原生圖形加速以及 Windows Server 支援的圖形虛擬化技術。 如需這些技術的相關資訊、其差異以及部署方式，請參閱 [Windows Server 中 GPU 加速的規劃](../../virtualization/hyper-v/plan/plan-for-gpu-acceleration-in-windows-server.md)。

在 RDS 環境中規劃圖形加速時，您選擇的使用者規模和使用者工作負載會驅動對圖形轉譯技術的選擇：
