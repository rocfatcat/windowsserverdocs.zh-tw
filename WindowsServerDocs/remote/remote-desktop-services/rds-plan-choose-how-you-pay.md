# <a name="remote-desktop-services---choose-how-you-pay"></a>遠端桌面服務-選擇支付方式

>適用於：Windows Server （半年通道），Windows Server 2016

選擇您以有意義的貴公司的授權。 每位使用者，若要讓遠端的 BYOD 案例中其裝置上的使用者授權。 每個裝置，如果使用者共用相同的裝置授權。 如果您是服務提供者 （HSP 或 MSP） 或 ISV，選擇每個使用者 Sal 授權有彈性、 隨用隨付模型。

如需詳細資訊，請參閱[與用戶端存取使用權 (Cal) 授權您的 RDS 部署](rds-client-access-license.md)。
