# <a name="remote-desktop-services---choose-how-you-pay"></a>遠端桌面服務 - 選擇支付方式

>適用於：Windows Server (半年通道)、Windows Server 2019、Windows Server 2016

選擇適用於貴公司的授權內容。 每位使用者的授權，讓使用者能夠在 BYOD 案例中的任何裝置上進行遠端存取。 每個裝置的授權 (如果使用者共用相同的裝置)。 如果您是服務提供者 (HSP 或 MSP) 或 ISV，請選擇每個使用者 SAL 授權，以獲得靈活的隨用隨付模型。

如需詳細資訊，請參閱[使用用戶端存取授權 (CAL) 授權您的 RDS 部署](rds-client-access-license.md)。
